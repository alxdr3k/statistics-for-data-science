# 14. SVM · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 마진을 최대화하는 이유를 설명하라.

가장 가까운 점들과의 여백이 넓으면 작은 데이터 흔들림에도 경계가 덜 민감해져 일반화에 도움이 된다.

## 2. 서포트 벡터가 중요한 이유를 설명하라.

마진을 직접 결정하는 경계 가까운 점들이기 때문에 이 점들이 경계 위치에 큰 영향을 준다.

## 3. 슬랙 변수의 역할을 설명하라.

완벽히 분리되지 않는 데이터에서 일부 마진 위반을 허용해 현실적인 경계를 만들게 한다.

## 4. C가 커질 때와 작아질 때의 차이를 설명하라.

C가 크면 위반을 강하게 벌해 좁은 마진을 선택할 수 있고, C가 작으면 위반을 더 허용해 넓은 마진을 선택할 수 있다.

## 5. 커널 트릭의 직관을 설명하라.

고차원 특징을 직접 만들지 않고도 그 공간에서의 유사도 계산 효과를 얻는 방법이다.

## 6. RBF 커널의 gamma가 너무 크면 어떤 문제가 생길 수 있는가?

각 점의 영향 범위가 좁아져 결정경계가 지나치게 복잡해지고 과대적합될 수 있다.

## 7. 왜 SVM은 가장 가까운 점들에 특히 민감한가?

- **한 문장 답**: 결정경계의 여백인 마진은 경계와 가장 가까운 점들이 정하기 때문이다.
- **이유**: 마진은 모든 점이 아니라 경계에 가장 가까운 점까지의 거리로 정해진다. 그 점들이 움직이면 최적 경계도 함께 바뀐다.
- **예시**: 멀리 있는 점은 경계를 조금 움직여도 여전히 잘 분류될 가능성이 크다.
- **주의점**: 이 가까운 점들을 서포트 벡터라고 부른다.

## 8. 왜 C가 크다고 항상 좋은 것은 아닌가?

- **한 문장 답**: 훈련 데이터 위반을 너무 강하게 벌하면 예외까지 맞추려 해 경계가 과하게 복잡해질 수 있기 때문이다.
- **이유**: C가 크면 위반을 거의 허용하지 않아서 잡음이나 예외를 맞추려고 경계가 꺾일 수 있다. 훈련 오류 감소가 일반화 향상을 보장하지 않는다.
- **예시**: 노이즈 한 점을 맞추려다 전체 경계가 흔들릴 수 있다.
- **주의점**: C는 마진 넓이와 위반 허용 사이의 균형이다.

## 9. 라그랑지안 승수와 KKT 조건이 SVM에서 하는 역할을 설명하라.

라그랑지안 승수는 마진 조건 같은 제약을 최적화 문제 안으로 가져온다. KKT 조건은 최적해에서 제약을 만족하는 방식과 승수의 관계를 정한다. 이 관계 덕분에 마진에 걸린 점들만 `alpha`가 남고, 그 점들이 서포트 벡터가 되어 경계를 결정한다.

## 10. 커널 함수가 대칭성과 양의 준정성을 만족해야 하는 이유를 설명하라.

커널은 고차원 공간에서의 내적처럼 동작해야 한다. 내적이라면 순서를 바꿔도 값이 같아야 하므로 대칭성이 필요하고, 커널 행렬이 음의 제곱거리처럼 이상하게 행동하지 않도록 양의 준정성이 필요하다. 이 조건이 있어야 커널 트릭을 최적화 문제 안에서 안정적으로 사용할 수 있다.

## 11. SVM의 선형 분류 함수 `f(x) = w^T x + b`에서 예측 클래스는 어떻게 정해지는가?

`f(x)`의 부호로 정한다. `f(x) >= 0`이면 `+1`, `f(x) < 0`이면 `-1` 클래스로 예측한다. 결정경계는 `f(x) = 0`, 즉 `w^T x + b = 0`인 지점이다.

## 12. hard-margin SVM에서 `1/2 w^T w`를 최소화하는 이유를 설명하라.

마진 폭이 `2 / ||w||`이기 때문이다. 마진을 크게 하려면 `||w||`가 작아져야 하고, 계산하기 쉬운 같은 방향의 목적함수로 `1/2 w^T w`를 최소화한다. 단, 모든 데이터는 `y_i(w^T x_i + b) >= 1` 조건을 만족해야 한다.

## 13. soft-margin SVM에서 `0 <= alpha_i <= C`가 의미하는 바를 설명하라.

`alpha_i`는 각 데이터의 마진 제약에 붙는 라그랑지안 승수다. `alpha_i = 0`이면 경계에 거의 영향을 주지 않는 점이고, `0 < alpha_i < C`이면 마진 경계 위의 서포트 벡터로 볼 수 있다. `alpha_i = C`이면 마진 안쪽에 있거나 오분류된 점일 수 있다.

## 14. 커널을 쓰는 쌍대 문제에서 `X X^T`가 무엇으로 바뀌는가?

데이터 사이 내적 행렬 `X X^T`가 커널 행렬 `K(X, X)`로 바뀐다. 그래서 예측식도 `f(x) = (alpha o y)^T K(X, x) + b`처럼 데이터와 새 점 사이의 커널값을 사용한다.

## 15. 다중 클래스 분류에서 OVR과 OVO의 차이를 설명하라.

OVR은 한 클래스를 양성으로 두고 나머지 모든 클래스를 음성으로 묶어 여러 이진 SVM을 만든다. OVO는 클래스 두 개씩 짝지어 여러 이진 SVM을 만든 뒤 투표처럼 결과를 합친다. 단일 SVM이 기본적으로 이진 분류기라서 이런 조합 방식이 필요하다.

## 16. Linear, Polynomial, RBF, Sigmoid 커널 공식을 쓰고 `gamma`가 결정경계에 미치는 영향을 설명하라.

대표 커널은 다음처럼 쓴다.

```text
Linear: K(x, y) = x^T y
Polynomial: K(x, y) = (gamma x^T y + b)^d
RBF: K(x, y) = exp(-gamma ||x - y||^2)
Sigmoid: K(x, y) = tanh(gamma x^T y + b)
```

RBF에서 `gamma`가 크면 한 점의 영향 범위가 좁아져 가까운 데이터에 강하게 반응한다. 그래서 결정경계가 복잡해지고 분산이 커질 수 있다. `gamma`가 작으면 영향 범위가 넓어져 경계가 더 부드러워진다.

## 17. 마진 경계 위 서포트 벡터로 `b*`와 커널 예측식 `f(x)`를 계산하는 구조를 설명하라.

`alpha_i > 0`인 점이 서포트 벡터이고, soft-margin에서 `0 < alpha_i < C`인 점은 마진 경계 위 서포트 벡터로 본다.

```text
Support Vectors = {x_i | alpha_i > 0}
SVMP = {x_i | 0 < alpha_i < C}
```

마진 경계 위 점들은 마진 조건을 등호로 만족하므로, 이 점들을 이용해 `b*`를 평균으로 계산할 수 있다.

```text
b* = (1 / N_svmp) * sum_{i in SVMP}
     [y_i - (alpha o y)^T K(X, x_i)]
```

커널 SVM의 예측식은 새 점 `x`와 학습 데이터 사이의 커널값을 이용한다.

```text
f(x) = (alpha o y)^T K(X, x) + b*
```

선형 커널이면 이 식은 `f(x) = w^T x + b*`와 같은 형태로 돌아온다.

## 18. `w^T x + b`의 부호와 경계까지의 거리가 각각 무엇을 뜻하는지 설명하라.

`w^T x + b`의 부호는 점이 결정경계의 어느 쪽에 있는지를 말한다. 값이 0 이상이면 `+1`, 0보다 작으면 `-1` 클래스로 예측한다. 경계까지의 실제 거리는 `|w^T x + b| / ||w||`로 읽을 수 있다. 따라서 부호는 클래스 방향이고, 거리는 경계에서 얼마나 여유 있게 떨어져 있는지를 나타낸다.

## 19. SVM에서 입력 변수 표준화가 중요한 이유를 설명하라.

SVM은 거리, 내적, 마진을 사용한다. 변수 스케일이 크게 다르면 숫자 범위가 큰 변수가 거리와 내적을 지배할 수 있다. 예를 들어 소득은 수천만 단위이고 나이는 두 자리 수라면 소득 차이가 마진 계산에 훨씬 크게 반영된다. 표준화는 변수들이 비슷한 기준에서 경계와 커널값에 영향을 주도록 돕는다.

## 20. `C`를 훈련 오류와 마진 폭의 균형으로 설명하라.

`C`는 마진 위반을 얼마나 강하게 벌할지 정하는 값이다. `C`가 크면 위반을 크게 벌하므로 훈련 데이터를 더 엄격히 맞추려 하고, 마진이 좁아지거나 경계가 복잡해질 수 있다. `C`가 작으면 일부 위반을 허용해 더 넓고 부드러운 마진을 선택할 수 있다. 그래서 `C`는 훈련 오류를 줄이는 것과 일반화를 위해 여유를 두는 것 사이의 균형이다.
