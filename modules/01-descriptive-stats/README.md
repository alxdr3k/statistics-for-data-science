# Module 1 · Descriptive Statistics

> 데이터를 *한두 개의 숫자* 로 압축할 때 무엇이 살아남고 무엇이 잘려 나가는지 익히는 모듈입니다.

## 이 모듈을 끝내면

- "평균"이라는 단어를 보면 즉시 *분포 모양* 을 함께 떠올린다.
- 같은 데이터를 *어떤 한 숫자* 로 요약하면 가장 정직한지를 판단할 기준이 생긴다.
- 분포의 *퍼짐* 을 측정하는 세 가지 도구(분산·표준편차·IQR) 를 손과 눈으로 다룬다.
- 박스플롯·히스토그램·바이올린플롯 같은 시각화가 *어느 결정에 어떻게 들어가는지* 를 안다.
