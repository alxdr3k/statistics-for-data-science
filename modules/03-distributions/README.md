# Module 3 · Probability Distributions

> *"이 데이터는 어떤 분포를 따르는가?"* — 이 질문에 답할 수 있어야 추론이 시작됩니다. 각 분포가 *왜* 등장하는지부터 익히는 모듈입니다.

## 이 모듈을 끝내면

- 베르누이·이항·포아송·정규 분포를 *상황의 언어* 로 구분할 수 있다 ("이 데이터는 이항 분포처럼 생겼다").
- 각 분포의 파라미터가 *현실의 무엇을 조절* 하는지 설명할 수 있다.
- t·χ²·F 분포가 *왜 필요한지* — 정규 분포만으로 부족한 이유 — 를 설명할 수 있다.
- 분포를 시각화하고, 파라미터를 바꾸면 모양이 어떻게 변하는지 직접 확인할 수 있다.
