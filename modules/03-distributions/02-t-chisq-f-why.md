# 02 · t · χ² · F 분포 — 왜 이 분포가 필요한가

> 정규 분포 하나로 모든 것을 해결할 수 있다면 좋겠지만, 현실에서는 *모분산을 모르거나*, *데이터가 범주형이거나*, *두 그룹의 분산을 비교해야 하는* 상황이 반드시 옵니다. 이 세 상황이 각각 t 분포, χ² 분포, F 분포를 요청합니다. 이 챕터는 "왜"에만 집중합니다 — 실제 검정 절차는 Module 6에서 다룹니다.

## 1. 시나리오

### 시나리오 — 신약 임상시험 계속

임상 1상이 끝났습니다. 이제 분석팀이 세 가지 질문을 합니다.

**질문 A**: 치료군 20명의 평균 회복 시간이 대조군과 다른가?
→ 표본이 작고, 모분산(σ²)을 모릅니다. 정규 분포를 그대로 쓸 수 없습니다.

**질문 B**: 부작용 종류(없음·경증·중증)의 비율이 사전 예상과 다른가?
→ 범주형 데이터입니다. 평균·분산 개념 자체가 없습니다.

**질문 C**: 두 임상 센터(서울·부산)의 부작용 발생 분산이 같은가?
→ 두 집단의 분산 자체를 비교해야 합니다.

세 질문이 각각 세 분포를 불러옵니다.

---

## 2. 이대로 가면 무엇이 위험한가 (Why)

**모분산을 모를 때 정규를 쓰면**: 표준 오차를 *알 수 없는 σ* 로 나눠야 하므로 z-검정을 쓸 수 없습니다. 표본 표준편차 $s$ 로 대체할 때 생기는 *추가 불확실성* 을 반영하지 않으면 기각 여부가 달라집니다.

**범주형 데이터에 평균 기반 검정을 쓰면**: 범주는 크기 순서가 없으므로(경증 < 중증, 하지만 중증 - 경증 = ?) 평균이 무의미합니다.

**분산 비교에 t-test를 쓰면**: t-test는 평균 비교 도구입니다. 분산 차이를 검정하는 것은 전혀 다른 문제입니다.

---

## 3. 어디서 쓰이나 (Where)

| 분포 | 주요 쓰임 |
|------|----------|
| **t 분포** | 소표본 평균 검정, 회귀계수 유의성 검정, 신뢰구간 |
| **χ² 분포** | 범주형 독립성 검정, 적합도 검정, 분산 신뢰구간 |
| **F 분포** | 분산 동질성 검정, ANOVA, 회귀 모델 전체 유의성 |

## 4. 직관 → 정의 → 수식

### t 분포 — "표본 표준편차로 대체할 때 생기는 불확실성"

**직관**: 표본 평균 $\bar{X}$ 를 표준화할 때 $\sigma$ 대신 $s$ (표본 표준편차)를 쓰면, 결과 통계량은 정규가 아니라 *꼬리가 두꺼운* 분포를 따릅니다. 꼬리가 두꺼울수록 극단값이 더 자주 나옵니다.

$$
T = \frac{\bar{X} - \mu}{s / \sqrt{n}} \sim t(n-1)
$$

- 자유도 $\nu = n-1$: 자유도가 클수록 t 분포는 표준 정규에 수렴.
- $n \geq 30$ 이면 t와 정규의 차이가 실용적으로 무시 가능 (경험칙).
- 자유도가 작을수록 꼬리가 두껍습니다 → 같은 효과에 더 큰 t 값이 필요 → 더 보수적.

---

### χ² 분포 — "표준 정규 변수의 제곱합"

**직관**: 표준 정규 변수 $Z_1, Z_2, \ldots, Z_k$ 를 각각 제곱해 더하면 χ² 분포가 됩니다.

$$
\chi^2 = Z_1^2 + Z_2^2 + \cdots + Z_k^2 \sim \chi^2(k)
$$

범주형 데이터 분석에 왜 등장하는가: 관측 빈도($O$)와 기대 빈도($E$)의 차이를 제곱해 합산하면 χ² 통계량이 됩니다.

$$
\chi^2 = \sum \frac{(O - E)^2}{E}
$$

- 항상 양수 (제곱합이니까).
- 자유도가 클수록 분포가 오른쪽으로 이동.
- 자유도 2일 때 지수 분포와 동일.

---

### F 분포 — "두 χ² 변수의 비"

**직관**: 두 집단의 *분산 비* 를 나타내는 통계량이 F 분포를 따릅니다.

$$
F = \frac{\chi^2_1 / \nu_1}{\chi^2_2 / \nu_2} \sim F(\nu_1, \nu_2)
$$

ANOVA에서의 해석:

$$
F = \frac{\text{집단 간 분산 (Between)}}{\text{집단 내 분산 (Within)}}
$$

F 값이 크면 집단 간 차이가 집단 내 산포보다 크다 → 집단 평균이 같다고 보기 어렵다.

---

### 세 분포의 관계

```
표준 정규 Z
├── Z² → χ²(1)
├── Z₁² + Z₂² + ... + Zₖ² → χ²(k)
├── Z / √(χ²(ν)/ν) → t(ν)
└── [χ²(ν₁)/ν₁] / [χ²(ν₂)/ν₂] → F(ν₁, ν₂)
```

모두 *정규 분포에서 파생된 분포* 입니다.

## 6. 함정 & 자주 하는 오해 (Pitfalls)

- **"n이 크면 t-test 대신 z-test"**: 맞지만, 실무에서는 t-test가 n이 커도 자동으로 정규에 수렴하므로 항상 t-test를 써도 무방합니다. 단, 정규성 자체가 깨진 경우(극단적 편포)는 별도 처리가 필요합니다.
- **χ² 검정의 기대 빈도 조건**: 각 셀의 기대 빈도 $E \geq 5$ 여야 χ² 근사가 타당합니다. 작으면 Fisher의 정확 검정을 씁니다.
- **F 분포는 양측 검정이 기본이 아닙니다**: 분산이 "더 큰가"를 보는 단측 검정이 자주 쓰입니다. 양측 F 검정은 $\alpha/2$ 를 양쪽에 나눠야 합니다.
- **ANOVA의 F-test가 유의해도 어느 쌍이 다른지는 모릅니다**: 사후 검정(Tukey HSD 등)이 필요합니다 — Module 8에서 다룹니다.
- **t 분포 꼬리가 두껍다 = 더 보수적**: 같은 효과에 더 강한 증거가 필요하므로, 소표본에서 t 를 쓰면 정규보다 기각하기 어렵습니다.
