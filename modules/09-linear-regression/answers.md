# 09. 선형회귀 분석 · 확인 문제 해설

## 1. 잔차가 중요한 이유를 설명하라.

잔차는 실제값에서 모델 예측값을 뺀 남은 오차다. 잔차가 무작위로 흩어지면 직선 모델이 관계를 어느 정도 설명했다고 볼 수 있다. 반대로 곡선이나 부채꼴 패턴이 보이면 선형성, 등분산성 같은 가정이 깨졌을 수 있다.

## 2. 선형회귀의 주요 가정 중 두 가지를 설명하라.

선형성은 독립변수와 종속변수의 평균 관계를 직선으로 표현할 수 있다는 가정이다. 등분산성은 예측값의 크기와 상관없이 잔차의 퍼짐이 비슷하다는 가정이다. 이 외에도 잔차 독립성, 잔차 정규성 등이 자주 점검된다.

## 3. R2가 높아도 모델이 좋지 않을 수 있는 이유를 설명하라.

R2는 훈련 데이터에서 변동을 얼마나 설명하는지 보여 주지만 과대적합을 직접 막아 주지는 않는다. 불필요한 변수를 많이 넣으면 R2가 높아져도 새 데이터 예측은 나빠질 수 있다. 잔차 진단과 검증 데이터 성능을 함께 봐야 한다.

## 4. 다중공선성이 계수 해석을 어렵게 하는 이유를 설명하라.

독립변수들이 서로 강하게 관련되어 있으면 각 변수의 순수한 효과를 분리하기 어렵다. 작은 데이터 변화에도 계수 크기와 부호가 크게 흔들릴 수 있다. 그래서 계수 해석이 불안정해진다.

## 5. 잔차 플롯에서 부채꼴 모양이 보이면 무엇을 의심해야 하는가?

예측값이 커질수록 잔차의 퍼짐도 커지거나 작아지는 등분산성 위반을 의심한다. 이런 경우 변환, 가중회귀, 다른 모델 구조를 검토할 수 있다. 단순히 평균 오차만 보면 이 문제를 놓치기 쉽다.

## 6. 회귀계수를 현실 단위로 해석하는 예를 들어라.

광고비를 만 원 단위로 넣고 매출을 만 원 단위로 예측한다고 하자. 광고비 계수가 2라면 다른 조건이 같을 때 광고비가 1만 원 늘 때 평균 매출이 2만 원 늘어난다고 해석한다. 계수 해석에는 변수 단위가 반드시 함께 필요하다.
