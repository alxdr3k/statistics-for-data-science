# 09. 선형회귀 분석 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 잔차가 중요한 이유를 설명하라.

잔차는 실제값에서 모델 예측값을 뺀 남은 오차다. 잔차가 무작위로 흩어지면 직선 모델이 관계를 어느 정도 설명했다고 볼 수 있다. 반대로 곡선이나 부채꼴 패턴이 보이면 선형성, 등분산성 같은 가정이 깨졌을 수 있다.

## 2. 선형회귀의 주요 가정 중 두 가지를 설명하라.

선형성은 독립변수와 종속변수의 평균 관계를 직선으로 표현할 수 있다는 가정이다. 등분산성은 예측값의 크기와 상관없이 잔차의 퍼짐이 비슷하다는 가정이다. 이 외에도 잔차 독립성, 잔차 정규성 등이 자주 점검된다.

## 3. R2가 높아도 모델이 좋지 않을 수 있는 이유를 설명하라.

R2는 훈련 데이터에서 변동을 얼마나 설명하는지 보여 주지만 과대적합을 직접 막아 주지는 않는다. 불필요한 변수를 많이 넣으면 R2가 높아져도 새 데이터 예측은 나빠질 수 있다. 잔차 진단과 검증 데이터 성능을 함께 봐야 한다.

## 4. 다중공선성이 계수 해석을 어렵게 하는 이유를 설명하라.

독립변수들이 서로 강하게 관련되어 있으면 각 변수의 순수한 효과를 분리하기 어렵다. 작은 데이터 변화에도 계수 크기와 부호가 크게 흔들릴 수 있다. 그래서 계수 해석이 불안정해진다.

## 5. 잔차 플롯에서 부채꼴 모양이 보이면 무엇을 의심해야 하는가?

예측값이 커질수록 잔차의 퍼짐도 커지거나 작아지는 등분산성 위반을 의심한다. 이런 경우 변환, 가중회귀, 다른 모델 구조를 검토할 수 있다. 단순히 평균 오차만 보면 이 문제를 놓치기 쉽다.

## 6. 회귀계수를 현실 단위로 해석하는 예를 들어라.

광고비를 만 원 단위로 넣고 매출을 만 원 단위로 예측한다고 하자. 광고비 계수가 2라면 다른 조건이 같을 때 광고비가 1만 원 늘 때 평균 매출이 2만 원 늘어난다고 해석한다. 계수 해석에는 변수 단위가 반드시 함께 필요하다.

## 7. 왜 잔차 플롯을 봐야 회귀 모델을 믿을 수 있는가?

- **한 문장 답**: 평균 오차만으로는 곡선 패턴, 부채꼴 분산, 이상점을 놓칠 수 있기 때문이다.
- **이유**: 회귀 요약 숫자는 평균적인 적합만 보여 주기 쉽다. 잔차를 그리면 모델이 설명하지 못한 구조가 입력값에 따라 남아 있는지 확인할 수 있다.
- **예시**: 잔차가 x가 커질수록 커지면 등분산성 가정이 약할 수 있다.
- **주의점**: R2가 높아도 잔차 패턴이 나쁘면 해석을 조심한다.

## 8. 왜 독립변수끼리 너무 비슷하면 계수 해석이 불안정해지는가?

- **한 문장 답**: 서로 같은 정보를 설명하려고 경쟁하기 때문이다.
- **이유**: 두 변수가 같은 방향으로 움직이면 모델이 어느 변수의 효과인지 분리하기 어렵다. 작은 데이터 변화에도 계수가 크게 바뀔 수 있다.
- **예시**: 키와 팔 길이를 같이 넣으면 어느 변수가 영향을 준 것인지 계수가 흔들릴 수 있다.
- **주의점**: 예측 성능과 계수 해석 안정성은 따로 봐야 한다.

## 9. `SST = SSR + SSE`가 회귀모델 해석에서 뜻하는 바를 설명하라.

전체 변동 `SST`는 모델이 설명한 변동 `SSR`과 설명하지 못한 오차 변동 `SSE`로 나뉜다. 좋은 회귀모델은 전체 변동 중 많은 부분을 설명하고, 남은 오차를 작게 만든다. 결정계수는 이 분해를 바탕으로 모델이 설명한 비율을 보여 준다.

## 10. AIC와 BIC가 훈련 오차만 보는 기준과 다른 점을 설명하라.

AIC와 BIC는 모델의 적합도뿐 아니라 복잡도 벌점도 함께 본다. 변수를 많이 넣으면 훈련 오차는 줄어들 수 있지만, 불필요하게 복잡해져 새 데이터에 약해질 수 있다. BIC는 대체로 복잡도에 더 강한 벌점을 준다.

## 11. 이상점과 영향점의 차이를 설명하라.

이상점은 관측값이 전체 패턴에서 멀리 떨어진 점이다. 영향점은 그 점을 포함하느냐 빼느냐에 따라 회귀계수나 회귀선이 크게 바뀌는 점이다. 모든 이상점이 영향점은 아니고, x공간에서 멀리 떨어진 점은 특히 큰 영향을 줄 수 있다.

## 12. 단순회귀와 다중회귀에서 최소제곱 추정식이 어떻게 달라지는지 설명하라.

단순회귀는 독립변수가 하나라서 기울기와 절편을 각각 `beta1_hat = sum((xi - xbar)(yi - ybar)) / sum((xi - xbar)^2)`, `beta0_hat = ybar - beta1_hat * xbar`로 쓸 수 있다. 다중회귀는 독립변수가 여러 개라 계수를 벡터로 묶고, `beta_hat = (X^T X)^(-1)X^T y`처럼 행렬식으로 푼다. 둘 다 잔차제곱합 SSE를 최소화한다는 원리는 같다.

## 13. 회귀모델 전체 유의성 F검정과 회귀계수 t검정의 차이를 설명하라.

F검정은 모델 전체가 의미 있는지 본다. 귀무가설은 모든 기울기 계수가 0이라는 주장이고, `MSR/MSE`를 F분포와 비교한다. 계수 t검정은 특정 독립변수 하나의 계수가 0인지 본다. 따라서 F검정은 모델 전체 질문이고, t검정은 개별 변수 질문이다.

## 14. VIF 기준과 다중공선성 대응 방법을 설명하라.

VIF는 한 독립변수가 나머지 독립변수들로 얼마나 잘 설명되는지 보는 지표다. 보통 VIF가 5~10이면 다중공선성을 의심하고, 10 이상이면 심각하다고 본다. 대응은 표준화가 아니라 변수 제거, 변수 결합, PCA/PLS 같은 차원축소, Ridge/Lasso 같은 규제를 사용한다. 표준화는 단위를 맞출 뿐 강한 상관 구조를 없애지 못한다.

## 15. 로지스틱 회귀에서 `exp(beta)`를 어떻게 해석하는지 설명하라.

로지스틱 회귀의 계수는 확률 자체의 증가량이 아니라 log-odds의 변화다. 독립변수 `X_j`가 1 증가하면 승산은 `exp(beta_j)`배가 된다. 예를 들어 `exp(beta_j) = 1.5`라면 다른 조건이 같을 때 성공 승산이 1.5배가 된다고 해석한다.

## 16. Softmax 방식과 One-vs-Rest 방식의 차이를 설명하라.

Softmax 방식은 여러 클래스 점수를 한 번에 정규화해 클래스 확률들이 합 1이 되도록 만든다. 클래스 간 경쟁 구조를 한 모델 안에서 함께 본다. One-vs-Rest는 각 클래스를 “그 클래스 vs 나머지”의 이진 문제로 나누어 여러 모델을 학습한다. 구현은 단순하지만 클래스 간 상호작용을 Softmax만큼 직접 반영하지는 못한다.

## 17. 단순 선형회귀의 최소제곱 목적함수와 `beta0_hat`, `beta1_hat` 식을 쓰라.

단순 선형회귀는 `yi = beta0 + beta1 xi + epsilon_i`로 둔다. 최소제곱법은 잔차제곱합을 최소화한다.

```text
L(beta0_hat, beta1_hat) = sum(yi - beta0_hat - beta1_hat xi)^2
SSE = sum(yi - yhat_i)^2
```

`dL / d beta0_hat = 0`, `dL / d beta1_hat = 0`을 만족하는 정규방정식을 풀면 다음 식이 나온다.

```text
beta1_hat = sum((xi - xbar)(yi - ybar)) / sum((xi - xbar)^2)
beta0_hat = ybar - beta1_hat * xbar
```

## 18. 다중회귀의 행렬식 `beta_hat = (X^T X)^(-1)X^T y`가 어떤 조건에서 나오는지 설명하라.

다중회귀는 `y = X beta + epsilon`으로 쓴다. 절편을 포함하려면 `X`에 값이 모두 1인 열을 추가한다. 최소제곱 목적함수는 `L = (y - X beta_hat)^T(y - X beta_hat)`이다. 이를 `beta_hat`으로 미분하면 `-2X^T(y - X beta_hat)`이고, 0으로 두면 `X^T X beta_hat = X^T y`가 된다. `X^T X`가 역행렬을 가질 때 `beta_hat = (X^T X)^(-1)X^T y`로 풀 수 있다.

## 19. `R^2`, 조정 `R^2`, AIC, BIC가 각각 모델을 어떤 관점에서 평가하는지 설명하라.

`R^2 = 1 - SSE / SST`는 전체 변동 중 모델이 설명한 비율을 본다. 변수를 추가하면 잘 떨어지지 않는 성질이 있어 과대평가될 수 있다. 조정 `R^2 = 1 - (SSE / (n - k)) / (SST / (n - 1))`는 자유도를 반영해 변수를 많이 넣는 효과를 보정한다.

AIC와 BIC는 최대우도와 모수 수 벌점을 함께 본다. `AIC = 2k - 2log(L)`, `BIC = klog(n) - 2log(L)`이다. 둘 다 작을수록 선호하며, BIC는 표본 수 `n`이 커질수록 복잡도에 더 큰 벌점을 준다.

## 20. 회귀계수 t검정에서 `SE(beta_j_hat)`와 `sigma_hat^2`가 어떻게 계산되는지 쓰라.

회귀계수 t검정은 `H0: beta_j = 0`, `H1: beta_j != 0`을 검정한다. 통계량은 다음과 같다.

```text
t_j = beta_j_hat / SE(beta_j_hat) ~ t(n - k)
SE(beta_j_hat) = sqrt(sigma_hat^2 * [(X^T X)^(-1)]_jj)
sigma_hat^2 = SSE / (n - k)
```

여기서 `[(X^T X)^(-1)]_jj`는 행렬의 `j`번째 대각원소다. 표준오차가 크면 같은 계수 크기라도 t값이 작아져 유의하다고 보기 어려워진다.

## 21. leverage, studentized residual, DFFITS, DFBETAS가 각각 무엇을 진단하는지 설명하라.

leverage는 관측치가 독립변수 공간에서 얼마나 멀리 있는지 본다. 식은 `h_ii = x_i^T(X^T X)^(-1)x_i`이다. 값이 크면 그 점이 회귀선 위치에 영향을 줄 가능성이 있다.

studentized residual은 잔차를 표준화해 y방향으로 얼마나 튀는지 본다. DFFITS는 한 관측치를 제외했을 때 그 관측치의 예측값이 얼마나 달라지는지 본다. DFBETAS는 한 관측치를 제외했을 때 특정 회귀계수가 얼마나 달라지는지 본다. 따라서 이상점은 주로 값이 튀는지, 영향점은 모델 자체를 흔드는지를 구분해 봐야 한다.

## 22. 로지스틱 회귀의 likelihood와 log-likelihood 식을 쓰고, 해석적 해가 어려운 이유를 설명하라.

로지스틱 회귀는 `P(Y = 1 | X = x_i) = sigma(z_i)`, `z_i = beta^T x_i`로 둔다. 관측값 `y_i`가 0 또는 1이면 각 관측치의 확률은 다음처럼 하나의 식으로 쓸 수 있다.

```text
P(Y = y_i | X = x_i)
= sigma(z_i)^y_i * (1 - sigma(z_i))^(1 - y_i)

L(beta) = product_i sigma(z_i)^y_i * (1 - sigma(z_i))^(1 - y_i)
log L(beta) = sum_i [y_i log sigma(z_i)
              + (1 - y_i)log(1 - sigma(z_i))]
```

이 식은 선형회귀처럼 계수에 대해 닫힌형 해가 바로 나오지 않는다. 그래서 경사하강법, 뉴턴 방법 같은 반복 최적화로 `log L(beta)`를 최대화하는 `beta_hat`을 구한다.

## 23. 다중회귀 계수 해석에서 “다른 조건이 같을 때”가 왜 필요한지 설명하라.

다중회귀는 여러 독립변수를 동시에 넣는다. 특정 계수 `beta_j`는 다른 변수들을 고정했을 때 `X_j`가 한 단위 변하면 `Y`의 평균이 얼마나 달라지는지를 뜻한다. 이 조건을 빼면 계수를 단순히 두 변수의 상관처럼 오해할 수 있다. 예를 들어 광고비 계수는 가격, 시즌, 지역 같은 다른 변수가 같은 조건에서 광고비가 변할 때의 평균 효과로 읽어야 한다.

## 24. 잔차가 양수와 음수일 때 각각 모델 예측이 어떤 방향으로 빗나간 것인지 설명하라.

잔차는 `실제값 - 예측값`이다. 잔차가 양수이면 실제값이 예측값보다 크므로 모델이 실제보다 작게 예측한 것이다. 잔차가 음수이면 실제값이 예측값보다 작으므로 모델이 실제보다 크게 예측한 것이다. 잔차의 부호와 크기를 보면 모델이 어디에서 과소예측 또는 과대예측하는지 확인할 수 있다.

## 25. 계수 검정과 신뢰구간을 해석할 때 잔차 가정을 확인해야 하는 이유를 설명하라.

회귀계수의 t검정과 신뢰구간은 잔차의 분산, 정규성, 독립성 같은 가정에 기대어 표준오차와 기준 분포를 정한다. 잔차에 곡선 패턴이 남아 있거나 분산이 일정하지 않으면 표준오차가 부정확해지고 p-value나 신뢰구간 해석도 흔들릴 수 있다. 그래서 계수의 유의성만 보기 전에 잔차 플롯, Q-Q plot, 등분산성, 영향점을 함께 점검해야 한다.
