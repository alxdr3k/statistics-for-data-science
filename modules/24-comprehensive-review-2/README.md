# 24. 종합 문제 2

## 학습 목표

이 차시는 계산, 모델 구조, 전처리, 검정, 차원 축소가 섞인 두 번째 종합 복습이다. 답을 바로 찍기보다 조건을 분해해 어떤 개념을 적용할지 고르는 연습을 한다.

- 신경망 구조와 파라미터 수를 입력·출력 모양으로 판단한다.
- 차원 축소, 속성 선택, 회귀 진단 문제를 목적 기준으로 구분한다.
- 결측치, 보간, 사건 수 모델링 같은 데이터 처리 문제를 상황에 맞게 고른다.

## 오늘의 한 줄

두 번째 종합 문제는 “비슷해 보이는 방법들 중 목적이 다른 하나”를 구분하는 연습이다.

## 오늘 반드시 이해할 3가지

1. 구조 문제는 입력과 출력의 개수, 연결 방식, 파라미터 수를 먼저 본다.
2. 선택 문제는 예측 성능을 높이려는지, 설명 가능한 변수를 고르려는지 구분한다.
3. 데이터 처리 문제는 값의 의미와 시간·공간 구조를 보존하는지 확인한다.

## 종합 복습 주제

이 차시는 다음 주제를 섞어 확인한다.

- RNN 입출력 구조
- 계층적 군집화 linkage
- 최소제곱법 계수 해석
- 속성 선택과 가설검정 기반 feature selection
- 활성화 함수와 기울기 소실 완화
- 머신러닝 모델의 단점 비교
- 일반화 효과가 다른 개념 구분
- 신경망 구조 식별
- 오토인코더 목적
- 표준화, 정규화, 정규성 관련 기법 구분
- SVD와 PCA
- 전진 선택법과 후진 제거법
- 신경망 가중치와 편향 수 계산
- 경사부스팅 손실함수
- 비선형 결정경계 기법
- 선형회귀 적합성 진단
- 포아송 분포와 단위 면적 사건 수
- 결측치 또는 이상치 처리
- 시계열 온도 데이터 보간

## 문제를 푸는 순서

### 1. 신경망 문제는 모양부터 본다

RNN 구조는 입력과 출력의 길이로 먼저 구분한다. 하나의 입력에서 여러 출력이 나오면 이미지 속 여러 물체 위치처럼 `one-to-many` 구조를 생각할 수 있고, 여러 입력에서 하나의 출력이 나오면 문장 분류 같은 `many-to-one` 구조를 떠올릴 수 있다.

파라미터 수 계산은 각 층의 `입력 수 x 출력 수 + 편향 수`로 계산한다. 그림이 복잡해 보여도 층별 입력 노드와 출력 노드를 세면 된다.

### 2. 차원 축소와 속성 선택은 목적이 다르다

PCA와 SVD는 원래 변수들을 조합해 새 축을 만든다. 정보 보존과 표현 압축이 목적이다. 반면 전진 선택법, 후진 제거법, filter/wrapper/embedded 방식은 원래 변수 중 어떤 것을 쓸지 고르는 속성 선택이다.

가설검정 기반 속성 선택은 각 속성과 타깃의 관계가 통계적으로 의미 있는지 본다. 다만 여러 속성을 반복 검정하면 우연히 유의해 보이는 변수가 생길 수 있으므로, 다중검정 문제와 검증 성능을 함께 고려한다.

### 3. 데이터 처리 문제는 구조를 보존해야 한다

시간에 따른 온도처럼 순서가 있는 데이터의 결측치는 앞뒤 시간 흐름을 무시하면 안 된다. 선형 보간은 인접 시점 사이가 비교적 부드럽게 변한다고 볼 수 있을 때 쓸 수 있다.

단위 면적당 효모 수처럼 일정 공간 안의 사건 수를 세는 문제는 포아송 분포를 떠올릴 수 있다. 핵심은 값이 연속 측정량인지, 사건의 개수인지 구분하는 것이다.

## 대표 판단 기준

| 문제 단서 | 먼저 떠올릴 기준 | 연결 차시 |
|---|---|---|
| RNN 입출력 그림 | one-to-one, one-to-many, many-to-one, many-to-many | 20차시 |
| linkage | 단일, 완전, 평균 연결 기준 | 16차시 |
| 최소제곱법 | 잔차제곱합 최소화 | 9차시 |
| feature selection | filter, wrapper, embedded | 18차시 |
| 기울기 소실 | ReLU 계열, skip connection, normalization | 19, 22차시 |
| SVD와 PCA | 특이값, 주성분, 분산 보존 | 17차시 |
| 파라미터 수 | 가중치 수 + 편향 수 | 19, 20차시 |
| 사건 수 | 포아송 분포 | 5차시 |
| 시계열 결측 | 보간과 시간 순서 | 2차시 |

## 예시 풀이

### 예시 1. 파라미터 수 계산

입력이 4개이고 다음 Dense layer의 뉴런이 3개라면 가중치는 `4 x 3 = 12`개다. 각 뉴런마다 편향이 하나씩 있으므로 편향은 3개다. 이 층의 파라미터 수는 15개다.

### 예시 2. PCA와 속성 선택 구분

PCA는 기존 변수를 버리고 새 축을 만든다. 그래서 해석이 어려워질 수 있지만 정보 압축에는 유리하다. 전진 선택법은 기존 변수 중 일부를 고르므로 변수 이름이 남아 해석은 상대적으로 쉽다.

## 확인 문제

1. 하나의 입력에서 여러 출력을 내는 RNN 구조가 필요한 문제를 설명하라.
2. Ward linkage가 군집을 결합하는 기준을 설명하라.
3. 최소제곱법으로 회귀계수를 구할 때 무엇을 최소화하는지 설명하라.
4. 단계적 전진선택법이 최적 변수 조합을 보장하지 않는 이유를 설명하라.
5. 가설검정 기반 속성 선택에서 검정 방법과 데이터 구조를 맞춰야 하는 이유를 설명하라.
6. Leaky ReLU가 dead ReLU 문제를 완화하는 방식을 설명하라.
7. 랜덤포레스트에서 나무 수를 늘리는 것이 부스팅 반복 증가와 다른 이유를 설명하라.
8. Dropout, smoothing, bagging과 kernel trick의 목적 차이를 설명하라.
9. GAN의 생성자와 판별자가 각각 어떤 역할을 하는지 설명하라.
10. Denoising autoencoder가 잡음 제거를 배우는 방식을 설명하라.
11. Grid Search와 Random Search의 차이를 설명하라.
12. SVD가 PCA의 주성분 방향과 어떻게 연결되는지 설명하라.
13. 전진 선택과 후진 제거, 단계적 선택의 한계를 설명하라.
14. Dense layer의 가중치와 편향 수를 층별로 계산하는 방법을 설명하라.
15. Gradient Boosting에서 기반 모델이 손실의 방향을 맞춘다는 뜻을 설명하라.
16. RBF SVM과 1-NN이 비선형 결정경계를 만들 수 있는 이유를 설명하라.
17. 선형회귀 적합성 진단에서 VIF, 잔차 정규성, 등분산성을 확인하는 이유를 설명하라.
18. 모분산을 모르는 정규 표본 평균 검정에서 t분포와 자유도를 고르는 방법을 설명하라.
19. EM 알고리즘이 잠재변수와 모수를 번갈아 갱신하는 방식을 설명하라.
20. 8시 18도, 11시 24도일 때 9시 값을 선형 보간으로 계산하라.

## 오늘의 요약 5줄

1. 신경망 구조 문제는 입력과 출력의 모양부터 본다.
2. PCA/SVD는 새 축을 만드는 차원 축소이고, 속성 선택은 기존 변수 일부를 고르는 일이다.
3. 파라미터 수는 가중치와 편향을 나누어 세면 덜 헷갈린다.
4. 사건 수 데이터는 포아송 분포, 시간 순서 결측은 보간을 먼저 떠올릴 수 있다.
5. 종합 문제에서는 비슷한 방법들의 목적 차이를 말로 설명할 수 있어야 한다.
