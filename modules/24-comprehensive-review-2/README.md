# 24. 종합 문제 2

## 학습 목표

이 차시는 계산, 모델 구조, 전처리, 검정, 차원 축소가 섞인 두 번째 종합 복습이다. 답을 바로 찍기보다 조건을 분해해 어떤 개념을 적용할지 고르는 연습을 한다.

- 신경망 구조와 파라미터 수를 입력·출력 모양으로 판단한다.
- 차원 축소, 속성 선택, 회귀 진단 문제를 목적 기준으로 구분한다.
- 결측치 보간, 평균 검정, 모델 진단 같은 데이터 처리 문제를 상황에 맞게 고른다.

## 오늘의 한 줄

두 번째 종합 문제는 “비슷해 보이는 방법들 중 목적이 다른 하나”를 구분하는 연습이다.

## 오늘 반드시 이해할 3가지

1. 구조 문제는 입력과 출력의 개수, 연결 방식, 파라미터 수를 먼저 본다.
2. 선택 문제는 예측 성능을 높이려는지, 설명 가능한 변수를 고르려는지 구분한다.
3. 데이터 처리 문제는 값의 의미와 시간·공간 구조를 보존하는지 확인한다.

## 종합 복습 주제

이 차시는 다음 주제를 섞어 확인한다.

- RNN 입출력 구조
- 계층적 군집화 linkage
- 최소제곱법 계수 해석
- 속성 선택과 가설검정 기반 feature selection
- 활성화 함수와 기울기 소실 완화
- 머신러닝 모델의 단점 비교
- 일반화 효과가 다른 개념 구분
- 신경망 구조 식별
- 오토인코더 목적
- 표준화, 정규화, 정규성 관련 기법 구분
- SVD와 PCA
- 전진 선택법과 후진 제거법
- 신경망 가중치와 편향 수 계산
- 경사부스팅 손실함수
- 비선형 결정경계 기법
- 선형회귀 적합성 진단
- 모분산 미지 정규 표본의 평균 검정
- 결측치 또는 이상치 처리
- 시계열 온도 데이터 보간

## 문제를 푸는 순서

### 1. 신경망 문제는 모양부터 본다

RNN 구조는 입력과 출력의 길이로 먼저 구분한다. 하나의 입력에서 여러 출력이 나오면 이미지 속 여러 물체 위치처럼 `one-to-many` 구조를 생각할 수 있고, 여러 입력에서 하나의 출력이 나오면 문장 분류 같은 `many-to-one` 구조를 떠올릴 수 있다.

파라미터 수 계산은 각 층의 `입력 수 x 출력 수 + 편향 수`로 계산한다. 그림이 복잡해 보여도 층별 입력 노드와 출력 노드를 세면 된다.

### 2. 차원 축소와 속성 선택은 목적이 다르다

PCA와 SVD는 원래 변수들을 조합해 새 축을 만든다. 정보 보존과 표현 압축이 목적이다. 반면 전진 선택법, 후진 제거법, filter/wrapper/embedded 방식은 원래 변수 중 어떤 것을 쓸지 고르는 속성 선택이다.

가설검정 기반 속성 선택은 각 속성과 타깃의 관계가 통계적으로 의미 있는지 본다. 다만 여러 속성을 반복 검정하면 우연히 유의해 보이는 변수가 생길 수 있으므로, 다중검정 문제와 검증 성능을 함께 고려한다.

### 3. 데이터 처리 문제는 구조를 보존해야 한다

시간에 따른 온도처럼 순서가 있는 데이터의 결측치는 앞뒤 시간 흐름을 무시하면 안 된다. 선형 보간은 인접 시점 사이가 비교적 부드럽게 변한다고 볼 수 있을 때 쓸 수 있다.

표본이 정규분포를 따르고 모분산을 모르는 평균 검정 문제에서는 표본분산으로 표준오차를 추정하므로 t분포를 사용한다. 표본 수가 `n`이면 자유도는 `n - 1`이다. 문제에서 정규분포와 모분산 미지를 명시하면 평균 검정의 조건을 먼저 따른다.

## 세부 판단 포인트

RNN 구조는 입력과 출력의 개수를 먼저 본다. 이미지 하나에서 여러 물체 후보를 순차적으로 찾는다면 하나의 입력에서 여러 출력을 만드는 `one-to-many` 구조가 자연스럽다.

계층적 군집화의 결합 기준은 거리 정의가 다르다. Single linkage는 두 군집 사이 데이터 포인트 거리의 최솟값, Complete linkage는 최댓값, Average linkage는 평균값을 쓴다. Ward 방식은 두 군집을 결합했을 때 군집 내부 분산이 얼마나 증가하는지를 본다.

최소제곱 회귀계수는 정규방정식으로 계산할 수 있다. 절편을 포함한 설계행렬이 `X`, 관측값이 `y`이면 `beta_hat = (X^T X)^(-1) X^T y`다. 예를 들어 `x = 1,2,3,4`, `y = 2,5,7,8`이면 계산 결과는 `beta_hat_1 = 2`, `beta_hat_0 = 0.5`다.

속성 선택에서 단계적 전진선택법은 계산량이 많아도 최적 속성 조합을 보장하지 않는다. 전진 선택, 후진 제거, 단계적 선택은 탐욕적 절차라 현재 단계에서 좋아 보이는 선택이 전체 최적과 다를 수 있다.

가설검정 기반 속성 선택은 검정의 목적과 데이터 구조가 맞아야 한다. 이진 대상과 연속형 변수의 평균 차이는 독립표본 t 검정으로 볼 수 있고, 연속형 대상과 연속형 변수의 관계는 상관분석으로 볼 수 있다. Levene 검정은 등분산성 검정이고, Wilcoxon 부호순위 검정은 대응 관계가 있을 때 쓰므로 제시된 일반 다중 클래스 분류 상황에는 맞지 않는다.

ReLU의 dead node 문제는 모든 학습 데이터에서 음수 구간으로 들어가 기울기가 0이 되는 상황이다. Leaky ReLU는 음수 구간에도 작은 기울기를 남겨 이 문제를 완화한다.

모델의 단점은 모델의 학습 방식과 연결해 판단한다. 랜덤포레스트는 bagging 기반이라 나무를 추가하면 분산을 평균화하는 효과가 크다. 그래서 “기반 모델을 추가할 때 과적합에 유의해야 한다”는 설명은 부스팅 쪽에는 더 자연스럽지만 랜덤포레스트의 핵심 단점으로는 부적절하다.

Dropout, smoothing, bagging은 과적합 완화 또는 분산 감소 쪽의 기법이다. Kernel trick은 선형 모델이 표현하지 못하는 비선형 경계를 다루도록 표현력을 키우는 방법이므로 과소적합 완화 쪽에 가깝다.

GAN은 생성자와 판별자로 구성된다. 생성자는 가짜 데이터를 만들고, 판별자는 진짜와 가짜를 구분한다. 판별자가 구분을 잘하면 생성자가 더 그럴듯한 데이터를 만들도록 압박받고, 판별자가 틀리면 판별자 쪽이 개선된다.

Denoising autoencoder는 손상된 입력에서 깨끗한 입력을 복원하도록 학습한다. 속성 사이에 선형 관계가 강한 경우 PCA로 낮은 차원 성분만 남기고 원래 공간으로 복원하면 선택되지 않은 성분에 많이 실린 노이즈가 줄어드는 직관과 연결할 수 있다.

Grid Search는 주어진 하이퍼파라미터 후보의 모든 조합을 평가한다. Random Search는 후보 공간에서 일부 조합을 무작위로 뽑아 평가한다. Tomek's Link는 최근접 이웃이 다른 클래스인 경계 표본을 다루는 불균형 데이터 정리 기법이고, 유전자 알고리즘은 진화 과정을 모방한 메타휴리스틱 최적화다.

SVD로 PCA를 유도할 때 중심화 행렬 `X_c`의 공분산행렬은 `A = X_c^T X_c / (n - 1)`이다. `X_c = U Sigma V^T`이면 `X_c^T X_c = V Sigma^T U^T U Sigma V^T = V Sigma^T Sigma V^T`가 되어, 오른쪽 특이벡터 `V`가 주성분 방향과 연결된다.

신경망 파라미터 수는 층별로 가중치와 편향을 나눠 센다. 입력 2개에서 은닉 뉴런 4개로 가면 가중치는 `2 x 4 = 8`, 편향은 `4`개다. 은닉 4개에서 출력 2개로 가면 가중치는 `4 x 2 = 8`, 편향은 `2`개다. 전체 가중치는 `16`, 편향은 `6`이다.

이진분류용 경사부스팅에서 손실이 `L(y_i, p_hat_i) = -y_i log p_hat_i - (1 - y_i) log(1 - p_hat_i)`일 때, `p_hat_i`는 기반 점수 `F_m(x_i)`에 로지스틱 함수를 적용해 얻을 수 있다. 기반 모델은 손실의 음의 기울기 방향을 맞추는 회귀 나무로 이해하므로, “의사결정나무 분류기”라고 단정하는 설명은 부정확하다. 로지스틱 손실의 기울기는 `dL / dF_m(x_i) = p_hat_i - y_i` 형태로 정리된다.

비선형 결정경계는 모델에 비선형 표현력이 있는지로 판단한다. 단층 퍼셉트론은 선형 경계만 만들고, 활성화 함수가 없는 다층 퍼셉트론도 선형 변환의 합성이라 비선형성을 만들지 못한다. RBF 커널 SVM과 1-NN은 비선형 결정경계를 만들 수 있다.

선형회귀 진단은 예측값 자체의 QQ plot보다 잔차 구조를 본다. VIF는 다중공선성, Shapiro-Wilk는 잔차 정규성, Bartlett 검정은 구간별 잔차 등분산성 점검에 연결된다. 예측값과 대상변수의 QQ plot은 선형성, 등분산성, 잔차 정규성, 독립성, 다중공선성 없음 중 어떤 것도 직접 확인하기 어렵다.

모분산을 모르는 정규 표본 평균 검정에서는 `t = (X_bar - mu_0) / (S / sqrt(n))`를 쓰고, 자유도는 `n - 1`이다. 표본 수가 `12`이면 `t(11)`을 사용한다.

EM은 잠재변수가 있는 모델에서 현재 모수로 잠재변수의 사후분포를 추정하고, 완전데이터 로그우도의 기대값을 최대화하도록 모수를 반복 갱신한다. GMM에서는 각 표본이 어떤 가우시안 성분에서 왔는지를 잠재변수로 두고 평균, 분산, 혼합비를 갱신한다. 초기값에 따라 지역해에 머물 수 있으므로 초기값과 무관하게 항상 최적값을 보장한다고 말하면 안 된다.

선형 보간은 앞뒤 시점의 값을 시간 비율로 연결한다. 8시 `18도`, 11시 `24도`이면 3시간 동안 6도 상승했으므로 9시는 8시에서 1시간 뒤, 즉 `18 + 6 x (1 / 3) = 20도`다.

## 대표 판단 기준

| 문제 단서 | 먼저 떠올릴 기준 | 답안에 포함할 말 |
|---|---|---|
| RNN 입출력 그림 | one-to-one, one-to-many, many-to-one, many-to-many | 입력 개수와 출력 개수를 먼저 센다. |
| linkage | 단일, 완전, 평균, Ward 기준 | 군집 사이 거리를 어떤 값으로 정의하는지 말한다. |
| 최소제곱법 | 잔차제곱합 최소화 | 예측값 자체가 아니라 잔차의 제곱합을 최소화한다. |
| feature selection | filter, wrapper, embedded | 기존 변수 중 일부를 고르는 일인지 새 축을 만드는 일인지 구분한다. |
| 기울기 소실 | ReLU 계열, skip connection, normalization | 과적합 대응 기법과 학습 신호 안정화 기법을 섞지 않는다. |
| SVD와 PCA | 특이값, 주성분, 분산 보존 | 중심화된 행렬의 오른쪽 특이벡터가 주성분 방향과 연결된다. |
| 파라미터 수 | 가중치 수 + 편향 수 | 층별 입력 수와 출력 뉴런 수를 나누어 센다. |
| 모분산 미지 정규 표본 평균 | t분포, 자유도 n - 1 | 표본분산으로 표준오차를 추정한다. |
| 시계열 결측 | 보간과 시간 순서 | 앞뒤 시점의 시간 간격과 변화가 부드럽다는 가정을 함께 본다. |

계산 문제에서는 숫자를 넣기 전에 어떤 구조를 세는지부터 적는다. Dense layer 파라미터 수는 전체 그림을 한 번에 세지 않고 층별로 가중치와 편향을 분리한다. 평균 검정은 표본 수, 모분산을 아는지, 정규성 조건을 먼저 확인한다. 보간은 앞뒤 관측값 사이가 비교적 부드럽게 변한다는 전제가 있을 때 자연스럽다.

## 예시 풀이

### 예시 1. 파라미터 수 계산

입력이 4개이고 다음 Dense layer의 뉴런이 3개라면 가중치는 `4 x 3 = 12`개다. 각 뉴런마다 편향이 하나씩 있으므로 편향은 3개다. 이 층의 파라미터 수는 15개다.

### 예시 2. PCA와 속성 선택 구분

PCA는 기존 변수를 버리고 새 축을 만든다. 그래서 해석이 어려워질 수 있지만 정보 압축에는 유리하다. 전진 선택법은 기존 변수 중 일부를 고르므로 변수 이름이 남아 해석은 상대적으로 쉽다.

## 확인 문제

1. 하나의 입력에서 여러 출력을 내는 RNN 구조가 필요한 문제를 설명하라.
2. Ward linkage가 군집을 결합하는 기준을 설명하라.
3. 최소제곱법으로 회귀계수를 구할 때 무엇을 최소화하는지 설명하라.
4. 단계적 전진선택법이 최적 변수 조합을 보장하지 않는 이유를 설명하라.
5. 가설검정 기반 속성 선택에서 검정 방법과 데이터 구조를 맞춰야 하는 이유를 설명하라.
6. Leaky ReLU가 dead ReLU 문제를 완화하는 방식을 설명하라.
