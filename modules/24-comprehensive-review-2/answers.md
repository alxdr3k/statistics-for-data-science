# 24. 종합 문제 2 · 확인 문제 해설

## 1. one-to-many RNN

하나의 입력에서 여러 출력을 내야 하면 one-to-many 구조를 본다. 이미지 하나에서 여러 객체 후보나 위치를 순차적으로 출력하는 문제는 단일 입력, 다중 출력 구조로 이해할 수 있다.

## 2. Ward linkage

Ward 방식은 두 군집을 결합했을 때 군집 내부 분산이 얼마나 증가하는지를 기준으로 결합 대상을 고른다. Single은 최소거리, Complete는 최대거리, Average는 평균거리를 쓴다.

## 3. 최소제곱 회귀계수

최소제곱법은 예측값과 실제값의 차이인 잔차를 제곱해 모두 더한 값을 최소화하는 계수를 구한다. 원문 예시처럼 \(x=(1,2,3,4)\), \(y=(2,5,7,8)\)이고 절편을 포함하면 정규방정식 계산 결과는 기울기 \(\hat{\beta}_1=2\), 절편 \(\hat{\beta}_0=0.5\)다. 중요한 것은 점을 가장 잘 지나는 직선을 눈으로 고르는 것이 아니라, 잔차제곱합을 가장 작게 하는 계수를 계산한다는 점이다.

## 4. 속성 선택 한계

전진 선택, 후진 제거, 단계적 선택은 탐욕적 탐색이다. 계산량을 더 쓴다고 전역 최적 변수 조합을 보장하지 않는다. 변수 간 상호작용과 다중공선성이 있으면 선택 경로가 흔들릴 수 있다.

## 5. 가설검정 기반 속성 선택

가설검정으로 속성을 고를 때는 변수와 대상의 형태를 먼저 맞춰야 한다. 연속형 변수와 이진 대상의 평균 차이는 독립표본 t 검정으로 볼 수 있고, 연속형 변수와 연속형 대상의 관계는 상관분석으로 볼 수 있다. 반면 Levene 검정은 등분산성 검정이라 범주형 변수 선택 자체의 근거로 쓰기 어렵고, Wilcoxon 부호순위 검정은 대응 관계가 있을 때 쓰는 검정이라 다중 클래스 대상과 연속형 변수의 일반적 관계 판단에는 맞지 않는다. 대응 관계가 있는지, 분포 가정이 맞는지, 여러 변수를 반복 검정하는지도 함께 확인해야 한다.

## 6. Leaky ReLU

ReLU는 음수 구간에서 기울기가 0이라 모든 입력에 대해 비활성화되면 dead node가 될 수 있다. Leaky ReLU는 음수 구간에도 작은 기울기를 남겨 이 문제를 완화한다.

## 7. 랜덤포레스트 단점

랜덤포레스트는 bagging 기반이라 나무를 더 추가하면 분산이 평균화되는 효과가 크다. 단일 부스팅처럼 기반 모델 추가 자체가 곧 과적합 위험을 크게 키우는 성격과는 다르다.

## 8. 과적합 완화와 과소적합 완화

Dropout, smoothing, bagging은 과적합이나 분산을 줄이는 쪽에 가깝다. Kernel trick은 선형 경계로 부족한 모델 표현력을 키워 과소적합을 완화하는 데 쓰일 수 있다.

## 9. GAN

생성자와 판별자가 경쟁하며 학습하는 구조는 GAN이다. 생성자는 가짜 데이터를 만들고, 판별자는 진짜와 가짜를 구분한다. 두 모델의 경쟁이 생성 품질을 높이는 방향으로 작동한다.

## 10. Denoising Autoencoder

Denoising autoencoder는 손상된 입력에서 깨끗한 입력을 복원하도록 학습한다. PCA로 차원을 줄였다가 다시 복원할 때 노이즈 성분이 줄어드는 직관과 연결해서 이해할 수 있다.

## 11. Grid Search

정해 둔 하이퍼파라미터 후보의 모든 조합을 평가하는 방법은 Grid Search다. Random Search는 후보 공간에서 일부 조합을 무작위로 뽑아 평가한다.

## 12. SVD와 PCA

중심화한 입력행렬의 공분산행렬은 `X_c^T X_c / (n - 1)`로 쓸 수 있다. SVD를 적용하면 오른쪽 특이벡터가 PCA의 주성분 방향과 연결되고, 특이값 제곱이 분산 크기와 연결된다.

## 13. 전진 선택과 후진 제거

전진 선택은 변수를 추가하고, 후진 제거는 변수를 제거한다. 단계적 선택은 둘을 섞을 수 있지만 성능 향상을 보장하지 않는다. 필터 방법보다 느리지만 모델 안에서 변수 조합 효과를 볼 수 있다.

## 14. 신경망 파라미터 수

층마다 `입력 수 x 출력 뉴런 수`로 가중치를 세고, 출력 뉴런 수만큼 편향을 더한다. 예를 들어 입력 2개에서 은닉 뉴런 4개로 가면 가중치가 `2 x 4 = 8`, 편향이 4개다. 은닉 4개에서 출력 2개로 가면 가중치가 `4 x 2 = 8`, 편향이 2개다. 따라서 전체 가중치는 16개, 전체 편향은 6개다.

## 15. Gradient Boosting 손실

이진분류 손실을 써도 Gradient Boosting의 각 기반 모델은 손실의 음의 기울기 또는 잔차 방향을 맞추는 회귀 나무로 이해한다. 그래서 "기반 모델은 의사결정나무 분류기"라는 설명은 정확하지 않다. 로지스틱 함수를 거친 확률 \(\hat{p}_i\)를 쓰면 손실의 \(F_m(x_i)\)에 대한 기울기는 \(\hat{p}_i-y_i\) 형태로 정리되며, 각 단계는 이전 단계의 부족한 방향을 조금씩 보완한다.

## 16. 비선형 결정경계

단층 퍼셉트론은 선형 경계만 만들고, 활성화가 없는 다층 퍼셉트론도 결국 선형 변환의 합성이다. RBF 커널 SVM은 비선형 경계를 만들 수 있고, 1-NN도 데이터 배치에 따라 매우 구불구불한 비선형 경계를 만들 수 있다.

## 17. 선형회귀 적합성 진단

VIF는 독립변수끼리 너무 강하게 관련되어 회귀계수가 불안정해지는지 보는 데 쓴다. 잔차 정규성은 추론 결과를 해석할 때 중요하고, 등분산성은 예측값의 크기에 따라 오차의 퍼짐이 달라지는지 확인하는 기준이다. 회귀 진단은 예측값 자체보다 잔차가 어떤 패턴을 보이는지에 초점을 둔다.

## 18. 모분산 미지 평균 검정

정규분포 표본이고 모분산을 모르면 표본분산으로 표준오차를 추정하므로 t분포를 쓴다. 표본 수가 12이면 자유도는 `12 - 1 = 11`이다. 표본평균의 표준오차는 표본분산을 12로 나눈 값의 제곱근, 즉 표본표준편차를 `sqrt(12)`로 나눈 형태로 이해하면 된다.

## 19. EM

잠재변수가 있고, 현재 모수로 잠재변수의 사후분포를 계산한 뒤 완전데이터 로그우도의 기대값을 최대화해 모수를 갱신하는 방법은 EM이다. GMM에서 각 표본의 성분 소속을 잠재변수로 두는 예가 대표적이다.

## 20. 선형 보간

8시 18도, 11시 24도라면 3시간 동안 6도 상승했다. 9시는 8시에서 1시간 뒤이므로 `18 + 6 x (1/3) = 20`도다. 시간 간격을 분모로 두고 결측 시점의 상대 위치를 곱한다.
