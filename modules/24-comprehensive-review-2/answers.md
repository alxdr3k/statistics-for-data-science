# 24. 종합 문제 2 해설

## 1. one-to-many RNN

하나의 입력에서 여러 출력이 필요한 문제에 어울린다. 예를 들어 한 이미지에서 여러 위치나 객체 후보를 순차적으로 출력하는 구조를 생각할 수 있다.

## 2. 단일 연결과 완전 연결

단일 연결은 두 군집에서 가장 가까운 두 점의 거리를 군집 간 거리로 본다. 완전 연결은 가장 먼 두 점의 거리를 본다. 단일 연결은 길게 이어지는 군집이 생기기 쉽고, 완전 연결은 더 조밀한 군집을 선호한다.

## 3. 최소제곱법

예측값과 실제값의 차이인 잔차를 제곱해 모두 더한 값을 최소화한다. 제곱을 쓰면 양수와 음수가 상쇄되지 않고 큰 오차에 더 큰 벌점이 붙는다.

## 4. 속성 선택 방식

Filter는 모델 학습 전 통계량이나 관계성 기준으로 변수를 고른다. Wrapper는 여러 변수 조합을 실제 모델 성능으로 평가한다. Embedded는 모델 학습 과정 안에서 변수 선택이 함께 일어난다.

## 5. ReLU와 기울기 소실

Sigmoid는 입력이 크거나 작을 때 기울기가 0에 가까워진다. ReLU는 양수 구간에서 기울기가 1이므로 깊은 층에서도 학습 신호가 상대적으로 잘 전달된다.

## 6. PCA와 전진 선택법

PCA는 원래 변수를 선형 결합해 새 주성분을 만든다. 전진 선택법은 기존 변수 중 성능을 개선하는 변수를 하나씩 추가한다. PCA는 압축, 전진 선택법은 변수 선택에 가깝다.

## 7. Dense layer 파라미터 수

입력 노드 수와 출력 뉴런 수를 곱하면 가중치 수가 된다. 여기에 출력 뉴런 수만큼 편향을 더한다. 공식은 `입력 수 x 출력 수 + 출력 수`다.

## 8. 포아송 분포 조건

일정 시간이나 공간 안에서 사건이 몇 번 일어나는지 세는 상황에 어울린다. 사건 발생률이 비교적 일정하고, 서로 독립적으로 발생한다고 볼 때 자주 사용한다.

## 9. 잔차 진단 이유

회귀모델의 가정이 맞는지 확인하기 위해서다. 잔차가 특정 패턴을 보이면 선형성, 등분산성, 독립성, 정규성 가정이 깨졌을 가능성이 있다.

## 10. 시계열 선형 보간 주의점

앞뒤 값 사이가 비교적 부드럽게 변한다는 가정이 필요하다. 갑작스러운 이벤트나 계절성이 강한 구간에서는 단순 직선 보간이 실제 변화를 왜곡할 수 있다.

## 원본 문항별 해설

### Q1. one-to-many RNN

하나의 입력에서 여러 출력을 내야 하면 one-to-many 구조를 본다. 이미지 하나에서 여러 객체 후보나 위치를 순차적으로 출력하는 문제는 단일 입력, 다중 출력 구조로 이해할 수 있다.

### Q2. Ward linkage

Ward 방식은 두 군집을 결합했을 때 군집 내부 분산이 얼마나 증가하는지를 기준으로 결합 대상을 고른다. Single은 최소거리, Complete는 최대거리, Average는 평균거리를 쓴다.

### Q3. 최소제곱 회귀계수

최소제곱법은 `X^T X`와 `X^T y`를 이용해 잔차제곱합을 최소화하는 계수를 구한다. 원본 계산에서는 기울기 2, 절편 0.5가 나온다. 중요한 것은 점을 가장 잘 지나는 직선이 아니라 잔차제곱합을 가장 작게 하는 직선이라는 점이다.

### Q4. 속성 선택 한계

전진 선택, 후진 제거, 단계적 선택은 탐욕적 탐색이다. 계산량을 더 쓴다고 전역 최적 변수 조합을 보장하지 않는다. 변수 간 상호작용과 다중공선성이 있으면 선택 경로가 흔들릴 수 있다.

### Q5. 가설검정 기반 속성 선택

연속형 변수와 이진 대상의 차이는 t-독립검정으로 볼 수 있고, 연속형 대상과 연속형 변수 관계는 상관분석으로 볼 수 있다. Levene 검정은 등분산성 검정이고, Wilcoxon 부호순위 검정은 대응 관계에 쓰므로 해당 상황과 맞지 않는다.

### Q6. Leaky ReLU

ReLU는 음수 구간에서 기울기가 0이라 모든 입력에 대해 비활성화되면 dead node가 될 수 있다. Leaky ReLU는 음수 구간에도 작은 기울기를 남겨 이 문제를 완화한다.

### Q7. 랜덤포레스트 단점

랜덤포레스트는 bagging 기반이라 나무를 더 추가하면 분산이 평균화되는 효과가 크다. 단일 부스팅처럼 기반 모델 추가 자체가 곧 과적합 위험을 크게 키우는 성격과는 다르다.

### Q8. 과적합 완화와 과소적합 완화

Dropout, smoothing, bagging은 과적합이나 분산을 줄이는 쪽에 가깝다. Kernel trick은 선형 경계로 부족한 모델 표현력을 키워 과소적합을 완화하는 데 쓰일 수 있다.

### Q9. GAN

생성자와 판별자가 경쟁하며 학습하는 구조는 GAN이다. 생성자는 가짜 데이터를 만들고, 판별자는 진짜와 가짜를 구분한다. 두 모델의 경쟁이 생성 품질을 높이는 방향으로 작동한다.

### Q10. Denoising Autoencoder

Denoising autoencoder는 손상된 입력에서 깨끗한 입력을 복원하도록 학습한다. PCA로 차원을 줄였다가 다시 복원할 때 노이즈 성분이 줄어드는 직관과 연결해서 이해할 수 있다.

### Q11. Grid Search

정해 둔 하이퍼파라미터 후보의 모든 조합을 평가하는 방법은 Grid Search다. Random Search는 후보 공간에서 일부 조합을 무작위로 뽑아 평가한다.

### Q12. SVD와 PCA

중심화한 입력행렬의 공분산행렬은 `X_c^T X_c / (n - 1)`로 쓸 수 있다. SVD를 적용하면 오른쪽 특이벡터가 PCA의 주성분 방향과 연결되고, 특이값 제곱이 분산 크기와 연결된다.

### Q13. 전진 선택과 후진 제거

전진 선택은 변수를 추가하고, 후진 제거는 변수를 제거한다. 단계적 선택은 둘을 섞을 수 있지만 성능 향상을 보장하지 않는다. 필터 방법보다 느리지만 모델 안에서 변수 조합 효과를 볼 수 있다.

### Q14. 신경망 파라미터 수

입력 2개에서 은닉 뉴런 4개로 가면 가중치가 `2 x 4 = 8`, 편향이 4개다. 은닉 4개에서 출력 2개로 가면 가중치가 `4 x 2 = 8`, 편향이 2개다. 전체는 가중치 16개, 편향 6개다.

### Q15. Gradient Boosting 손실

이진분류 손실을 써도 Gradient Boosting의 각 기반 모델은 손실의 음의 기울기 또는 잔차 방향을 맞추는 회귀 나무로 이해한다. 기반 모델이 직접 클래스 분류기라고 단정하면 부정확하다.

### Q16. 비선형 결정경계

단층 퍼셉트론은 선형 경계만 만들고, 활성화가 없는 다층 퍼셉트론도 결국 선형 변환의 합성이다. RBF 커널 SVM은 비선형 경계를 만들 수 있고, 1-NN도 데이터 배치에 따라 매우 구불구불한 비선형 경계를 만들 수 있다.

### Q17. 선형회귀 적합성 진단

VIF는 다중공선성, Shapiro-Wilk는 잔차 정규성, Bartlett은 집단별 잔차 등분산성 확인에 연결된다. 예측값과 대상변수의 QQ plot은 회귀 가정 진단의 핵심 도구로 보기 어렵다.

### Q18. 모분산 미지 평균 검정

정규분포 표본이고 모분산을 모르면 표본분산으로 표준오차를 추정하므로 t분포를 쓴다. 자유도는 표본 수 12에서 1을 뺀 11이다. 표본평균의 표준오차는 표본표준편차를 `sqrt(n)`으로 나눈 형태다.

### Q19. EM

잠재변수가 있고, 현재 모수로 잠재변수의 사후분포를 계산한 뒤 완전데이터 로그우도의 기대값을 최대화해 모수를 갱신하는 방법은 EM이다. GMM에서 각 표본의 성분 소속을 잠재변수로 두는 예가 대표적이다.

### Q20. 선형 보간

8시 18도, 11시 24도라면 3시간 동안 6도 상승했다. 9시는 8시에서 1시간 뒤이므로 `18 + 6 x (1/3) = 20`도다. 시간 간격을 분모로 두고 결측 시점의 상대 위치를 곱한다.
