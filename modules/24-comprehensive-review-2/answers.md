# 24. 종합 문제 2 해설

## 1. one-to-many RNN

하나의 입력에서 여러 출력이 필요한 문제에 어울린다. 예를 들어 한 이미지에서 여러 위치나 객체 후보를 순차적으로 출력하는 구조를 생각할 수 있다.

## 2. 단일 연결과 완전 연결

단일 연결은 두 군집에서 가장 가까운 두 점의 거리를 군집 간 거리로 본다. 완전 연결은 가장 먼 두 점의 거리를 본다. 단일 연결은 길게 이어지는 군집이 생기기 쉽고, 완전 연결은 더 조밀한 군집을 선호한다.

## 3. 최소제곱법

예측값과 실제값의 차이인 잔차를 제곱해 모두 더한 값을 최소화한다. 제곱을 쓰면 양수와 음수가 상쇄되지 않고 큰 오차에 더 큰 벌점이 붙는다.

## 4. 속성 선택 방식

Filter는 모델 학습 전 통계량이나 관계성 기준으로 변수를 고른다. Wrapper는 여러 변수 조합을 실제 모델 성능으로 평가한다. Embedded는 모델 학습 과정 안에서 변수 선택이 함께 일어난다.

## 5. ReLU와 기울기 소실

Sigmoid는 입력이 크거나 작을 때 기울기가 0에 가까워진다. ReLU는 양수 구간에서 기울기가 1이므로 깊은 층에서도 학습 신호가 상대적으로 잘 전달된다.

## 6. PCA와 전진 선택법

PCA는 원래 변수를 선형 결합해 새 주성분을 만든다. 전진 선택법은 기존 변수 중 성능을 개선하는 변수를 하나씩 추가한다. PCA는 압축, 전진 선택법은 변수 선택에 가깝다.

## 7. Dense layer 파라미터 수

입력 노드 수와 출력 뉴런 수를 곱하면 가중치 수가 된다. 여기에 출력 뉴런 수만큼 편향을 더한다. 공식은 `입력 수 x 출력 수 + 출력 수`다.

## 8. 포아송 분포 조건

일정 시간이나 공간 안에서 사건이 몇 번 일어나는지 세는 상황에 어울린다. 사건 발생률이 비교적 일정하고, 서로 독립적으로 발생한다고 볼 때 자주 사용한다.

## 9. 잔차 진단 이유

회귀모델의 가정이 맞는지 확인하기 위해서다. 잔차가 특정 패턴을 보이면 선형성, 등분산성, 독립성, 정규성 가정이 깨졌을 가능성이 있다.

## 10. 시계열 선형 보간 주의점

앞뒤 값 사이가 비교적 부드럽게 변한다는 가정이 필요하다. 갑작스러운 이벤트나 계절성이 강한 구간에서는 단순 직선 보간이 실제 변화를 왜곡할 수 있다.
