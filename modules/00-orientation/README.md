# 00. 학습 방법

## 목표

이 커리큘럼은 데이터 사이언스를 처음 공부하는 학습자를 위한 읽기 자료입니다. “코드를 실행했다”보다 “처음 보는 말을 쉬운 말로 바꿔 설명할 수 있다”를 목표로 둡니다.

## 권장 루틴

1. `오늘 반드시 이해할 3가지`를 먼저 읽습니다.
2. 본문에서 링크가 걸린 낯선 개념을 누르고 하단 `개념 주석`에서 쉬운 뜻을 확인합니다.
3. 주석에 `처음 설명된 차시`가 있으면 기억이 가물가물할 때만 원 설명으로 이동합니다.
4. 그림이 있으면 본문과 번갈아 봅니다.
5. `예시 풀이`를 먼저 보고 확인 문제를 풉니다.
6. 막히면 `answers.md`를 보고 다시 본문으로 돌아옵니다.

## 이론 노트 작성법

각 차시를 읽은 뒤 다음 다섯 줄을 남기면 충분합니다.

```text
오늘 처음 본 단어:
내 말로 바꾼 설명:
기억할 그림/예시:
조심할 오해:
다음 차시와 연결:
```
