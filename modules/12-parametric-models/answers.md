# 12. 모수적 모델 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 가중치 기반 모델의 학습을 설명하라.

모델은 입력값에 가중치를 곱하고 편향을 더해 예측값을 만든다. 예측값과 실제값의 차이로 손실을 계산하고, 손실이 줄어드는 방향으로 가중치를 수정한다. 이 과정을 반복하는 것이 학습이다.

## 2. 학습률이 너무 크거나 작을 때 생기는 문제를 설명하라.

학습률이 너무 크면 최저점을 지나쳐 손실이 튀거나 발산할 수 있다. 너무 작으면 안정적이지만 학습이 매우 느리거나 좋은 지점에 도달하기 어렵다. 그래서 보폭을 적절히 고르는 것이 중요하다.

## 3. L1 규제와 L2 규제의 차이를 설명하라.

L1 규제는 가중치 절댓값의 합을 벌점으로 주며 일부 가중치를 0으로 만들기 쉽다. L2 규제는 가중치 제곱합을 벌점으로 주며 전체 가중치를 부드럽게 줄인다. 그래서 L1은 변수 선택 효과, L2는 안정화 효과가 강하다.

## 4. 규제가 과대적합을 줄일 수 있는 이유를 설명하라.

과대적합 모델은 훈련 데이터에 맞추려고 가중치가 지나치게 커지는 경우가 많다. 규제는 큰 가중치에 벌점을 주어 모델을 덜 복잡하게 만든다. 그 결과 훈련 데이터의 우연한 흔들림을 덜 따라가게 된다.

## 5. 나이브 베이즈의 “나이브”가 뜻하는 바를 설명하라.

naive는 “순진한”이라는 뜻이다. 나이브 베이즈는 클래스가 주어졌을 때 특징들이 서로 독립이라고 단순하게 가정한다. 현실에서는 완벽히 맞지 않는 가정이지만 계산이 쉽고 빠르다는 장점이 있다.

## 6. 독립 가정이 완벽하지 않아도 나이브 베이즈가 쓰일 수 있는 이유를 설명하라.

독립 가정이 완벽하지 않아도 클래스별 상대적인 가능성 순서가 꽤 잘 맞을 수 있다. 특히 텍스트 분류처럼 특징이 많고 빠른 기준 모델이 필요한 경우 유용하다. 다만 확률값 자체는 과신하지 않아야 한다.

## 7. 왜 학습률이 너무 크면 손실이 줄지 않을 수 있는가?

- **한 문장 답**: 최적점을 향해 가다가도 보폭이 커서 지나치며 흔들리거나 발산할 수 있기 때문이다.
- **이유**: 경사하강법은 손실 표면을 조금씩 내려가야 하는데, 보폭이 너무 크면 낮은 지점을 지나쳐 반대편으로 튀며 안정적으로 수렴하지 못한다.
- **예시**: 내리막길에서 너무 큰 걸음으로 뛰면 계곡 바닥을 지나 반대편으로 넘어갈 수 있다.
- **주의점**: 작다고 항상 좋은 것도 아니며 너무 작으면 학습이 매우 느리다.

## 8. 왜 L1 규제는 변수 선택 효과가 생길 수 있는가?

- **한 문장 답**: 일부 가중치를 정확히 0으로 만들기 쉬운 형태의 벌점을 주기 때문이다.
- **이유**: L1 벌점은 0 근처에서도 일정한 압력으로 가중치를 줄인다. 최적해가 축 위에 놓이기 쉬워 일부 계수가 정확히 0이 된다.
- **예시**: 많은 단어 특징 중 일부만 남기고 나머지 가중치를 0으로 만들 수 있다.
- **주의점**: 해석에는 도움 되지만 필요한 약한 신호까지 지울 수 있다.

## 9. Ridge, Lasso, Elastic Net의 차이를 설명하라.

Ridge는 L2 벌점으로 가중치를 전체적으로 작게 만든다. Lasso는 L1 벌점으로 일부 가중치를 정확히 0으로 만들 수 있다. Elastic Net은 L1과 L2를 함께 써서 변수 선택 효과와 안정화 효과를 동시에 노린다.

## 10. Gaussian, Multinomial, Bernoulli Naive Bayes가 어울리는 데이터 차이를 설명하라.

Gaussian Naive Bayes는 연속형 특징이 클래스별 정규분포를 따른다고 볼 때 쓴다. Multinomial Naive Bayes는 문서의 단어 빈도처럼 카운트 데이터에 어울린다. Bernoulli Naive Bayes는 단어 등장 여부처럼 0/1 특징에 어울린다.

## 11. 나이브 베이즈에서 평활화가 필요한 이유를 설명하라.

어떤 클래스에서 특정 특징이 한 번도 관측되지 않으면 그 조건부 확률이 0이 되고, 곱셈 구조 때문에 전체 확률도 0이 될 수 있다. 평활화는 작은 가짜 카운트를 더해 이런 극단적 결론을 막는다. 특히 단어처럼 가능한 특징이 많은 데이터에서 중요하다.

## 12. 수식 기반 추정과 최적화 알고리즘 기반 추정의 차이를 설명하라.

수식 기반 추정은 정규방정식처럼 해를 직접 계산할 수 있는 방법이다. 같은 데이터와 같은 식이면 보통 같은 결과가 바로 나온다. 최적화 알고리즘 기반 추정은 손실을 줄이는 방향으로 모수를 반복해서 갱신한다. 모델이 복잡하거나 데이터가 커서 닫힌 해를 쓰기 어렵다면 경사하강법 같은 반복 최적화를 사용한다.

## 13. 경사하강법의 업데이트식과 에폭, 배치의 의미를 설명하라.

경사하강법은 `theta = theta - eta * gradient(J(theta))`처럼 목적함수의 기울기 반대 방향으로 모수를 움직인다. `eta`는 학습률로 한 번에 움직이는 보폭이다. 에폭은 전체 학습 데이터를 한 번 모두 사용해 학습을 끝낸 단위다. 배치는 큰 데이터를 한 번에 처리하지 않고 나누어 학습할 때의 데이터 묶음이다.

## 14. 나이브 베이즈에서 로그를 취해 계산하는 이유를 설명하라.

나이브 베이즈는 여러 조건부 확률을 곱해 클래스별 점수를 만든다. 특징 수가 많으면 작은 확률을 계속 곱하면서 값이 컴퓨터에서 0처럼 작아지는 underflow가 생길 수 있다. 로그를 취하면 곱셈이 덧셈으로 바뀌어 계산이 안정적이고, 로그는 단조 증가 함수라 가장 큰 클래스를 고르는 결과는 유지된다.

## 15. Laplace Smoothing과 Lidstone Smoothing, alpha의 편향-분산 효과를 설명하라.

평활화는 관측되지 않은 특징 때문에 확률이 0이 되는 것을 막기 위해 가짜 카운트를 더하는 방법이다. `alpha = 1`이면 Laplace Smoothing이고, 일반적으로 `alpha > 0`이면 Lidstone Smoothing이라고 부른다. alpha가 작으면 관측 데이터에 민감해져 분산이 커질 수 있고, alpha가 크면 확률을 더 평평하게 만들어 편향이 커질 수 있다.

## 16. 선형회귀 MSE 목적함수에서 `w`와 `b`의 경사하강법 기울기 식을 쓰라.

선형회귀 예측값을 `yhat_i = w^T x_i + b`로 두고, 목적함수를 MSE로 두면 다음과 같다.

```text
J = (1 / N) * sum_i (y_i - yhat_i)^2
```

기울기는 예측 오차 `y_i - yhat_i`를 이용해 계산한다.

```text
gradient_w J = (-2 / N) * sum_i x_i(y_i - yhat_i)
             = (-2 / N) * X^T(y - yhat)

dJ / db = (-2 / N) * sum_i (y_i - yhat_i)
```

업데이트는 `w = w - eta * gradient_w J`, `b = b - eta * dJ/db`이다.

## 17. Ridge, Lasso, Elastic Net의 손실 함수와 규제항을 각각 쓰라.

기본 손실을 `sum_i (y_i - yhat_i)^2`로 두면 Ridge는 L2 규제를 더한다.

```text
Ridge loss = sum_i (y_i - yhat_i)^2 + lambda * sum_j beta_j^2
```

Lasso는 L1 규제를 더한다.

```text
Lasso loss = sum_i (y_i - yhat_i)^2 + alpha * sum_j |beta_j|
```

Elastic Net은 L1과 L2를 함께 쓴다.

```text
Elastic Net loss = sum_i (y_i - yhat_i)^2
                   + alpha * sum_j |beta_j|
                   + lambda * sum_j beta_j^2
```

Ridge는 큰 계수를 부드럽게 줄이고, Lasso는 일부 계수를 0으로 만들 수 있으며, Elastic Net은 두 효과를 섞는다.

## 18. 나이브 베이즈의 베이즈 정리 유도, 조건부 독립 가정, 로그 점수 식을 순서대로 설명하라.

베이즈 정리에서 시작하면 다음과 같다.

```text
P(y = c | X1 = x1, ..., Xk = xk)
= P(X1 = x1, ..., Xk = xk | y = c)P(y = c)
  / P(X1 = x1, ..., Xk = xk)
```

나이브 베이즈는 클래스가 주어졌을 때 입력 특성이 서로 독립이라고 가정한다.

```text
P(X1 = x1, ..., Xk = xk | y = c)
= product_i P(X_i = x_i | y = c)
```

분모는 클래스별 비교에서 같으므로 생략하고, 가장 큰 값을 주는 클래스를 고른다.

```text
yhat = argmax_c P(y = c) * product_i P(X_i = x_i | y = c)
```

underflow를 막기 위해 로그를 취하면 곱셈이 덧셈으로 바뀐다.

```text
yhat = argmax_c [log P(y = c) + sum_i log P(X_i = x_i | y = c)]
```

## 19. Bernoulli NB와 Multinomial NB의 평활화 공식을 쓰고 `alpha`가 하는 일을 설명하라.

Bernoulli NB는 0/1 특징의 등장 확률을 추정한다.

```text
p = (sum_i x_i + alpha) / (n + 2alpha)
P(X_i = x_i | y = c) = p^x_i(1 - p)^(1 - x_i)
```

Multinomial NB는 카운트 특징의 범주별 확률을 추정한다.

```text
p_j = (sum_i x_ij + alpha) / (sum_i sum_l x_il + k alpha)
```

`alpha`는 관측되지 않은 특징의 확률이 0이 되는 것을 막는 가짜 카운트다. `alpha = 1`이면 Laplace Smoothing이고, `alpha > 0`인 일반형은 Lidstone Smoothing이다. alpha가 작으면 데이터에 민감하고, alpha가 크면 확률을 더 평평하게 만든다.
