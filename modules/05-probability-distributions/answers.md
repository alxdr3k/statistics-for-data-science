# 05. 확률 분포 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 수식 또는 조건 → 쉬운 해석 → 주의점** 순서로 확장한다.

## 1. 확률변수에서 대문자 `X`와 소문자 `x`가 어떻게 다른지 설명하라.

대문자 `X`는 아직 어떤 값이 나올지 정해지지 않은 확률변수이고, 소문자 `x`는 실제로 관측된 특정 값이다. 예를 들어 `P(X=x)`는 “확률변수 `X`가 특정 값 `x`로 관측될 확률”이라는 뜻이다.

## 2. 두 개의 주사위 합 `X`의 분포에서 7의 확률이 2의 확률보다 큰 이유를 설명하라.

두 주사위 합이 2가 되는 경우는 `(1,1)` 하나뿐이지만, 7이 되는 경우는 `(1,6)`, `(2,5)`, `(3,4)`, `(4,3)`, `(5,2)`, `(6,1)`처럼 여섯 가지다. 각 주사위 조합의 가능성이 같다면 경우의 수가 많은 합일수록 확률도 크다.

## 3. 이산 확률분포의 확률질량함수 조건 `sum_x f(x)=1`의 의미를 설명하라.

`f(x)=P(X=x)`는 각 점의 확률이다. 가능한 모든 값 `x`의 확률을 전부 더하면 표본공간 전체를 덮으므로 합이 1이어야 한다. 만약 합이 1보다 작거나 크면 확률분포로 볼 수 없다.

## 4. 연속 확률분포에서 `P(a <= X <= b)`를 왜 적분으로 계산하는지 설명하라.

연속값은 가능한 점이 무한히 촘촘해서 한 점의 확률은 0으로 본다. 그래서 확률밀도함수의 높이를 직접 확률로 읽지 않고, `a`부터 `b`까지 곡선 아래 면적을 적분해 구간확률을 계산한다.

```text
P(a <= X <= b) = integral from a to b of f(x) dx
```

## 5. 기대값의 선형성 `E[aX+b]=aE[X]+b`를 쉬운 말로 설명하라.

확률변수의 모든 값에 `a`를 곱하면 평균도 `a`배가 되고, 모든 값에 `b`를 더하면 평균도 `b`만큼 이동한다는 뜻이다. 예를 들어 점수 전체에 10점을 더하면 평균도 10점 올라간다.

## 6. `E[X+Y]=E[X]+E[Y]`와 `E[XY]=E[X]E[Y]`의 조건 차이를 설명하라.

합의 기대값 `E[X+Y]=E[X]+E[Y]`는 독립이 아니어도 성립한다. 하지만 곱의 기대값 `E[XY]=E[X]E[Y]`는 일반적으로 성립하지 않고, `X`와 `Y`가 독립일 때 성립한다. 곱은 두 변수가 함께 움직이는 정도의 영향을 받기 때문이다.

## 7. 분산 공식 `Var(X)=E[X^2]-E[X]^2`가 무엇을 계산하는지 설명하라.

분산은 기대값을 기준으로 값들이 얼마나 퍼져 있는지 계산한다. 원래 정의는 `Var(X)=E[(X-mu)^2]`이고, 이를 전개하면 `E[X^2]-E[X]^2`가 된다. 계산할 때는 전개된 식이 더 편한 경우가 많다.

## 8. 공분산이 `Var(X+Y)`에 들어가는 이유를 설명하라.

`X+Y`의 퍼짐은 `X`의 퍼짐과 `Y`의 퍼짐만으로 결정되지 않는다. 두 변수가 같은 방향으로 같이 움직이면 합의 변동이 더 커지고, 반대로 움직이면 줄어든다. 그래서 `Var(X+Y)=Var(X)+Var(Y)+2Cov(X,Y)`가 된다.

## 9. `X ~ D(theta)` 표기에서 `X`, `D`, `theta`가 각각 무엇을 뜻하는지 설명하라.

`X`는 확률변수, `D`는 분포의 이름, `theta`는 분포의 모양을 정하는 모수다. 전체 표기는 “확률변수 `X`가 모수 `theta`를 가진 분포 `D`를 따른다”는 뜻이다.

## 10. 베르누이분포와 이항분포의 관계를 설명하라.

베르누이분포는 성공/실패 한 번의 결과를 나타낸다. 이항분포는 독립적인 베르누이 시행을 `n`번 반복했을 때 성공 횟수의 분포다. 그래서 이항분포의 기대값은 `np`, 분산은 `np(1-p)`이다.

## 11. 기하분포와 음이항분포가 각각 어떤 질문에 답하는 분포인지 설명하라.

기하분포는 “첫 성공이 나올 때까지 몇 번 시행하는가?”를 묻는다. 음이항분포는 “목표 성공 수 `r`번을 달성하기 전까지 실패가 몇 번 나오는가?”를 묻는다. 둘 다 성공확률 `p`인 독립 베르누이 시행을 바탕으로 한다.

## 12. 포아송분포의 PMF, 기대값, 분산을 쓰고 의미를 설명하라.

포아송분포는 한 단위 안에서 사건이 몇 번 발생하는지 나타내는 이산분포다.

```text
P(X=x) = lambda^x e^(-lambda) / x!
E[X] = lambda
Var(X) = lambda
```

`lambda`는 한 단위에서 평균적으로 발생하는 사건 수다. 포아송분포에서는 평균과 분산이 모두 `lambda`라는 점이 중요하다.

## 13. `n=1000`, `p=0.02`인 이항분포를 포아송분포로 근사할 때 `lambda`를 구하라.

포아송 근사에서는 `lambda=np`를 쓴다.

```text
lambda = 1000 * 0.02 = 20
```

따라서 `Bin(1000,0.02)`는 대략 `Pois(20)`으로 볼 수 있다.

## 14. 정규분포, 지수분포, 감마분포, 균등분포가 각각 어떤 연속값을 다루는지 설명하라.

정규분포는 평균 주변에 좌우대칭으로 모이는 연속값을 다룬다. 지수분포는 다음 사건까지의 대기시간을 다룬다. 감마분포는 여러 번째 사건이 발생할 때까지의 시간을 다룬다. 균등분포는 구간 안의 값들이 같은 밀도를 갖는 상황을 다룬다.

## 15. 지수분포의 무기억성을 수식과 말로 설명하라.

무기억성은 이미 기다린 시간이 앞으로의 대기시간 분포를 바꾸지 않는다는 뜻이다.

```text
P(X > s+t | X > s) = P(X > t)
```

예를 들어 평균 대기시간이 10분인 지수분포라면, 이미 5분 기다렸더라도 앞으로의 기대 대기시간은 5분이 아니라 다시 10분으로 본다.

## 16. 카이제곱분포가 표준정규변수의 제곱합과 어떻게 연결되는지 설명하라.

서로 독립인 표준정규변수 `Z1, ..., Zk`가 있을 때, 제곱합 `Z1^2 + ... + Zk^2`는 자유도 `k`인 카이제곱분포를 따른다. 그래서 카이제곱분포는 분산처럼 제곱된 흔들림을 다룰 때 등장한다.

## 17. 정규모집단에서 `(n-1)S^2/sigma^2`가 어떤 분포를 따르는지 설명하라.

정규모집단에서 표본분산 `S^2`을 모분산 `sigma^2`로 표준화하면 다음이 성립한다.

```text
(n - 1)S^2 / sigma^2 ~ chi-square(n - 1)
```

자유도가 `n-1`인 이유는 표본분산을 계산할 때 표본평균 `Xbar`를 먼저 추정해 자유도 하나를 사용하기 때문이다.

## 18. 모분산을 알 때와 모를 때 표본평균 표준화가 어떻게 달라지는지 설명하라.

모분산 `sigma^2`을 알면 표본평균은 표준정규분포로 표준화한다.

```text
(Xbar - mu) / (sigma / sqrt(n)) ~ N(0,1)
```

모분산을 모르면 `sigma` 대신 표본표준편차 `S`를 쓰므로 추가 불확실성이 생기고, t분포를 사용한다.

```text
(Xbar - mu) / (S / sqrt(n)) ~ t(n - 1)
```

## 19. t분포의 자유도가 커질수록 표준정규분포에 가까워지는 이유를 설명하라.

자유도가 커진다는 것은 보통 표본 수가 커진다는 뜻이다. 표본 수가 커지면 표본표준편차 `S`가 모표준편차 `sigma`에 가까워져 분모의 불확실성이 줄어든다. 그래서 t분포의 두꺼운 꼬리가 점점 가늘어지고 표준정규분포에 가까워진다.

## 20. F분포가 두 분산의 비율 비교에서 등장하는 이유를 설명하라.

정규모집단의 표본분산은 카이제곱분포와 연결된다. 두 표본분산을 각각 모분산과 자유도로 표준화한 뒤 비율을 만들면 F분포가 된다.

```text
F = (U/d1) / (V/d2),  U ~ chi-square(d1), V ~ chi-square(d2)
```

그래서 두 분산의 비교나 분산분석의 평균제곱 비율 비교에서 F분포가 등장한다.

## 21. 적률의 1차, 2차, 3차, 4차가 각각 어떤 정보와 연결되는지 설명하라.

1차 적률 `E[X]`는 평균, 즉 중심 위치와 연결된다. 2차 적률 `E[X^2]`는 분산 계산에 사용된다. 3차 적률은 왜도, 4차 적률은 첨도 계산에 사용된다. 적률은 분포의 모양을 숫자로 요약하는 방법이다.

## 22. 적률생성함수 `M_X(t)=E[e^(tX)]`를 미분하면 왜 적률이 나오는지 설명하라.

`e^(tX)`를 전개하면 `1 + tX + t^2X^2/2! + ...`처럼 `X`, `X^2`, `X^3` 항이 차례로 나온다. 기대값을 취하면 `E[X]`, `E[X^2]`, `E[X^3]`이 계수로 들어간다. 그래서 `M_X(t)`를 미분하고 `t=0`을 넣으면 해당 차수의 적률이 나온다.

## 23. 포아송분포의 적률생성함수로 `E[X]`와 `Var(X)`를 구하는 흐름을 설명하라.

포아송분포의 적률생성함수는 다음과 같다.

```text
M_X(t) = exp(lambda(e^t - 1))
```

한 번 미분하면 `M'_X(0)=lambda`이므로 `E[X]=lambda`다. 두 번 미분하면 `M''_X(0)=lambda+lambda^2`이므로 `E[X^2]=lambda+lambda^2`다. 따라서

```text
Var(X) = E[X^2] - E[X]^2
       = (lambda + lambda^2) - lambda^2
       = lambda
```

## 24. 중심극한정리가 원자료가 아니라 표본평균에 대한 말이라는 점을 설명하라.

중심극한정리는 독립이고 같은 분포를 따르는 확률변수들의 평균 `Xbar`가 표본 수가 커질수록 정규분포에 가까워진다는 말이다. 원래 데이터의 분포 자체가 정규분포로 변한다는 뜻이 아니다.

```text
Xbar approximately N(mu, sigma^2/n)
```

## 25. 다음 설명 중 틀린 것을 고르고 이유를 설명하라: “포아송분포의 기대값은 `lambda`, 분산은 `lambda^2`이다.”

틀린 부분은 분산이다. 포아송분포의 기대값과 분산은 모두 `lambda`다.

```text
E[X] = lambda
Var(X) = lambda
```

`lambda^2`는 포아송분포의 분산이 아니다.

## 26. 다음 설명 중 틀린 것을 고르고 이유를 설명하라: “t분포는 자유도가 커질수록 양쪽 꼬리가 두꺼워진다.”

틀렸다. t분포는 자유도가 커질수록 표준정규분포에 가까워지고 양쪽 꼬리는 가늘어진다. 표본 수가 늘어 표본분산 추정의 불확실성이 줄어들기 때문이다.

## 27. 다음 설명 중 틀린 것을 고르고 이유를 설명하라: “버스 대기시간이 지수분포를 따르고 평균 10분에 한 대가 온다면, 이미 5분 기다린 뒤의 기대 대기시간은 5분이다.”

틀렸다. 지수분포는 무기억성을 가지므로 이미 기다린 시간은 앞으로의 대기시간 분포를 바꾸지 않는다. 평균 대기시간이 10분이라면 이미 5분 기다렸더라도 앞으로의 기대 대기시간은 10분으로 본다.
