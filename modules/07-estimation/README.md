# 07. 점 추정과 구간 추정

## 학습 목표

이 차시를 마치면 다음을 쉬운 말로 설명할 수 있으면 충분하다.

- 점 추정과 구간 추정의 차이를 설명한다.
- 모평균, 모분산, 모비율 추정을 각각 구분한다.
- 좋은 추정량의 기준인 불편성, 효율성, 일관성, 충분성을 이해한다.
- 표준오차와 오차한계가 신뢰구간 폭을 어떻게 정하는지 설명한다.
- 신뢰구간은 모수가 들어갈 절차의 신뢰도이지 특정 구간의 확률이 아님을 이해한다.

## 오늘의 한 줄

추정은 표본으로 모집단의 모수를 맞히되, 맞히는 방식의 불확실성을 함께 말하는 일이다.

## 오늘 반드시 이해할 3가지

1. 점 추정과 구간 추정의 차이를 설명한다.
2. 모평균, 모분산, 모비율은 점 추정량과 구간 추정 기준 분포가 다르다.
3. 신뢰구간은 모수가 들어갈 절차의 신뢰도이지 특정 구간의 확률이 아님을 이해한다.

## 이 차시 전에 알면 좋은 것

- **표본**: 표본으로 모수를 추정한다는 전제
- **분포**: 추정량의 흔들림을 말하는 기준
- **가설 검정**: 검정과 추정이 같은 불확실성을 다루는 방식임을 이해
- **표준오차**: 표본 통계량이 얼마나 흔들리는지 나타내는 크기

## 학습 우선순위

- **필수**: 점 추정과 구간 추정 차이, 모평균/모분산/모비율 추정 구분, 신뢰구간의 반복 절차 해석
- **심화**: 최대우도법과 적률법의 차이

## 이 차시에서 꼭 붙잡을 설명 방식

<a id="ref-07-신뢰구간"></a>[신뢰구간](#note-07-신뢰구간) 95%는 “이번에 계산한 구간 안에 모수가 95% 확률로 있다”는 뜻이 아니다. 모수는 고정되어 있고 구간이 표본마다 달라진다. 같은 방법으로 구간을 아주 많이 만들면 그중 약 95%가 참 모수를 포함한다는 절차의 성질이다.

신뢰구간은 보통 다음 구조로 읽는다.

```text
추정값 ± 기준값 * 표준오차
```

`기준값 * 표준오차`를 오차한계라고 생각하면 된다. 표준오차가 커지면 표본 통계량이 많이 흔들린다는 뜻이므로 구간이 넓어진다. 표본 수가 커지면 표준오차가 줄어 구간이 좁아지는 경향이 있다. 그래서 신뢰구간 폭은 “표본이 얼마나 불확실한가”를 보여 주는 숫자다.

## 핵심 이론

### 먼저 잡는 직관

- **점 추정과 구간 추정**: 점 추정은 하나의 숫자로 말하고, 구간 추정은 표본이 흔들린다는 사실까지 함께 말한다.
- **좋은 추정량의 기준**: 좋은 추정량은 평균적으로 빗나가지 않고, 덜 흔들리며, 표본이 많아질수록 참값에 가까워지고, 데이터가 가진 모수 정보를 잘 담는다.
- **적률법과 최대우도법**: 적률법은 평균 같은 요약값을 맞추고, 최대우도법은 지금 데이터가 가장 그럴듯해지는 모수를 고른다.
- **대표 신뢰구간**: 분산을 아는지 모르는지, 평균인지 비율인지에 따라 쓰는 분포와 구간 공식이 달라진다.

### 1. 점 추정과 구간 추정

점 추정은 하나의 숫자로 모수를 추정한다. 구간 추정은 불확실성을 인정하고 가능한 범위를 함께 제시한다.

점 추정값은 표본에서 나온 결과이고, 참 모수는 직접 보이지 않는다. 표본평균 `xbar=170`이라고 해서 모평균 `mu`가 정확히 170이라는 뜻은 아니다. 추정은 `xbar`를 증거로 `mu`를 말하는 과정이고, 구간 추정은 그 말에 표본오차의 크기를 붙이는 방식이다.

### 2. 좋은 추정량의 기준

불편성은 반복 <a id="ref-07-평균"></a>[평균](#note-07-평균)이 참값에 맞는 성질이고, 효율성은 분산이 작은 성질이다. 일관성은 <a id="ref-07-표본"></a>[표본](#note-07-표본) 수가 커질수록 참값에 가까워지는 성질이다. 충분성은 추정량이 표본 안의 모수 관련 정보를 충분히 담아, 원자료를 더 봐도 모수에 대해 추가로 얻을 정보가 거의 없게 만드는 성질이다.

모수 `theta`를 추정량 `theta_hat`으로 추정한다고 하자. 불편성은 `E[theta_hat] = theta`로 쓴다. 같은 표본추출을 반복했을 때 추정량의 평균이 참 모수와 같다는 뜻이다. 효율성은 여러 불편 추정량 중 분산이 가장 작은 것을 고르는 기준으로, `theta_hat* = argmin Var(theta_hat_i)`처럼 표현한다. 일치성은 표본 수 `n`이 커질수록 `theta_hat`이 `theta`에 가까워지는 성질이고, 충분성은 모수에 관한 정보를 한 통계량이 충분히 담는 성질이다.

![추정량의 선택 기준](assets/estimator-criteria-184.png)

> **그림 읽기**: 좋은 추정량을 편향과 흔들림 관점에서 비교한다. 평균적으로 맞는지와 얼마나 안정적인지를 나누어 본다.

표본평균이 모평균의 불편 추정량이라는 사실은 기대값의 선형성으로 바로 확인한다.

```text
Xbar = (X1 + X2 + ... + Xn) / n
E[Xbar] = E[(X1 + X2 + ... + Xn) / n]
        = (E[X1] + E[X2] + ... + E[Xn]) / n
        = (mu + mu + ... + mu) / n
        = mu
```

표본분산은 `n`이 아니라 `n - 1`로 나눌 때 모분산의 불편 추정량이 된다.

```text
S^2 = sum((Xi - Xbar)^2) / (n - 1)
sum((Xi - Xbar)^2) = sum((Xi - mu)^2) - n(Xbar - mu)^2
E[sum((Xi - Xbar)^2)] = n sigma^2 - n Var(Xbar)
                      = n sigma^2 - n(sigma^2 / n)
                      = (n - 1)sigma^2
E[S^2] = sigma^2
```

이 유도는 “표본평균을 이미 데이터에서 추정했기 때문에 자유도 하나를 썼다”는 말의 수식 버전이다. 그래서 분산을 불편하게 추정하려면 `n - 1`로 나눈다.

### 3. 적률법과 최대우도법

적률법은 표본 평균 같은 요약량을 이론값과 맞춘다. 최대우도법은 관측된 데이터가 가장 그럴듯해지는 모수를 찾는다.

적률 방법은 “이론적 적률 = 표본 적률”이라고 놓고 모수의 수만큼 방정식을 세운다. `k`차 표본 적률은 보통 `mk = (1 / n) * sum(xi^k)`로 쓴다. 포아송분포에서는 1차 적률 `E[X] = lambda`와 표본평균을 맞추면 `lambda_hat = m1`이 된다. 2차 적률은 `E[X^2] = lambda^2 + lambda`이므로 `m2 = lambda^2 + lambda`를 풀어 `lambda_hat = (-1 + sqrt(1 + 4m2)) / 2`도 만들 수 있다. 다만 2차 적률만 쓰는 추정량은 1차 적률 추정량보다 효율성이 떨어질 수 있어 보통 `lambda_hat = m1`을 먼저 쓴다.

감마분포처럼 모수가 두 개인 경우에는 1차 적률과 2차 적률을 함께 쓴다. 여기서는 감마분포를 rate 모수 `beta`로 표기하면 `E[X] = alpha / beta`, `E[X^2] = alpha(alpha + 1) / beta^2`이다. 이를 1차 표본 적률 `m1`, 2차 표본 적률 `m2`와 맞추면 다음처럼 푼다.

```text
alpha_hat / beta_hat = m1
alpha_hat(alpha_hat + 1) / beta_hat^2 = m2

beta_hat = m1 / (m2 - m1^2)
alpha_hat = m1^2 / (m2 - m1^2)
```

처음에는 공식을 외우기보다 “모수가 두 개라서 적률 방정식도 두 개가 필요하다”는 구조를 잡으면 된다.

![최대우도법의 직관](assets/maximum-likelihood-187.png)

> **그림 읽기**: 관측된 데이터가 가장 그럴듯해지는 모수를 고르는 생각을 본다. 확률을 거꾸로 읽어 파라미터를 찾는 과정이다.

최대우도법은 관측값을 고정하고 모수를 움직여 본다. 예를 들어 동전을 10번 던져 앞면이 7번 나왔다면 `X ~ Bin(n = 10, p)`이고, 우도는 `p^7(1 - p)^3`에 비례한다. 로그를 취해 미분하면 최대가 되는 값은 `p_hat = 0.7`이다. 확률을 “미래 결과의 가능성”으로만 보지 않고, 관측된 결과를 가장 잘 설명하는 모수를 찾는 관점이다.

```text
L(p | 7) = C(10, 7)p^7(1 - p)^3 ∝ p^7(1 - p)^3
log L(p) = 7log(p) + 3log(1 - p)
d log L(p) / dp = 7 / p - 3 / (1 - p)
7 / p - 3 / (1 - p) = 0
p_hat = 0.7
```

### 4. 점 추정량의 세 대상

점 추정량은 무엇을 추정하느냐에 따라 읽는 값이 달라진다. 모평균 추정에서는 표본평균이 중심 역할을 하고, 모분산 추정에서는 표본분산을 보되 `n`으로 나눌지 `n - 1`로 나눌지의 차이를 조심한다. 모비율 추정에서는 표본비율이 모집단의 성공 비율을 어림하는 값이 된다.

| 추정 대상 | 자주 쓰는 점 추정량 | 먼저 확인할 것 |
|---|---|---|
| 모평균 | `Xbar = (1 / n) * sum(xi)` | 표본이 독립인지, 극단값이 평균을 흔드는지 |
| 모분산 | `S^2 = sum((xi - Xbar)^2) / (n - 1)` | 평균에서 얼마나 퍼져 있는지, 자유도 보정이 필요한지 |
| 모비율 | `phat = (1 / n) * sum(xi)` (`xi`는 0 또는 1) | 성공/실패가 분명한지, 표본 수가 충분한지 |

### 5. 대표 신뢰구간

구간 추정도 모평균, 모비율, 모분산을 나누어 생각한다. 모평균은 모분산을 알면 표준정규분포를 쓰고, 모분산을 모르면 t분포를 쓴다. 모비율은 표본 수가 충분할 때 정규근사를 사용한다. 모분산은 표본분산을 카이제곱분포와 연결해 구간을 만든다.

| 구간 추정 대상 | 기준 분포 | 핵심 조건 |
|---|---|---|
| 모평균, 모분산을 아는 경우 | 표준정규분포 | `Xbar ± z(alpha/2) * sigma / sqrt(n)` |
| 모평균, 모분산을 모르는 경우 | t분포 | `Xbar ± t(alpha/2, n - 1) * S / sqrt(n)` |
| 모비율 | 정규근사 | `phat ± z(alpha/2) * sqrt(phat(1 - phat) / n)` |
| 모분산 | 카이제곱분포 | `(n - 1)S^2 / sigma^2`를 카이제곱분포와 연결 |

모평균 구간은 표본평균의 분포에서 출발한다. 표본이 서로 독립이고 `X ~ N(mu, sigma^2)`이면 `Xbar ~ N(mu, sigma^2 / n)`이다.

모분산 `sigma^2`를 알고 있을 때는 표준화한 값이 표준정규분포를 따른다.

```text
Z = (Xbar - mu) / (sigma / sqrt(n))
P(-z(alpha/2) <= Z <= z(alpha/2)) = 1 - alpha

Xbar - z(alpha/2) * sigma / sqrt(n)
<= mu <=
Xbar + z(alpha/2) * sigma / sqrt(n)

z(0.025) = Phi^-1(0.975) ≈ 1.96
```

모분산을 모르면 `sigma` 대신 표본표준편차 `S`를 넣는다. 이때는 표준정규분포가 아니라 자유도 `n - 1`인 t분포를 쓴다.

```text
t = (Xbar - mu) / (S / sqrt(n)) ~ t(n - 1)

Xbar - t(alpha/2, n - 1) * S / sqrt(n)
<= mu <=
Xbar + t(alpha/2, n - 1) * S / sqrt(n)
```

모비율은 각 관측값을 성공이면 1, 실패면 0인 베르누이 확률변수로 본다. 표본 수가 충분하고 표본이 독립이면 중심극한정리로 표본비율을 정규분포로 근사한다.

```text
X ~ Bern(p), X in {0, 1}
phat = (1 / n) * sum(xi)
E[X] = p
Var(X) = p(1 - p)
phat ≈ N(p, p(1 - p) / n)
phat ≈ N(phat, phat(1 - phat) / n)

phat - z(alpha/2) * sqrt(phat(1 - phat) / n)
<= p <=
phat + z(alpha/2) * sqrt(phat(1 - phat) / n)
```

정규분포의 모분산 구간은 표본분산과 카이제곱분포의 관계를 쓴다.

```text
(n - 1)S^2 / sigma^2 ~ chi-square(n - 1)

(n - 1)S^2 / chi-square(1 - alpha/2, n - 1)
<= sigma^2 <=
(n - 1)S^2 / chi-square(alpha/2, n - 1)
```

분산 구간은 평균 구간과 달리 좌우가 대칭으로 벌어지지 않는다. 카이제곱분포가 오른쪽으로 치우친 분포이기 때문이다.

![신뢰구간의 직관](assets/confidence-interval-191.png)

> **그림 읽기**: 점 하나가 아니라 가능한 범위를 만드는 절차를 본다. 반복했을 때 참값을 포함하는 비율이 신뢰수준의 의미다.

### 6. 추정량 선택과 구간 유도

추정량을 고를 때 불편성, 일치성, 효율성, 충분성을 함께 본다. 불편성은 평균적으로 참값을 맞히는 성질이고, 일치성은 표본 수가 커질수록 참값에 가까워지는 성질이다. 효율성은 같은 조건에서 더 작게 흔들리는 추정량을 선호한다는 뜻이다. 충분성은 표본의 모수 관련 정보를 한 통계량이 충분히 요약하는지 보는 기준이다.

적률 방법은 표본에서 계산한 평균이나 분산을 이론적 기대값과 맞춰 모수를 구한다. 최대우도법은 “이 표본이 관찰될 가능성을 가장 크게 만드는 모수”를 찾는다. 감마분포처럼 모수가 여러 개인 분포에서는 표본평균과 표본분산이 각각 어떤 모수 조합과 연결되는지 세워 풀 수 있다.

구간 추정에서는 추정 대상부터 분리해야 한다. 모평균 구간은 모분산을 아는지 모르는지가 중요하다. 모분산을 알면 표준정규분포를 기준으로 평균 구간을 만들 수 있지만, 보통은 모분산을 모른다. 이때 표본표준편차로 대신 추정하므로 추가 불확실성을 반영하는 t분포를 쓴다. 모비율은 표본비율의 표준오차를 기준으로 하고, 모분산은 카이제곱분포를 기준으로 구간을 만든다.

지수분포의 최대우도 추정은 대기 시간 문제에서 자주 나온다. `X1, ..., Xn`이 지수분포를 따른다고 보면, 발생률 모수는 보통 표본평균의 역수로 추정된다. 즉 `lambda_hat = n / sum(x_i) = 1 / Xbar`이다. 대기 시간이 길수록 발생률 추정값은 작아지고, 대기 시간이 짧을수록 발생률 추정값은 커진다. 평균 대기 시간 모수를 `theta = 1 / lambda`로 쓰면 `theta_hat = Xbar`이다.

## 판단 기준

1. 숫자 하나가 필요한 문제인지, 불확실한 범위까지 필요한 문제인지 구분한다.
2. <a id="ref-07-추정량"></a>[추정량](#note-07-추정량)이 반복 평균에서 참값에 맞는지, 흔들림은 작은지 함께 본다.
3. 모분산을 아는지 모르는지에 따라 z분포와 t분포를 구분한다.
4. 신뢰수준은 “이번 구간의 확률”이 아니라 반복 절차의 장기 포함률로 해석한다.
5. 표본 수가 작거나 <a id="ref-07-분포"></a>[분포](#note-07-분포)가 심하게 치우친 경우 구간 해석을 조심한다.

## 오해와 반례

### 오해 1. 신뢰구간 95%는 이번 구간에 모수가 있을 확률 95%다.

모수는 고정되어 있고 구간이 표본마다 달라진다. 95%는 같은 절차를 반복했을 때의 장기 포함률이다.

### 오해 2. 점 추정값 하나면 충분하다.

표본은 흔들리므로 하나의 숫자만 말하면 불확실성이 사라져 보인다. 구간을 함께 봐야 한다.

### 오해 3. 불편 추정량이면 항상 가장 좋다.

불편성이 좋아도 분산이 크면 실제 <a id="ref-07-추정"></a>[추정](#note-07-추정)은 많이 흔들릴 수 있다. 효율성도 함께 봐야 한다.

## 예시 풀이

### 예시 1. 평균 키를 하나의 숫자로 말할 때

표본 평균 170cm는 점 추정이다. 하지만 표본을 다시 뽑으면 169cm나 171cm가 나올 수 있으므로, 168~172cm처럼 구간을 함께 제시하면 불확실성을 더 정직하게 표현한다.

### 예시 2. 모분산을 모를 때 t분포를 쓰는 이유

모분산을 모르기 때문에 표본표준편차로 대신한다. 이 추가 불확실성 때문에 표준정규분포보다 꼬리가 두꺼운 t분포를 사용한다.

## 오늘의 요약 5줄

1. 추정은 표본으로 모집단 모수를 어림하되 그 불확실성까지 표현하는 일이다.
2. 점 추정은 숫자 하나, 구간 추정은 모수가 있을 만한 범위를 제시한다.
3. 불편성, 효율성, 일관성, 충분성은 추정량을 고를 때 보는 대표 기준이다.
4. 최대우도법은 관측 데이터가 가장 그럴듯해지는 모수를 찾는다.
5. 신뢰구간은 특정 구간의 확률이 아니라 같은 절차를 반복했을 때의 포함률이다.

## 확인 문제

1. 점 추정과 구간 추정의 차이를 설명하라.
2. 불편성이 좋은 성질인 이유를 설명하라.
3. 효율성이 의미하는 바를 설명하라.
4. 최대우도법의 직관을 설명하라.
5. 95% 신뢰구간의 올바른 해석을 설명하라.
6. 모분산을 모를 때 t분포를 쓰는 이유를 설명하라.
7. 왜 점 추정값 하나만 제시하면 부족할 수 있는가?
8. 왜 신뢰구간 95%를 “이번 구간에 참값이 있을 확률”로 해석하면 안 되는가?
9. 적률 방법과 최대우도법의 출발점 차이를 설명하라.
10. 모분산을 모를 때 평균 신뢰구간에 t분포를 쓰는 이유를 설명하라.
11. 모평균, 모분산, 모비율의 점 추정량을 각각 설명하라.
12. 모평균, 모비율, 모분산 구간 추정에서 기준 분포가 어떻게 달라지는지 설명하라.
13. 충분성이 추정량 선택 기준에서 의미하는 바를 설명하라.
14. 포아송분포와 감마분포의 적률 방법 예시가 어떻게 다른지 설명하라.
15. 동전 10번 중 앞면 7번의 최대우도 추정값이 왜 0.7인지 설명하라.
16. 지수분포 대기 시간 표본에서 발생률 모수를 최대우도로 추정하는 직관을 설명하라.
17. `theta_hat1 = (X1 + X2 + X3 + X4 + X5) / 5`, `theta_hat2 = (X1 + X2 + X3 + 2X4) / 5` 중 어떤 추정량이 불편 추정량인지, 어느 쪽이 더 효율적인지 설명하라.
18. 표본평균이 모평균의 불편 추정량임을 기대값으로 보이라.
19. 표본분산을 `n - 1`로 나누는 이유를 불편성 관점에서 설명하라.
20. 포아송분포에서 1차 적률과 2차 적률을 각각 쓰면 `lambda` 추정식이 어떻게 달라지는지 설명하라.
21. 감마분포에서 적률 방법으로 `alpha`, `beta`를 추정할 때 왜 식이 두 개 필요한지 설명하라.
22. 모평균, 모비율, 모분산 신뢰구간의 기준 분포와 공식을 각각 쓰라.
23. 동전 10번 중 앞면 7번인 경우 로그우도를 미분해 최대우도 추정값을 구하라.
24. 대기 시간 `3, 2, 5, 4, 1`이 지수분포를 따른다고 할 때 발생률 `lambda`와 평균 대기 시간 `theta`의 최대우도 추정값을 구하라.
25. 신뢰구간의 일반 구조 `추정값 ± 기준값 * 표준오차`에서 각 항이 무엇을 뜻하는지 설명하라.
26. 표본 수가 커질수록 평균 신뢰구간이 보통 좁아지는 이유를 표준오차로 설명하라.
27. 점 추정값과 참 모수를 같은 숫자로 단정하면 왜 위험한지 설명하라.
