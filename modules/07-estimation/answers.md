# 07. 점 추정과 구간 추정 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 점 추정과 구간 추정의 차이를 설명하라.

점 추정은 모수를 하나의 숫자로 어림하고, 구간 추정은 모수가 있을 만한 범위를 제시한다. 구간 추정은 표본의 흔들림을 함께 표현한다.

## 2. 불편성이 좋은 성질인 이유를 설명하라.

같은 방법으로 표본을 여러 번 뽑아 추정했을 때 평균적으로 참 모수에 맞기 때문이다. 한 번은 빗나갈 수 있지만 장기적으로 한쪽으로 치우치지 않는다.

## 3. 효율성이 의미하는 바를 설명하라.

여러 불편 추정량 중 분산이 더 작은 추정량이 효율적이다. 즉 추정값이 참값 주변에서 덜 흔들린다.

## 4. 최대우도법의 직관을 설명하라.

관측된 데이터가 나올 가능도를 가장 크게 만드는 모수를 찾는 방법이다. 데이터가 가장 그럴듯해지는 설명을 고르는 것이다.

## 5. 95% 신뢰구간의 올바른 해석을 설명하라.

같은 표본추출과 계산 절차를 많이 반복하면 만들어진 구간의 약 95%가 참 모수를 포함한다는 뜻이다.

## 6. 모분산을 모를 때 t분포를 쓰는 이유를 설명하라.

표본표준편차로 분산을 추정하면서 불확실성이 더 커진다. t분포는 이 추가 불확실성을 두꺼운 꼬리로 반영한다.

## 7. 왜 점 추정값 하나만 제시하면 부족할 수 있는가?

- **한 문장 답**: 표본은 매번 달라지므로 추정값도 흔들리기 때문이다.
- **이유**: 점 추정은 표본 하나에서 나온 대표값이라 표본을 다시 뽑으면 달라진다. 불확실성의 크기를 함께 보여 주지 않으면 숫자의 신뢰도를 판단하기 어렵다.
- **예시**: 평균 키를 170cm라고만 말하는 것보다 168~172cm 정도라고 말하면 불확실성이 드러난다.
- **주의점**: 구간이 넓으면 정보가 부족하다는 신호일 수 있다.

## 8. 왜 신뢰구간 95%를 “이번 구간에 참값이 있을 확률”로 해석하면 안 되는가?

- **한 문장 답**: 빈도주의 관점에서는 모수는 고정되어 있고 구간이 표본마다 달라지기 때문이다.
- **이유**: 모수는 확률적으로 움직이는 대상이 아니라 고정된 값으로 본다. 무작위인 것은 표본을 뽑을 때마다 달라지는 구간 생성 절차다.
- **예시**: 같은 절차를 100번 반복하면 약 95번 참값을 포함한다는 뜻이다.
- **주의점**: 이번에 계산된 하나의 구간만 놓고 확률을 붙이는 표현은 조심한다.

## 9. 적률 방법과 최대우도법의 출발점 차이를 설명하라.

적률 방법은 표본평균이나 표본분산 같은 표본 적률을 이론적 적률과 맞춰 모수를 푼다. 최대우도법은 관찰된 데이터가 나올 가능성, 즉 우도를 가장 크게 만드는 모수를 찾는다. 전자는 모양 요약값을 맞추는 방식이고, 후자는 관찰 데이터 전체의 가능성을 기준으로 삼는 방식이다.

## 10. 모분산을 모를 때 평균 신뢰구간에 t분포를 쓰는 이유를 설명하라.

모분산을 모르면 표본표준편차로 표준오차를 추정해야 한다. 이 추정 과정 때문에 불확실성이 더 커지고, 그 불확실성을 반영한 분포가 t분포다. 표본 수가 작을수록 t분포의 꼬리가 더 두껍다.

## 11. 모평균, 모분산, 모비율의 점 추정량을 각각 설명하라.

모평균은 표본평균으로 추정한다. 모분산은 표본분산으로 추정하되, 불편성을 맞추기 위해 보통 `n - 1`로 나눈 불편분산을 사용한다. 모비율은 표본에서 성공한 비율인 표본비율로 추정한다.

## 12. 모평균, 모비율, 모분산 구간 추정에서 기준 분포가 어떻게 달라지는지 설명하라.

모평균은 모분산을 알면 표준정규분포, 모분산을 모르면 t분포를 기준으로 삼는다. 모비율은 성공 수와 실패 수가 충분할 때 정규근사로 구간을 만든다. 모분산은 표본분산과 카이제곱분포의 관계를 이용해 구간을 만든다.

## 13. 충분성이 추정량 선택 기준에서 의미하는 바를 설명하라.

충분성은 추정량이 표본 안의 모수 관련 정보를 충분히 담고 있는지를 보는 기준이다. 충분한 추정량을 알고 나면 원자료를 더 보더라도 그 모수에 대해 추가로 얻을 정보가 거의 없다는 뜻이다. 초보자 관점에서는 “모수 판단에 필요한 정보를 잘 압축한 통계량인가”로 이해하면 된다.

## 14. 포아송분포와 감마분포의 적률 방법 예시가 어떻게 다른지 설명하라.

포아송분포는 모수가 `lambda` 하나라서 1차 적률 `E[X] = lambda`를 표본평균과 맞추면 바로 추정할 수 있다. 감마분포는 보통 `alpha`, `beta` 두 모수를 가지므로 1차 적률과 2차 적률을 함께 사용해 연립방정식을 세운다. 모수가 몇 개인지에 따라 필요한 적률 방정식 수가 달라진다.

## 15. 동전 10번 중 앞면 7번의 최대우도 추정값이 왜 0.7인지 설명하라.

앞면 확률을 `p`라고 하면 앞면이 7번, 뒷면이 3번 나올 우도는 상수를 제외하고 `p^7(1 - p)^3`에 비례한다. 이 값을 가장 크게 만드는 `p`를 찾으면 관측된 앞면 비율과 같은 `7/10 = 0.7`이 된다. 즉 최대우도법은 관측 결과를 가장 그럴듯하게 만드는 모수를 고른다.

## 16. 지수분포 대기 시간 표본에서 발생률 모수를 최대우도로 추정하는 직관을 설명하라.

지수분포에서 발생률이 크면 사건이 자주 일어나므로 평균 대기 시간이 짧아진다. 반대로 발생률이 작으면 평균 대기 시간이 길어진다. 독립 대기 시간 표본의 평균이 `Xbar`라면 최대우도 추정은 보통 `lambda_hat = 1 / Xbar`로 정리된다. 관측된 평균 대기 시간이 길수록 발생률 추정값은 작아진다.

## 17. `theta_hat1 = (X1 + X2 + X3 + X4 + X5) / 5`, `theta_hat2 = (X1 + X2 + X3 + 2X4) / 5` 중 어떤 추정량이 불편 추정량인지, 어느 쪽이 더 효율적인지 설명하라.

둘 다 불편 추정량이다. 각 `Xi`의 기대값이 `mu`이면 `theta_hat1`의 가중치 합은 `1`, `theta_hat2`의 가중치 합도 `1/5 + 1/5 + 1/5 + 2/5 = 1`이므로 기대값이 모두 `mu`가 된다.

효율성은 `theta_hat1`이 더 높다. 독립 표본의 분산이 모두 `sigma^2`라면 `Var(theta_hat1) = sigma^2 / 5`이고, `Var(theta_hat2) = (1^2 + 1^2 + 1^2 + 2^2)sigma^2 / 25 = 7sigma^2 / 25`이다. `sigma^2 / 5 = 5sigma^2 / 25`가 더 작으므로 `theta_hat1`이 덜 흔들린다.

## 18. 표본평균이 모평균의 불편 추정량임을 기대값으로 보이라.

표본평균은 `Xbar = (X1 + X2 + ... + Xn) / n`이다. 기대값의 선형성을 쓰면 `E[Xbar] = (E[X1] + E[X2] + ... + E[Xn]) / n`이다. 각 표본의 기대값이 `mu`이므로 `E[Xbar] = (mu + mu + ... + mu) / n = mu`가 된다. 따라서 표본평균은 모평균의 불편 추정량이다.

## 19. 표본분산을 `n - 1`로 나누는 이유를 불편성 관점에서 설명하라.

표본분산을 계산할 때는 참 평균 `mu`가 아니라 표본에서 계산한 `Xbar`를 쓴다. 이 과정에서 평균을 하나 추정했으므로 자유도 하나가 줄어든다. 실제로 `sum((Xi - Xbar)^2) = sum((Xi - mu)^2) - n(Xbar - mu)^2`이고, 기대값을 취하면 `(n - 1)sigma^2`가 된다. 따라서 `sum((Xi - Xbar)^2)`를 `n - 1`로 나누어야 기대값이 `sigma^2`가 된다.

## 20. 포아송분포에서 1차 적률과 2차 적률을 각각 쓰면 `lambda` 추정식이 어떻게 달라지는지 설명하라.

포아송분포는 `E[X] = lambda`이므로 1차 적률을 쓰면 `lambda_hat = m1 = (1 / n)sum(xi)`이다. 2차 적률은 `E[X^2] = lambda^2 + lambda`이므로 `m2 = lambda^2 + lambda`를 풀어 `lambda_hat = (-1 + sqrt(1 + 4m2)) / 2`가 된다. 둘 다 적률법이지만 1차 적률 추정량이 더 단순하고 보통 더 자연스럽게 쓰인다.

## 21. 감마분포에서 적률 방법으로 `alpha`, `beta`를 추정할 때 왜 식이 두 개 필요한지 설명하라.

감마분포를 `alpha`, `beta` 두 모수로 표현하면 미지수가 두 개다. 미지수가 두 개이면 하나의 식만으로는 값을 정할 수 없으므로 1차 적률과 2차 적률을 함께 맞춘다. `E[X] = alpha / beta`, `E[X^2] = alpha(alpha + 1) / beta^2`를 `m1`, `m2`와 같게 두면 `beta_hat = m1 / (m2 - m1^2)`, `alpha_hat = m1^2 / (m2 - m1^2)`로 풀 수 있다.

## 22. 모평균, 모비율, 모분산 신뢰구간의 기준 분포와 공식을 각각 쓰라.

모평균은 모분산을 알면 표준정규분포를 쓴다. 공식은 `Xbar ± z(alpha/2) * sigma / sqrt(n)`이다. 모분산을 모르면 t분포를 쓰고, 공식은 `Xbar ± t(alpha/2, n - 1) * S / sqrt(n)`이다.

모비율은 표본 수가 충분할 때 정규근사를 쓴다. 공식은 `phat ± z(alpha/2) * sqrt(phat(1 - phat) / n)`이다.

모분산은 카이제곱분포를 쓴다. 공식은 `(n - 1)S^2 / chi-square(1 - alpha/2, n - 1) <= sigma^2 <= (n - 1)S^2 / chi-square(alpha/2, n - 1)`이다.

## 23. 동전 10번 중 앞면 7번인 경우 로그우도를 미분해 최대우도 추정값을 구하라.

앞면 확률을 `p`라고 하면 우도는 상수를 제외하고 `L(p) ∝ p^7(1 - p)^3`이다. 로그우도는 `log L(p) = 7log(p) + 3log(1 - p)`이다. 미분하면 `7 / p - 3 / (1 - p)`이고, 이를 0으로 두면 `7(1 - p) = 3p`, 즉 `7 = 10p`가 된다. 따라서 `p_hat = 0.7`이다.

## 24. 대기 시간 `3, 2, 5, 4, 1`이 지수분포를 따른다고 할 때 발생률 `lambda`와 평균 대기 시간 `theta`의 최대우도 추정값을 구하라.

지수분포의 발생률 모수를 `lambda`라고 하면 독립 표본의 우도는 `lambda^n exp(-lambda sum(xi))` 꼴이다. 여기서는 `n = 5`, `sum(xi) = 15`이므로 로그우도는 `5log(lambda) - 15lambda`이다. 미분하면 `5 / lambda - 15 = 0`이고, 따라서 `lambda_hat = 1 / 3`이다. 평균 대기 시간을 `theta = 1 / lambda`로 쓰면 `theta_hat = 3`분이다.
