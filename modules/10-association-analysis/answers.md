# 10. 연관성 분석 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 모수적 검정과 비모수적 검정의 차이를 설명하라.

모수적 검정은 정규성이나 등분산성처럼 모집단 분포에 대한 가정을 비교적 강하게 둔다. 비모수적 검정은 이런 가정이 어렵거나 표본이 작을 때 순위나 빈도 같은 정보를 활용한다. 어느 쪽이든 데이터 형태와 가정을 먼저 확인해야 한다.

## 2. 피어슨 상관계수와 스피어만 상관계수의 차이를 설명하라.

피어슨 상관계수는 두 연속형 변수가 선형으로 함께 움직이는 정도를 본다. 스피어만 상관계수는 값의 순위를 이용하므로 단조 관계나 이상치에 조금 더 강하다. 직선 관계를 보는지, 순위 관계를 보는지가 핵심 차이다.

## 3. 상관관계를 인과관계로 해석하면 안 되는 이유를 설명하라.

두 변수가 함께 움직인다고 해서 하나가 다른 하나의 원인이라는 뜻은 아니다. 제3의 변수가 둘 모두에 영향을 줄 수도 있고, 우연히 함께 움직일 수도 있다. 인과를 말하려면 연구 설계나 추가 근거가 필요하다.

## 4. ANOVA가 유의할 때 사후검정이 필요한 이유를 설명하라.

ANOVA는 여러 집단 평균이 모두 같다는 가설을 한 번에 검정한다. 유의하다는 결과는 “어딘가 차이가 있다”는 뜻이지 어느 두 집단이 다른지는 알려 주지 않는다. 그래서 Tukey 같은 사후검정으로 차이가 나는 집단 쌍을 확인한다.

## 5. 카이제곱 독립성 검정의 직관을 설명하라.

두 범주형 변수가 독립이라면 각 셀에 기대되는 빈도가 있다. 실제 관측 빈도가 기대빈도와 많이 다르면 두 변수가 독립이라고 보기 어려워진다. 카이제곱 통계량은 그 차이를 셀별로 모아 판단한다.

## 6. 기대빈도가 너무 작을 때 생기는 문제를 설명하라.

기대빈도가 너무 작으면 카이제곱 근사가 부정확해질 수 있다. 그러면 p-value가 믿기 어려워진다. 범주를 합치거나 Fisher의 정확검정 같은 대안을 고려할 수 있다.

## 7. 왜 상관관계만으로 인과관계를 말할 수 없는가?

- **한 문장 답**: 두 변수가 함께 움직여도 제3의 원인이나 반대 방향 설명이 가능하기 때문이다.
- **이유**: 상관은 두 값이 함께 변한다는 관찰일 뿐 시간 순서, 개입, 숨은 변수 통제를 보장하지 않는다.
- **예시**: 아이스크림 판매와 익사 사고가 함께 늘어도 더운 날씨가 둘 다에 영향을 줄 수 있다.
- **주의점**: 시간 순서와 통제 설계 없이는 인과 표현을 피한다.

## 8. 왜 ANOVA가 유의해도 바로 어느 두 집단이 다른지는 알 수 없는가?

- **한 문장 답**: ANOVA는 모든 집단 평균이 같다는 전체 가설을 먼저 검정하기 때문이다.
- **이유**: ANOVA의 귀무가설은 모든 평균이 같다는 하나의 전체 주장이다. 기각되면 적어도 하나가 다르다는 뜻이지 어떤 쌍이 다른지는 추가 비교가 필요하다.
- **예시**: 세 반의 평균 점수가 다르다는 결론은 어느 반과 어느 반이 다른지까지 말하지 않는다.
- **주의점**: 구체적 쌍 비교는 사후검정이 필요하다.

## 9. Pearson, Spearman, Kendall 상관을 구분하는 기준을 설명하라.

Pearson 상관은 두 연속형 변수의 선형 관계를 본다. Spearman 상관은 값을 순위로 바꿔 단조 관계를 보므로 이상치와 비선형 단조 관계에 더 유연하다. Kendall 상관도 순위 기반이며 쌍의 순서 일치 정도를 기준으로 관계를 본다.

## 10. 카이제곱 적합도, 독립성, 동질성 검정의 차이를 설명하라.

적합도 검정은 하나의 범주형 변수 분포가 기대 비율과 맞는지 본다. 독립성 검정은 한 모집단에서 두 범주형 변수가 서로 관련 있는지 본다. 동질성 검정은 여러 집단의 범주 분포가 같은지 비교한다.

## 11. Shapiro-Wilk, Jarque-Bera, Kolmogorov-Smirnov 정규성 검정의 차이를 설명하라.

Shapiro-Wilk는 정렬된 표본과 정규분포 분위수 기반 가중치를 이용하며 소표본에서 자주 쓰인다. Jarque-Bera는 왜도와 첨도가 정규분포의 기준값인 왜도 0, 첨도 3에서 얼마나 벗어나는지 본다. Kolmogorov-Smirnov는 표본 누적분포함수와 비교 대상 누적분포함수의 최대 차이를 본다. 셋 다 정규성 점검이지만 보는 정보가 다르다.

## 12. Bartlett 검정과 Levene 검정을 등분산성 점검 관점에서 비교하라.

Bartlett 검정은 여러 모집단의 분산이 같은지 보는 등분산성 검정이며, 모집단이 정규분포를 따른다는 가정에 민감하다. 정규성 가정이 흔들리면 Bartlett 결과가 불안정할 수 있다. Levene 검정은 비모수적 또는 더 강건한 대안으로 생각할 수 있어, 정규성이 의심될 때 함께 고려한다.

## 13. Mann-Whitney U 검정과 Wilcoxon 부호순위 검정의 차이를 설명하라.

Mann-Whitney U 검정은 서로 독립인 두 집단의 위치 차이를 순위합으로 비교한다. Wilcoxon 부호순위 검정은 같은 대상의 전후 비교처럼 대응 관계가 있는 두 집단의 차이를 보고, 차이의 부호와 순위를 이용한다. 독립 표본인지 대응 표본인지가 선택 기준이다.

## 14. 이원분산분석 표에서 전체 제곱합, 회귀 제곱합, 오차 제곱합과 F값을 어떻게 연결하는지 설명하라.

전체 제곱합은 설명된 변동과 오차 변동으로 나뉘어 `SST = SSR + SSE`가 된다. 이원분산분석에서는 설명된 변동 `SSR`이 요인 A, 요인 B, 교호작용 A x B의 제곱합으로 다시 나뉜다. 각 요인의 평균제곱은 제곱합을 자유도로 나눈 값이고, F값은 해당 평균제곱을 오차 평균제곱으로 나눈 값이다.

## 15. “등분산성 검정: 분산분석”이라는 연결이 왜 부적절한지 설명하라.

분산분석은 범주형 집단에 따른 수치형 결과의 평균 차이를 검정하는 방법이다. 등분산성 검정은 집단들의 분산이 같은지 점검하는 별도 절차이며 Bartlett이나 Levene 같은 검정을 사용한다. 따라서 “등분산성 검정: 분산분석”은 목적과 방법을 잘못 연결한 것이다.

## 16. Shapiro-Wilk, Jarque-Bera, Kolmogorov-Smirnov의 검정통계량이 각각 무엇을 비교하는지 식으로 설명하라.

Shapiro-Wilk는 정렬된 표본과 정규분포 분위수 기반 가중치의 선형결합을 표본분산과 비교한다.

```text
W = (sum_i a_i x(i))^2 / sum_i (x_i - xbar)^2
```

Jarque-Bera는 정규분포의 왜도 0, 첨도 3에서 얼마나 벗어나는지 본다.

```text
JB = (n / 6) * (S^2 + ((K - 3)^2 / 4)) ~ chi-square(2)
```

Kolmogorov-Smirnov는 표본 누적분포함수와 비교 대상 누적분포함수의 최대 차이를 본다.

```text
D = sup_x |F_n(x) - F(x)|
```

## 17. Mann-Whitney U와 Wilcoxon 부호순위 검정통계량을 쓰고 독립/대응 표본 차이를 설명하라.

Mann-Whitney U 검정은 독립인 두 집단의 순위합을 비교한다.

```text
U1 = R1 - m(m + 1) / 2
U2 = R2 - n(n + 1) / 2
U = min(U1, U2)
```

Wilcoxon 부호순위 검정은 대응 표본의 차이를 계산한 뒤, 차이의 절댓값 순위와 부호를 이용한다.

```text
d_i = x_i - y_i
W+ = 양의 부호 순위합
W- = 음의 부호 순위합
W = min(W+, W-)
```

핵심 차이는 독립 표본 비교인지, 같은 대상의 전후처럼 짝지어진 표본 비교인지다.

## 18. Pearson, Spearman, Kendall 상관계수 공식을 쓰고 어떤 관계를 포착하는지 설명하라.

Pearson은 원자료의 선형 관계를 본다.

```text
r =
sum((x_i - Xbar)(y_i - Ybar))
/ sqrt(sum((x_i - Xbar)^2)sum((y_i - Ybar)^2))
```

Spearman은 순위의 Pearson 상관이다.

```text
r_s = Pearson(R_X, R_Y)
r_s = 1 - 6sum(d_i^2) / (n(n^2 - 1))  # 동점 순위가 없을 때
```

Kendall은 가능한 쌍의 순서가 일치하는지 비교한다.

```text
tau = (C - D) / (0.5 * n(n - 1))
```

Pearson은 선형 관계, Spearman은 단조 순위 관계, Kendall은 쌍별 순서 일치 정도를 포착한다.

## 19. 일원분산분석을 그룹별 평균으로 예측하는 선형회귀 관점에서 설명하라.

일원분산분석은 범주 `j`에 속한 관측값을 그 집단 평균 `a_j`로 예측하는 모델로 볼 수 있다.

```text
y_ij = a_j + e_ij
SSE = sum_j sum_i (y_ij - a_j)^2
a_j = sum_i y_ij / n_j
SSR = sum_j n_j(a_j - ybar)^2
SST = SSR + SSE
```

집단 평균들이 서로 비슷하면 `SSR`이 작고, 집단 평균 차이가 크면 `SSR`이 커진다. 그래서 `F = (SSR / (k - 1)) / (SSE / (n - k))`로 집단 간 변동이 집단 내 오차 변동에 비해 충분히 큰지 검정한다.

## 20. 카이제곱 적합도와 독립성 검정에서 기대빈도를 계산하는 식을 각각 쓰라.

적합도 검정은 기대 확률 `p_i`가 주어졌을 때 기대빈도를 `E_i = Np_i`로 계산한다.

```text
chi-square = sum_i (O_i - E_i)^2 / E_i ~ chi-square(k - 1)
```

독립성 검정은 교차표의 행합계와 열합계를 이용한다.

```text
E_ij = row_i_total * column_j_total / N
chi-square = sum_i sum_j (O_ij - E_ij)^2 / E_ij
~ chi-square((m - 1)(n - 1))
```

동질성 검정도 계산 방식은 독립성 검정과 같지만, 여러 집단의 분포가 같은지를 묻는 해석으로 읽는다.

## 21. 수치형-수치형, 범주형-수치형, 범주형-범주형 조합에서 대표 분석 방법을 각각 고르라.

수치형 변수 두 개가 함께 움직이는지 보려면 상관분석을 생각한다. 선형 관계는 Pearson, 순위나 단조 관계는 Spearman 또는 Kendall을 고려한다. 범주형 집단에 따라 수치형 결과의 평균이나 위치가 다른지 보려면 t검정, ANOVA, Mann-Whitney, Wilcoxon 같은 평균/위치 비교 검정을 생각한다. 범주형 변수 두 개의 관련성을 보려면 교차표와 카이제곱 독립성 검정을 생각한다.

## 22. 선형 관계와 단조 관계의 차이를 설명하고 Pearson과 Spearman 선택에 연결하라.

선형 관계는 두 변수의 관계가 직선에 가까운 경우다. Pearson 상관은 이런 선형 관계의 방향과 강도를 잘 포착한다. 단조 관계는 한 변수가 커질 때 다른 변수가 대체로 계속 커지거나 계속 작아지는 관계다. 곡선 형태라도 순서가 안정적으로 증가하면 Spearman이 더 적합할 수 있다. Spearman은 원래 값보다 순위를 사용하므로 비선형이지만 단조적인 관계에 강하다.

## 23. 기대빈도가 관측빈도와 다른 이유만으로 바로 인과관계를 말할 수 없는 이유를 설명하라.

카이제곱 검정에서 관측빈도와 기대빈도가 다르면 두 범주형 변수가 독립이라는 가정과 맞지 않는다는 증거가 된다. 하지만 이것은 함께 나타나는 패턴을 말할 뿐, 한 변수가 다른 변수를 원인으로 만든다는 뜻은 아니다. 숨은 제3변수, 표본 구성, 시간 순서, 수집 편향이 같은 패턴을 만들 수 있다. 인과를 말하려면 연구 설계와 시간 순서, 교란 요인 통제가 추가로 필요하다.
