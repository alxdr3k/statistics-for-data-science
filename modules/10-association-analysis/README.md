# 10. 연관성 분석

## 학습 목표

이 차시를 마치면 다음을 쉬운 말로 설명할 수 있으면 충분하다.

- 모수적 검정과 비모수적 검정을 가정의 차이로 구분한다.
- 피어슨, 스피어만, 켄달 상관의 차이를 설명한다.
- ANOVA와 카이제곱 검정이 각각 어떤 질문에 답하는지 이해한다.

## 오늘의 한 줄

연관성 분석은 변수들이 함께 움직이는지, 집단 간 차이가 있는지, 범주 빈도가 기대와 다른지 판단한다.

## 오늘 반드시 이해할 3가지

1. 모수적 검정과 비모수적 검정을 가정의 차이로 구분한다.
2. 피어슨, 스피어만, 켄달 상관의 차이를 설명한다.
3. ANOVA와 카이제곱 검정이 각각 어떤 질문에 답하는지 이해한다.

## 이 차시 전에 알면 좋은 것

- **변수 타입**: 상관/ANOVA/카이제곱 선택 기준
- **가설 검정**: p-value와 유의수준 해석 ([처음 설명된 차시](../06-hypothesis-testing/README.md#2-p-value와-유의수준))
- **분산**: 집단 간 차이와 집단 내 흔들림 비교

## 개념 지도

```text
연관성 분석
├── 검정 전 가정 점검
├── 상관 분석
├── 분산분석
├── 카이제곱 검정
└── 확인 문제와 해설
```

## 학습 우선순위

- **필수**: 상관이 인과가 아님을 이해, ANOVA가 전체 평균 차이를 먼저 묻는다는 점, 카이제곱이 빈도 차이를 본다는 점
- **심화**: 이원분산분석의 교호작용
- **확장**: 비모수 검정으로의 확장

## 이 차시에서 꼭 붙잡을 설명 방식

<a id="ref-10-상관계수"></a>[상관계수](#note-10-상관계수)가 높아도 한 <a id="ref-10-변수"></a>[변수](#note-10-변수)가 다른 변수를 원인으로 만든다는 뜻은 아니다. 두 변수가 함께 움직인다는 숫자일 뿐이다. 숨은 제3변수, 시간 흐름, 집단 구조가 같은 패턴을 만들 수 있으므로 상관과 인과를 분리해야 한다.

## 핵심 이론

### 먼저 잡는 직관

- **검정 전 가정 점검**: 연관성 분석은 변수 형태와 분포 가정에 따라 쓸 수 있는 검정이 달라진다.
- **상관 분석**: 상관계수는 두 변수가 함께 움직이는 방향과 강도를 요약하지만 원인을 증명하지는 않는다.
- **분산분석**: 세 집단 이상 평균 차이를 한 번에 비교해 전체적으로 차이가 있는지 먼저 본다.
- **카이제곱 검정**: 범주형 변수의 관측 빈도가 독립이라고 기대한 빈도와 얼마나 다른지 비교한다.

### 1. 검정 전 가정 점검

정규성 검정과 등분산성 검정은 어떤 검정을 쓸지 정하기 위한 점검이다. Shapiro-Wilk는 소표본, 특히 `n <= 50`에서 자주 쓰고, Jarque-Bera는 왜도와 첨도가 정규분포의 값에서 얼마나 벗어나는지 본다. Kolmogorov-Smirnov는 표본의 누적분포함수와 비교 대상 누적분포함수의 최대 차이를 본다.

정규성 검정의 가설은 보통 다음처럼 둔다.

```text
H0: 데이터는 정규분포를 따른다.
H1: 데이터는 정규분포를 따르지 않는다.
```

Shapiro-Wilk는 정렬된 표본 `x(1) <= x(2) <= ... <= x(n)`과 정규분포 분위수 기반 가중치 `a_i`를 사용한다.

```text
W = (sum_i a_i x(i))^2 / sum_i (x_i - xbar)^2
```

`W`가 클수록 정규분포에 가깝다. Jarque-Bera는 왜도 `S`와 첨도 `K`를 이용한다.

```text
JB = (n / 6) * (S^2 + ((K - 3)^2 / 4)) ~ chi-square(2)
```

Kolmogorov-Smirnov는 표본 누적분포함수 `F_n(x)`와 비교 대상 누적분포함수 `F(x)`의 가장 큰 차이를 본다.

```text
D = sup_x |F_n(x) - F(x)|
```

등분산성은 Bartlett 검정으로 확인할 수 있지만, Bartlett은 각 모집단이 정규분포를 따른다는 가정에 민감하다. 정규성 가정이 부담되면 Levene 검정을 대안으로 생각한다. 가정이 맞지 않으면 Wilcoxon 부호순위 검정, Mann-Whitney U 검정처럼 순위나 중앙값 차이에 기대는 비모수적 방법을 고려한다.

Bartlett 검정은 두 개 이상 모집단의 분산이 같은지 묻는다.

```text
H0: 전체 모집단의 분산이 같다.
H1: 전체 모집단의 분산이 모두 같지는 않다.
```

검정통계량은 각 집단 표본분산 `s_i^2`와 pooled variance `sbar^2`를 이용해 만들며, 근사적으로 `chi-square(k - 1)`를 따른다. 여기서 `k`는 집단 수, `N`은 전체 표본 수다.

Mann-Whitney U 검정은 독립인 두 집단의 위치 차이를 순위합으로 비교한다. Wilcoxon 부호순위 검정은 같은 대상의 전후처럼 대응 관계가 있는 두 집단에서 차이의 부호와 순위를 이용한다. 소표본에서는 permutation test처럼 더 직접적인 방법이 강건할 수 있고, 표본이 커지면 정규근사를 사용한다.

```text
Mann-Whitney:
U1 = R1 - m(m + 1) / 2
U2 = R2 - n(n + 1) / 2
U = min(U1, U2)

Wilcoxon signed-rank:
d_i = x_i - y_i
d_i = 0인 항목은 제외
W+ = 양의 부호 순위합
W- = 음의 부호 순위합
W = min(W+, W-)
```

### 2. 상관 분석

피어슨은 선형 관계와 수치값, 스피어만은 순위의 단조 관계, 켄달은 순위쌍의 일치 정도를 본다.

Pearson 상관의 귀무가설은 `r = 0`이고, 검정통계량은 `t = r * sqrt(n - 2) / sqrt(1 - r^2)`로 t분포와 비교한다. Spearman은 값을 순위로 바꾼 뒤 Pearson 상관을 계산한다고 보면 된다. Kendall은 가능한 쌍을 비교해 순서가 일치한 쌍과 불일치한 쌍의 차이를 본다.

```text
H0: r = 0
H1: r != 0

Pearson r =
sum((x_i - Xbar)(y_i - Ybar))
/ sqrt(sum((x_i - Xbar)^2)sum((y_i - Ybar)^2))

t = r * sqrt(n - 2) / sqrt(1 - r^2) ~ t(n - 2)
```

Spearman 상관은 `r_s = Pearson(R_X, R_Y)`로 계산한다. 동점 순위가 없으면 순위 차이 `d_i`를 써서 다음처럼 쓸 수 있다.

```text
r_s = 1 - 6sum(d_i^2) / (n(n^2 - 1))
```

Kendall의 순위 상관계수는 모든 쌍의 순서 비교를 이용한다.

```text
tau = (C - D) / (0.5 * n(n - 1))
```

`C`는 일치쌍 수, `D`는 불일치쌍 수다. Kendall은 이상치에 더 강건하고 작은 데이터에서도 안정적이지만, 모든 쌍을 비교하므로 큰 데이터에서는 느릴 수 있다.

![상관 분석의 직관](assets/correlation-analysis-274.png)

> **그림 읽기**: 두 변수가 함께 움직이는 방향과 강도를 본다. 점들이 직선에 가까워도 그것만으로 원인이라고 말할 수는 없다.

### 3. 분산분석

셋 이상 집단의 <a id="ref-10-평균"></a>[평균](#note-10-평균) 차이를 검정한다. 이원분산분석은 두 범주형 요인과 그 <a id="ref-10-교호작용"></a>[교호작용](#note-10-교호작용)까지 본다.

ANOVA는 범주형 변수로 나뉜 집단의 수치형 결과 평균을 비교한다. 귀무가설은 모든 집단 평균이 같다는 것이고, 검정통계량은 `F = MSR / MSE = (SSR / (k - 1)) / (SSE / (n - k))`이다. 정규방정식으로 추정된 선형회귀모델과 대응되는 관점으로 보면, 그룹별 평균으로 예측하는 회귀모델의 설명 변동과 오차 변동을 비교하는 것이다. 정규성/등분산성 가정이 어렵다면 Kruskal-Wallis를 비모수 대안으로 고려한다.

```text
H0: mu1 = mu2 = ... = muk
H1: 적어도 하나의 집단 평균은 다르다.

F = MSR / MSE
  = (SSR / (k - 1)) / (SSE / (n - k))
```

그룹별 평균을 예측값으로 쓰면 일원분산분석은 선형회귀와 같은 틀에서 볼 수 있다.

```text
y_ij = a_j + e_ij
SSE = sum_j sum_i (y_ij - a_j)^2
a_j = sum_i y_ij / n_j
SSR = sum_j n_j(a_j - ybar)^2
SST = SSR + SSE
```

![이원분산분석의 교호작용](assets/two-way-anova-interaction-280.png)

> **그림 읽기**: 두 요인이 따로 작용할 때와 함께 작용할 때 결과가 달라지는지 본다. 선이 평행하지 않으면 상호작용을 의심한다.

![이원분산분석 표](assets/two-way-anova-table-282.png)

> **그림 읽기**: 제곱합, 자유도, 평균제곱, F값이 어떤 행에 배치되는지 본다. 집단 차이를 변동의 분해로 읽는 표다.

### 4. 카이제곱 검정

적합도 검정은 한 범주의 <a id="ref-10-분포"></a>[분포](#note-10-분포)가 기대와 맞는지 보고, 독립성 검정은 두 범주형 변수가 관련 있는지 본다. 동질성 검정은 여러 집단의 범주 분포가 같은지 본다.

카이제곱 검정통계량은 관측빈도와 기대빈도의 차이를 모은 값이다.

```text
chi-square = sum((O_i - E_i)^2 / E_i)
```

적합도 검정에서는 `E_i = N p_i`로 기대빈도를 계산한다. 독립성 검정에서는 교차표의 행합계와 열합계를 이용해 `E_ij = row_i_total * column_j_total / N`으로 기대빈도를 계산한다. 동질성 검정은 계산 방식은 독립성 검정과 같지만, 여러 집단의 범주 분포가 같은지 묻는다는 해석이 다르다.

적합도 검정은 하나의 범주형 변수가 이론적 분포를 따르는지 묻는다.

```text
H0: P(X = 1) = p1, ..., P(X = k) = pk
H1: 적어도 하나의 범주 확률이 기대값과 다르다.

chi-square = sum_i (O_i - E_i)^2 / E_i ~ chi-square(k - 1)
E_i = Np_i
```

독립성 검정은 두 범주형 변수가 서로 독립인지 묻는다.

```text
H0: 두 변수는 독립이다.
H1: 두 변수는 독립이 아니다.

H0: P(A_i and B_j) = P(A_i)P(B_j)

chi-square = sum_i sum_j (O_ij - E_ij)^2 / E_ij
~ chi-square((m - 1)(n - 1))

E_ij = row_i_total * column_j_total / N
```

![카이제곱 검정의 직관](assets/chi-square-analysis-289.png)

> **그림 읽기**: 관측 빈도와 기대 빈도가 얼마나 다른지 본다. 범주형 변수의 관계는 평균이 아니라 빈도 차이로 판단한다.

### 5. 검정 선택표

연관성 분석에서는 정규성, 등분산성, 위치 차이, 상관, 분산분석, 카이제곱 검정을 한 흐름으로 묶는다. 문제를 풀 때는 “변수 타입”과 “가정 충족 여부”를 먼저 확인해야 한다.

| 상황 | 먼저 고려할 방법 | 핵심 질문 |
|---|---|---|
| 한 변수의 정규성 확인 | Shapiro-Wilk, Jarque-Bera, Kolmogorov-Smirnov | 정규분포라고 볼 수 있는가? |
| 집단 간 분산 비교 | Bartlett, Levene | 분산이 같다고 볼 수 있는가? |
| 정규성 가정이 어려운 위치 비교 | Wilcoxon, Mann-Whitney | 평균 대신 순위나 중앙값 차이를 볼 것인가? |
| 두 연속형 변수 관계 | Pearson/Spearman/Kendall 상관 | 선형 관계인가, 순위 관계인가? |
| 여러 집단 평균 비교 | ANOVA | 집단 평균 중 적어도 하나가 다른가? |
| 두 범주형 변수 관계 | 카이제곱 독립성 검정 | 두 변수가 독립인가? |
| 관측 빈도와 기대 비율 비교 | 카이제곱 적합도 검정 | 기대 분포와 맞는가? |
| 여러 집단의 범주 분포 비교 | 카이제곱 동질성 검정 | 집단별 분포가 같은가? |

Pearson 상관은 선형 관계에 민감하고, Spearman과 Kendall은 순위 기반이라 단조 관계나 이상치가 있는 상황에서 더 안정적일 수 있다. ANOVA가 유의하다는 말은 모든 집단이 다르다는 뜻이 아니라, 적어도 하나의 평균 차이가 있다는 뜻이다. 어느 집단끼리 다른지는 사후검정으로 확인한다.

## 판단 기준

1. 변수가 연속형인지 범주형인지 먼저 구분한다.
2. 정규성, 등분산성, <a id="ref-10-표본"></a>[표본](#note-10-표본) 독립성 같은 가정을 확인한다.
3. 상관계수는 산점도와 함께 보고 비선형 관계를 놓치지 않는다.
4. <a id="ref-10-anova"></a>[ANOVA](#note-10-anova)가 유의하면 사후검정으로 어느 집단 차이인지 확인한다.
5. 카이제곱 검정에서는 <a id="ref-10-기대빈도"></a>[기대빈도](#note-10-기대빈도)가 너무 작은 셀이 없는지 본다.

## 오해와 반례

### 오해 1. 상관관계는 인과관계다.

상관은 함께 움직인다는 뜻이다. 원인이라고 말하려면 설계, 시간 순서, 교란 요인 통제가 필요하다.

### 오해 2. 정규성 검정 p-value만 보면 된다.

표본 수에 따라 민감도가 달라진다. 그래프와 도메인 지식도 함께 봐야 한다.

### 오해 3. ANOVA는 어느 집단끼리 다른지 바로 알려 준다.

ANOVA는 전체 평균 차이가 있는지 먼저 본다. 구체적 쌍 차이는 사후검정이 필요하다.

## 예시 풀이

### 예시 1. 공부 시간과 시험 점수

둘 다 수치형이고 선형 관계를 보고 싶다면 피어슨 상관을 생각한다. 순위 관계만 안정적이면 스피어만을 고려한다.

### 예시 2. 지역과 구매 여부의 관계

두 변수가 범주형이면 교차표를 만들고 카이제곱 독립성 검정으로 기대빈도와 관측빈도의 차이를 본다.

## 오늘의 요약 5줄

1. 연관성 분석은 변수들이 함께 움직이는지, 집단 차이가 있는지, 빈도가 기대와 다른지 본다.
2. 상관관계는 함께 움직임을 말할 뿐 인과관계를 증명하지 않는다.
3. 모수적 검정은 분포 가정을 더 강하게 쓰고, 비모수적 검정은 순위나 빈도 정보를 더 활용한다.
4. ANOVA는 여러 평균이 모두 같다는 귀무가설을 한 번에 검정한다.
5. 카이제곱 검정은 범주형 변수의 독립성이나 적합도를 판단할 때 쓴다.

## 확인 문제

1. 모수적 검정과 비모수적 검정의 차이를 설명하라.
2. 피어슨 상관계수와 스피어만 상관계수의 차이를 설명하라.
3. 상관관계를 인과관계로 해석하면 안 되는 이유를 설명하라.
4. ANOVA가 유의할 때 사후검정이 필요한 이유를 설명하라.
5. 카이제곱 독립성 검정의 직관을 설명하라.
6. 기대빈도가 너무 작을 때 생기는 문제를 설명하라.
7. 왜 상관관계만으로 인과관계를 말할 수 없는가?
8. 왜 ANOVA가 유의해도 바로 어느 두 집단이 다른지는 알 수 없는가?
9. Pearson, Spearman, Kendall 상관을 구분하는 기준을 설명하라.
10. 카이제곱 적합도, 독립성, 동질성 검정의 차이를 설명하라.
11. Shapiro-Wilk, Jarque-Bera, Kolmogorov-Smirnov 정규성 검정의 차이를 설명하라.
12. Bartlett 검정과 Levene 검정을 등분산성 점검 관점에서 비교하라.
13. Mann-Whitney U 검정과 Wilcoxon 부호순위 검정의 차이를 설명하라.
14. 이원분산분석 표에서 전체 제곱합, 회귀 제곱합, 오차 제곱합과 F값을 어떻게 연결하는지 설명하라.
15. “등분산성 검정: 분산분석”이라는 연결이 왜 부적절한지 설명하라.
16. Shapiro-Wilk, Jarque-Bera, Kolmogorov-Smirnov의 검정통계량이 각각 무엇을 비교하는지 식으로 설명하라.
17. Mann-Whitney U와 Wilcoxon 부호순위 검정통계량을 쓰고 독립/대응 표본 차이를 설명하라.
18. Pearson, Spearman, Kendall 상관계수 공식을 쓰고 어떤 관계를 포착하는지 설명하라.
19. 일원분산분석을 그룹별 평균으로 예측하는 선형회귀 관점에서 설명하라.
20. 카이제곱 적합도와 독립성 검정에서 기대빈도를 계산하는 식을 각각 쓰라.

## 개념 주석

본문에서 연결된 개념을 잠깐 확인하는 공간이다. 용어를 누르면 본문에서 처음 표시된 위치로 돌아간다.

- <a id="note-10-상관계수"></a>[상관계수](#ref-10-상관계수): 두 변수가 함께 움직이는 방향과 강도.
- <a id="note-10-변수"></a>[변수](#ref-10-변수): 관측 대상의 특징을 적어 둔 열. ([처음 설명된 차시](../01-data-understanding/README.md#4-단위-변수-관측치))
- <a id="note-10-평균"></a>[평균](#ref-10-평균): 모든 값을 더해 개수로 나눈 대표값. ([처음 설명된 차시](../04-statistics-probability/README.md#4-중심-경향))
- <a id="note-10-교호작용"></a>[교호작용](#ref-10-교호작용): 한 요인의 효과가 다른 요인의 수준에 따라 달라지는 현상.
- <a id="note-10-분포"></a>[분포](#ref-10-분포): 값들이 어떤 모양으로 흩어져 있는지 나타내는 구조. ([처음 설명된 차시](../05-probability-distributions/README.md#1-확률변수와-분포))
- <a id="note-10-표본"></a>[표본](#ref-10-표본): 전체 대신 관찰한 일부 대상. ([처음 설명된 차시](../04-statistics-probability/README.md#2-모집단과-표본))
- <a id="note-10-anova"></a>[ANOVA](#ref-10-anova): 셋 이상 집단 평균 차이를 분산으로 검정하는 방법. 이름: Analysis of Variance다. 평균 차이를 분산의 분해로 검정한다.
- <a id="note-10-기대빈도"></a>[기대빈도](#ref-10-기대빈도): 독립이나 가정이 맞다면 기대되는 빈도.
