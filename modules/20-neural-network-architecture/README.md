## 핵심 이론

### 먼저 잡는 직관

- **Dense와 Embedding**: Dense는 모든 입력을 가중치로 연결하고, Embedding은 범주나 단어를 의미 있는 벡터로 바꾼다.
- **CNN**: CNN은 작은 필터를 움직이며 이미지나 시계열의 지역 패턴을 잡는다.
- **RNN, LSTM, GRU**: 순서가 있는 데이터에서는 이전 정보가 다음 판단에 영향을 주므로 순환 구조가 필요하다.
- **보조 레이어**: Dropout과 Pooling은 일반화와 정보 압축을 돕는다.

### 1. Dense와 Embedding

Dense는 일반적인 수치 벡터를 처리하는 기본 층이다. Embedding은 단어, 사용자, 상품 같은 범주를 연속 벡터로 바꿔 의미적 관계를 학습한다.

Dense Layer는 완전연결층(Fully Connected Layer)이다. 이전 레이어의 입력 벡터 `x`가 현재 레이어의 모든 뉴런과 완전히 연결된다. 입력 크기가 `m`, 뉴런 수가 `n`이면 가중치 행렬은 `W`의 크기가 `n x m`, 편향 벡터 `b`의 크기가 `n`이다. 연산은 `y = f(Wx + b)`로 쓰고, 파라미터 수는 `m x n + n`이다.

Dense Layer는 충분한 수의 뉴런과 층이 있으면 복잡한 특징을 표현할 수 있고, 다층 퍼셉트론의 핵심 구성 요소다. 하지만 이미지처럼 공간적·시각적 정보가 중요한 데이터에서는 펼치는 과정에서 위치 구조가 약해질 수 있고, 특성이 늘어날수록 파라미터 수가 폭증한다.

Embedding Layer는 범주형 데이터를 연속적인 실수 벡터 공간으로 매핑한다. 입력은 범주의 정수 인덱스이며, 예를 들어 `사과 -> 1`, `배 -> 2`, `Unknown -> 0`처럼 둘 수 있다. 범주 수가 `k`, 임베딩 크기가 `n`, 입력 길이가 `m`이면 임베딩 테이블은 `k x n`이고 출력은 `m x n`이다. 연산은 입력 인덱스가 가리키는 위치의 벡터를 반환하는 것이다.

Embedding은 역전파를 통해 데이터에 나타난 범주의 의미적 관계를 학습한다. 높은 가짓수를 가진 범주형 변수를 one-hot으로 만들 때 생기는 고차원 희소벡터 비효율을 줄이고, 단어·사용자 ID·상품 ID처럼 범주 수가 큰 데이터에 유용하다. 다만 새로 등장한 범주나 등장 빈도가 적은 범주에는 성능이 낮을 수 있고, 최적의 임베딩 크기를 정해야 한다.

### 2. CNN

합성곱은 필터로 지역 패턴을 찾고, 풀링은 영역별 대표값으로 정보를 압축한다. 위치 기반 패턴을 잘 다룬다.

합성곱 레이어는 필터를 입력 데이터 전체에 걸쳐 이동시키며 영역의 특징을 추출한다. 채널(channel)은 위치의 특징을 공유하는 데이터 단위이며, `C_in`은 입력 채널 수, `C_out`은 출력 채널 수다. 커널 또는 필터는 지역적 패턴을 나타내는 가중치 텐서이고, padding `P`는 가장자리에 0을 추가해 출력 크기를 조절하며, stride `S`는 필터가 한 번에 이동하는 거리다.

한 축의 출력 크기는 `O = (I - K + 2P) / S + 1`로 계산한다. 예를 들어 입력 `23 x 23`, 커널 `5 x 5`, padding `2`, stride `2`이면 `O = (23 - 5 + 4) / 2 + 1 = 12`이므로 출력은 `12 x 12`다.

다채널 합성곱에서는 파라미터 수가 `커널 높이 x 커널 너비 x C_in x C_out + C_out`이다. 예를 들어 입력 `5 x 5 x 3`, 커널 `3 x 3`, 출력 채널 `4`라면 커널 파라미터는 `3 x 3 x 3 x 4 = 108`, 편향은 `4`개다.

### 3. RNN, LSTM, GRU

RNN은 순서 데이터를 처리하지만 긴 의존성을 잘 보존하기 어렵다. LSTM과 GRU는 게이트 구조로 어떤 정보를 유지하고 버릴지 조절한다.

시퀀스 데이터를 Dense Layer만으로 처리하면 두 가지 문제가 생긴다. 시퀀스 전체를 하나의 입력셋으로 보면 뉴런 크기와 파라미터 수가 크게 늘고, 각 시점을 따로 처리하면 순서 정보를 표현하기 어렵다. RNN은 이전 은닉상태 `h_{t-1}`와 현재 입력 `x_t`를 함께 사용해 `h_t = tanh(W_x x_t + W_h h_{t-1} + b_h)`처럼 현재 상태를 만든다.

RNN의 문제점은 기울기 소실과 기울기 폭증이다. 역전파 과정에서 작은 값이 반복해서 곱해지면 기울기가 사라지고, 큰 값이 반복되면 학습이 불안정해진다. 이를 줄이기 위해 LSTM과 GRU 같은 게이트 구조가 도입된다.

LSTM은 중요한 정보는 기억하고 불필요한 정보는 버리도록 여러 게이트를 둔다. 입력 게이트는 새 정보를 기억 영역에 얼마나 반영할지, 망각 게이트는 이전 정보를 얼마나 유지할지, 출력 게이트는 셀 상태를 은닉상태에 얼마나 반영할지 조절한다. 셀 상태는 장기 메모리 역할을 한다.

GRU는 LSTM보다 간단한 구조로 계산량을 줄인다. 갱신 게이트(Update Gate)는 새 정보 반영과 이전 상태 유지의 균형을 조절하고, 초기화 또는 리셋 게이트(Reset Gate)는 이전 상태를 얼마나 초기화해 새 후보 상태를 만들지 조절한다. 이런 게이트는 정보 반영 비율을 다루므로 보통 sigmoid처럼 0과 1 사이 값을 내는 함수가 적합하고, ReLU로 대체하기 어렵다.

### 5. 레이어별 입력과 출력

Dense, Embedding, CNN, RNN, LSTM, GRU, Dropout, Pooling은 각각의 역할로 구분한다. Dense는 모든 입력과 모든 뉴런을 연결하므로 파라미터 수가 빠르게 늘어난다. Embedding은 범주 ID를 의미 있는 연속 벡터로 바꾸며, 라벨 인코딩처럼 숫자 순서를 의미한다고 가정하지 않는다.

CNN에서는 필터 크기, stride, padding이 출력 크기를 정한다. 필터는 지역 패턴을 찾고, 1x1 convolution은 공간 크기보다 채널 간 조합을 바꾸는 데 자주 쓰인다. Pooling은 대표값을 뽑아 공간 크기를 줄이고, 작은 위치 변화에 덜 민감하게 만든다.

RNN 계열은 시퀀스 모델링 유형을 입력과 출력의 길이로 구분한다. many-to-one은 문장 분류처럼 여러 입력에서 하나의 출력을 만들고, one-to-many는 하나의 입력에서 여러 출력을 만든다. one-per-one은 각 시점 입력마다 각 시점 출력을 만들고, many-to-many는 번역이나 시계열 예측처럼 입력과 출력이 모두 순서 구조를 가진다. LSTM과 GRU는 게이트로 장기 의존성을 보존하려는 구조다.

Dropout은 학습 중 무작위로 일부 뉴런을 제외해 과적합에 대응한다. 특정 뉴런에 지나치게 의존하지 않게 만들고, 서로 다른 서브네트워크를 무작위로 학습하는 앙상블 효과를 준다. Dropout rate는 뉴런이 제외될 확률이다.

Pooling은 영역별 대표값을 추출해 압축한다. 공간 차원을 줄여 연산량과 메모리를 절약하고, 작은 위치 변화에 강인함을 줄 수 있지만 정보 손실에 유의해야 한다. Max Pooling은 윈도우 안 최대값을, Average Pooling은 평균값을 대표값으로 쓴다. Global Pooling은 채널 전체를 단위 영역으로 집계한다. Pooling의 출력 크기도 `O = (I - K + 2P) / S + 1`로 계산한다.

## 판단 기준

1. 데이터가 표 형태, 이미지, 텍스트, 시계열 중 무엇인지 먼저 확인한다.
2. 레이어가 입력 형태를 어떤 출력 형태로 바꾸는지 차원을 추적한다.
3. 파라미터 수가 너무 커져 과대적합 위험이 생기지 않는지 본다.
4. Embedding은 단순 번호가 아니라 학습되는 표현임을 구분한다.
5. Dropout과 Pooling이 각각 일반화와 압축에 어떤 역할을 하는지 확인한다.

## 오해와 반례

### 오해 1. Dense Layer는 모든 데이터에 항상 최선이다.

이미지나 시퀀스처럼 구조가 있는 데이터는 CNN이나 RNN 계열이 더 자연스러울 수 있다.

### 오해 2. Embedding은 단순한 번호 붙이기다.

번호가 아니라 학습되는 연속 벡터다. 범주 사이 관계를 표현할 수 있다.

### 오해 3. Dropout은 예측할 때도 뉴런을 계속 버린다.

훈련 중 과적합을 줄이기 위해 사용하고, 추론 시에는 보정된 전체 네트워크를 사용한다.

## 개념 주석

본문에서 연결된 개념을 잠깐 확인하는 공간이다. 용어를 누르면 본문에서 처음 표시된 위치로 돌아간다.

- <a id="note-20-dense"></a>[Dense](#ref-20-dense): 이전 층의 모든 입력이 현재 층의 모든 뉴런과 연결되는 층. 이름: 모든 입력과 뉴런이 빽빽하게 연결된 층이다.
- <a id="note-20-cnn"></a>[CNN](#ref-20-cnn): 필터로 지역 패턴을 찾는 합성곱 신경망.
