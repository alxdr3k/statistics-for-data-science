# 20. 신경망의 구성 · 확인 문제 해설

## 1. Dense Layer의 파라미터 수를 설명하라.

Dense Layer는 모든 입력 뉴런이 모든 출력 뉴런과 연결된다. 입력 차원이 m이고 출력 뉴런 수가 n이면 가중치는 m x n개이고, 편향은 n개다. 따라서 전체 파라미터 수는 m x n + n이다.

## 2. Embedding이 단순 라벨 인코딩과 다른 이유를 설명하라.

라벨 인코딩은 범주에 번호를 붙일 뿐이라 번호 사이의 의미 관계를 배우지 않는다. Embedding은 각 범주를 여러 차원의 벡터로 바꾸고 그 벡터를 학습한다. 비슷한 단어나 범주는 가까운 표현을 갖도록 학습될 수 있다.

## 3. CNN의 필터가 하는 일을 설명하라.

필터는 이미지의 작은 영역을 훑으면서 특정 패턴이 있는지 감지한다. 낮은 층에서는 선이나 색 변화 같은 단순 패턴을, 깊은 층에서는 더 복잡한 조합을 잡을 수 있다. 같은 필터를 여러 위치에 적용하기 때문에 위치가 조금 달라도 패턴을 찾기 쉽다.

## 4. RNN이 순서 데이터에 적합한 이유를 설명하라.

RNN은 현재 입력뿐 아니라 이전 시점의 상태를 함께 사용한다. 그래서 문장, 시계열처럼 앞의 정보가 뒤의 판단에 영향을 주는 데이터에 맞다. 다만 긴 의존성에서는 기울기 문제가 생길 수 있다.

## 5. LSTM과 GRU가 단순 RNN의 어떤 문제를 줄이려는지 설명하라.

단순 RNN은 긴 시퀀스에서 오래전 정보를 유지하기 어렵고 기울기 소실이 생기기 쉽다. LSTM과 GRU는 게이트 구조로 어떤 정보를 기억하고 버릴지 조절한다. 이 덕분에 더 긴 의존성을 다루기 쉬워진다.

## 6. Dropout이 과대적합을 줄이는 직관을 설명하라.

Dropout은 훈련 중 일부 뉴런을 임시로 꺼서 특정 뉴런 조합에만 의존하지 못하게 한다. 모델은 여러 부분 조합으로도 예측하도록 학습한다. 이 효과가 앙상블처럼 작동해 과대적합을 줄일 수 있다.
