# 11. 머신 러닝 · 확인 문제 해설

## 1. 지도학습과 비지도학습의 차이를 설명하라.

지도학습은 입력과 정답 라벨을 함께 사용해 새 입력의 정답을 예측한다. 비지도학습은 정답 라벨 없이 데이터 안의 구조나 묶음을 찾는다. 분류와 회귀는 지도학습, 군집화와 차원 축소는 비지도학습에 가깝다.

## 2. 분류 문제에서 정확도만 보면 위험한 상황을 설명하라.

사기 거래가 1%뿐인 데이터에서 모든 거래를 정상이라고 예측해도 정확도는 99%가 될 수 있다. 하지만 정작 중요한 사기 거래는 하나도 잡지 못한다. 이런 경우 재현율, 정밀도, F1, AUC를 함께 봐야 한다.

## 3. 정밀도와 재현율의 차이를 예로 설명하라.

정밀도는 모델이 양성이라고 한 것 중 실제 양성이 얼마나 되는지다. 재현율은 실제 양성 중 모델이 얼마나 찾아냈는지다. 암 진단처럼 놓치면 위험한 문제는 재현율이 특히 중요할 수 있다.

## 4. 훈련, 검증, 테스트 데이터의 역할을 설명하라.

훈련 데이터는 모델이 패턴을 배우는 데 사용한다. 검증 데이터는 하이퍼파라미터나 모델 선택에 사용한다. 테스트 데이터는 모든 선택이 끝난 뒤 최종 일반화 성능을 확인하는 데만 사용한다.

## 5. 과소적합과 과대적합을 구분하는 방법을 설명하라.

과소적합은 훈련 성능도 낮고 검증 성능도 낮은 상태다. 모델이 너무 단순해 데이터의 중요한 패턴을 못 잡은 것이다. 과대적합은 훈련 성능은 높지만 검증 성능이 낮아 훈련 데이터의 우연한 패턴까지 외운 상태다.

## 6. 교차검증이 필요한 이유를 설명하라.

한 번의 훈련/검증 분할만 보면 우연히 쉬운 검증 세트나 어려운 검증 세트가 걸릴 수 있다. 교차검증은 여러 분할에서 성능을 확인해 평가의 흔들림을 줄인다. 특히 데이터가 많지 않을 때 유용하다.
