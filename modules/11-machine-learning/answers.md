# 11. 머신 러닝 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 지도학습과 비지도학습의 차이를 설명하라.

지도학습은 입력과 정답 라벨을 함께 사용해 새 입력의 정답을 예측한다. 비지도학습은 정답 라벨 없이 데이터 안의 구조나 묶음을 찾는다. 분류와 회귀는 지도학습, 군집화와 차원 축소는 비지도학습에 가깝다.

## 2. 분류 문제에서 정확도만 보면 위험한 상황을 설명하라.

사기 거래가 1%뿐인 데이터에서 모든 거래를 정상이라고 예측해도 정확도는 99%가 될 수 있다. 하지만 정작 중요한 사기 거래는 하나도 잡지 못한다. 이런 경우 재현율, 정밀도, F1, AUC를 함께 봐야 한다.

## 3. 정밀도와 재현율의 차이를 예로 설명하라.

정밀도는 모델이 양성이라고 한 것 중 실제 양성이 얼마나 되는지다. 재현율은 실제 양성 중 모델이 얼마나 찾아냈는지다. 암 진단처럼 놓치면 위험한 문제는 재현율이 특히 중요할 수 있다.

## 4. 훈련, 검증, 테스트 데이터의 역할을 설명하라.

훈련 데이터는 모델이 패턴을 배우는 데 사용한다. 검증 데이터는 하이퍼파라미터나 모델 선택에 사용한다. 테스트 데이터는 모든 선택이 끝난 뒤 최종 일반화 성능을 확인하는 데만 사용한다.

## 5. 과소적합과 과대적합을 구분하는 방법을 설명하라.

과소적합은 훈련 성능도 낮고 검증 성능도 낮은 상태다. 모델이 너무 단순해 데이터의 중요한 패턴을 못 잡은 것이다. 과대적합은 훈련 성능은 높지만 검증 성능이 낮아 훈련 데이터의 우연한 패턴까지 외운 상태다.

## 6. 교차검증이 필요한 이유를 설명하라.

한 번의 훈련/검증 분할만 보면 우연히 쉬운 검증 세트나 어려운 검증 세트가 걸릴 수 있다. 교차검증은 여러 분할에서 성능을 확인해 평가의 흔들림을 줄인다. 특히 데이터가 많지 않을 때 유용하다.

## 7. 왜 훈련 성능만 높으면 좋은 모델이라고 말할 수 없는가?

- **한 문장 답**: 훈련 데이터의 우연한 패턴까지 외웠을 수 있기 때문이다.
- **이유**: 모델은 훈련 데이터에서 맞힌 규칙이 새 데이터에도 통하는지 보여야 한다. 외운 패턴은 훈련 점수만 높이고 일반화 성능은 낮출 수 있다.
- **예시**: 문제집 답을 외운 학생은 같은 문제는 잘 풀지만 새 문제에는 약할 수 있다.
- **주의점**: 검증/테스트 성능을 분리해서 봐야 한다.

## 8. 왜 정확도 대신 재현율이나 정밀도를 봐야 하는 문제가 있는가?

- **한 문장 답**: 틀리는 방식의 비용이 문제마다 다르기 때문이다.
- **이유**: 정확도는 맞은 비율만 보므로 어떤 종류의 오답이 위험한지 반영하지 못한다. 양성 놓침과 오탐의 비용이 다르면 다른 지표가 필요하다.
- **예시**: 암 진단은 실제 환자를 놓치는 것이 특히 위험해 재현율을 중시할 수 있다.
- **주의점**: 지표 선택은 모델보다 업무 비용에서 먼저 나온다.

## 9. 모수적 모델과 비모수적 모델의 차이를 설명하라.

모수적 모델은 미리 정한 함수 형태와 제한된 수의 파라미터 안에서 학습한다. 비모수적 모델은 정해진 파라미터 수보다 데이터 구조에 따라 더 유연하게 형태가 바뀐다. 모수적 모델은 단순하고 빠르지만 가정이 틀리면 약하고, 비모수적 모델은 유연하지만 과적합과 계산 비용을 조심해야 한다.

## 10. ROC-AUC와 PR-AUC가 각각 어떤 상황에서 유용한지 설명하라.

ROC-AUC는 양성과 음성을 전반적으로 구분하는 능력을 임계값 전체에서 본다. PR-AUC는 양성 예측의 정밀도와 실제 양성 포착률을 보므로 양성 클래스가 드문 불균형 문제에서 더 민감하게 성능을 보여 준다. 희귀질환 탐지처럼 양성이 적은 문제에서는 PR-AUC가 더 해석하기 좋을 수 있다.

## 11. 편향-분산 관점에서 과소적합과 과대적합을 설명하라.

과소적합은 모델이 너무 단순해 진짜 패턴을 못 잡는 상태라 편향이 크다. 과대적합은 훈련 데이터의 우연한 흔들림까지 따라가 새 데이터에서 예측이 흔들리는 상태라 분산이 크다. 좋은 모델 선택은 편향과 분산의 균형을 찾는 일이다.

## 12. ROC curve의 x축과 y축이 각각 무엇인지 설명하라.

ROC curve의 x축은 False Positive Rate이고, y축은 True Positive Rate다. FPR은 실제 음성 중 양성으로 잘못 예측한 비율이고, TPR은 실제 양성 중 양성으로 맞힌 비율이다. 낮은 FPR에서 높은 TPR을 유지할수록 분류기의 변별력이 좋다.

## 13. hold-out, cross validation, repeated validation의 차이를 설명하라.

hold-out은 데이터를 한 번 나누어 훈련과 검증을 한다. cross validation은 데이터를 여러 fold로 나누고 각 fold를 번갈아 검증에 사용해 분할 운의 영향을 줄인다. repeated validation은 이런 분할이나 검증을 여러 번 반복해 성능의 변동성을 더 잘 본다.

## 14. Stratified Split, Grouped Split, Time Series Split을 언제 쓰는지 설명하라.

Stratified Split은 클래스 비율이 치우쳐 있어 훈련과 검증에서 타깃 비율을 비슷하게 유지해야 할 때 쓴다. Grouped Split은 같은 사용자나 같은 기관처럼 묶인 데이터가 양쪽에 동시에 들어가 데이터 누수가 생기는 것을 막을 때 쓴다. Time Series Split은 시간 순서가 중요한 데이터에서 미래 정보가 과거 학습에 섞이지 않게 할 때 쓴다.

## 15. 실루엣 점수가 군집화에서 무엇을 보는지 설명하라.

실루엣 점수는 한 점이 자기 군집 안의 다른 점들과 얼마나 가깝고, 다른 군집과는 얼마나 잘 떨어져 있는지를 함께 본다. 자기 군집 응집도가 높고 다른 군집과의 분리도가 크면 점수가 좋아진다. 군집 라벨의 정답이 없을 때 구조가 그럴듯한지 보는 내부 평가 지표다.

## 16. 이진분류 혼동행렬의 `TP`, `FP`, `FN`, `TN`과 Accuracy, Precision, Recall, Specificity, F1, FPR 공식을 쓰라.

`TP`는 실제 양성을 양성으로 맞힌 경우, `FP`는 실제 음성을 양성으로 잘못 예측한 경우다. `FN`은 실제 양성을 음성으로 놓친 경우, `TN`은 실제 음성을 음성으로 맞힌 경우다.

```text
Accuracy = (TP + TN) / (TP + FP + FN + TN)
Precision = TP / (TP + FP)
Recall = TPR = TP / (TP + FN)
Specificity = TN / (TN + FP)
F1 = 2 * Precision * Recall / (Precision + Recall)
FPR = FP / (TN + FP)
```

ROC curve는 x축에 FPR, y축에 TPR을 둔다.

## 17. 회귀 성능 지표 MAE, MSE, RMSE, R2, MAPE의 공식을 쓰고 각각 무엇을 벌하는지 설명하라.

```text
MAE = (1 / n) * sum(|y_i - yhat_i|)
MSE = (1 / n) * sum((y_i - yhat_i)^2)
RMSE = sqrt(MSE)
R^2 = 1 - sum((y_i - yhat_i)^2) / sum((y_i - ybar)^2)
MAPE = (100 / n) * sum(|(y_i - yhat_i) / y_i|)
```

MAE는 오차의 실제 크기를 평균 내므로 해석이 쉽다. MSE와 RMSE는 큰 오차를 더 강하게 벌한다. R2는 전체 변동 중 모델이 설명한 비율을 본다. MAPE는 상대 오차를 퍼센트로 보지만 실제값이 0에 가까우면 불안정하다.

## 18. 실루엣 계수의 `a(i)`, `b(i)`, `s(i)` 공식을 쓰고 값의 해석을 설명하라.

`a(i)`는 점 `i`가 같은 군집 안의 다른 샘플들과 떨어진 평균거리다. `b(i)`는 다른 군집들과의 평균거리 중 가장 작은 값이다.

```text
s(i) = (b(i) - a(i)) / max(a(i), b(i))
```

`s(i)`가 1에 가까우면 자기 군집에는 가깝고 다른 군집과는 멀다. 0 근처이면 경계에 있는 점일 수 있고, 음수이면 다른 군집에 더 가까운 점일 수 있다.

## 19. 편향-분산 분해식에서 Bias, Variance, Irreducible Error가 각각 무엇을 뜻하는지 설명하라.

일반화 오차는 다음처럼 분해해 읽는다.

```text
E_D[(f_hat(x) - y)^2]
= (E_D[f_hat(x)] - f(x))^2
  + E_D[(f_hat(x) - E_D[f_hat(x)])^2]
  + sigma^2
```

첫 항은 Bias 제곱으로, 평균적인 예측이 참 함수에서 얼마나 떨어졌는지다. 둘째 항은 Variance로, 학습 데이터가 바뀔 때 예측이 얼마나 흔들리는지다. 셋째 항 `sigma^2`는 데이터 자체의 노이즈라 모델이 줄일 수 없는 오차다. 과소적합은 Bias가 큰 경우가 많고, 과대적합은 Variance가 큰 경우가 많다.
