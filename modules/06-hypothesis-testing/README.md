# 06. 가설 검정

## 학습 목표

이 차시를 마치면 다음을 쉬운 말로 설명할 수 있으면 충분하다.

- 귀무가설과 대립가설을 주장과 증거의 관계로 이해한다.
- 가설은 모수에 대해 세우고, 표본 통계량은 증거로 사용한다는 점을 구분한다.
- p-value는 귀무가설이 맞다고 가정했을 때 지금보다 극단적인 결과가 나올 가능성임을 설명한다.
- 검정통계량을 `표본에서 나온 차이 / 표준오차` 구조로 읽는다.
- 유의수준과 기각역이 결론을 자르는 기준이라는 점을 이해한다.

## 오늘의 한 줄

가설 검정은 표본이 보여 준 차이가 우연으로도 설명되는지 따져 보는 절차다.

## 오늘 반드시 이해할 3가지

1. 귀무가설과 대립가설을 주장과 증거의 관계로 이해한다.
2. 검정통계량은 표본 결과가 기준값에서 몇 표준오차만큼 떨어졌는지 보는 숫자다.
3. p-value는 귀무가설이 맞다고 가정했을 때 지금보다 극단적인 결과가 나올 가능성이다.

## 이 차시 전에 알면 좋은 것

- **분포**: 검정통계량이 얼마나 드문지 판단하는 기준
- **표본평균**: 표본 결과를 하나의 판단 숫자로 바꾸는 재료
- **조건부 확률**: p-value를 “가정했을 때”의 확률로 읽는 감각 ([처음 설명된 차시](../04-statistics-probability/README.md#9-조건부-확률))
- **모수와 통계량**: `mu`는 모집단의 기준값, `Xbar`는 표본에서 계산한 값이라는 구분 ([처음 설명된 차시](../05-probability-distributions/README.md#8-모수와-통계량))
- **표준오차**: 표본평균 같은 통계량이 표본추출마다 흔들리는 정도 ([처음 설명된 차시](../05-probability-distributions/README.md#14-표준화-표준오차-표본분포))

## 학습 우선순위

- **필수**: 귀무가설과 대립가설 구분, p-value와 유의수준의 의미, 단측/양측 검정을 질문에서 결정
- **심화**: 1종/2종 오류의 균형
- **확장**: 검정력과 표본수 설계

## 이 차시에서 꼭 붙잡을 설명 방식

<a id="ref-06-p-value"></a>[p-value](#note-06-p-value)를 “<a id="ref-06-귀무가설"></a>[귀무가설](#note-06-귀무가설)이 맞을 확률”로 해석하면 틀린다. p-value는 귀무가설이 맞다고 이미 가정한 뒤, 그런 세계에서 지금 같은 <a id="ref-06-표본"></a>[표본](#note-06-표본) 결과가 얼마나 드문지를 계산한다. 즉 방향이 반대다. 이 차이를 놓치면 0.03이라는 숫자를 “귀무가설이 3% 확률로 참”이라고 오해한다.

검정은 대부분 같은 문장 구조를 가진다.

```text
표본에서 나온 차이 / 우연히 흔들릴 수 있는 크기
```

분자는 `Xbar - mu0`, `phat - p0`, `S^2 - sigma0^2`처럼 표본 결과가 기준값에서 얼마나 벗어났는지다. 분모는 표준오차처럼 그 정도 차이가 우연히도 생길 수 있는 흔들림의 크기다. 차이가 커 보여도 표준오차가 크면 드문 결과가 아닐 수 있고, 차이가 작아 보여도 표준오차가 매우 작으면 드문 결과일 수 있다.

## 핵심 이론

### 먼저 잡는 직관

- **가설 검정의 출발점**: 검정은 “기본 주장이 맞다”고 일단 놓고, 표본이 그 주장을 얼마나 흔드는지 보는 절차다.
- **p-value와 유의수준**: p-value는 귀무가설이 맞는 세계에서 지금 결과가 얼마나 드문지이고, 유의수준은 그 드묾을 판단하는 기준선이다.
- **검정통계량과 기각역**: 표본 결과를 바로 비교하기 어렵기 때문에 하나의 판단 숫자로 바꾸고, 그 숫자가 기각역에 들어가는지 본다.
- **단측검정과 양측검정**: 관심이 한쪽 방향인지, 어느 방향이든 차이인지에 따라 꼬리를 보는 방식이 달라진다.

### 1. 가설 검정의 출발점

검정은 표본으로 <a id="ref-06-모집단"></a>[모집단](#note-06-모집단)의 주장에 답하는 방법이다. 예를 들어 우유 200ml가 실제로 부족한지 알고 싶다면, 먼저 “<a id="ref-06-평균"></a>[평균](#note-06-평균)은 200ml다”를 기본 주장으로 놓고 표본이 이 주장을 얼마나 흔드는지 본다.

귀무가설(Null Hypothesis, `H0`)은 원래 기준이 되는 가설, 즉 기본 입장이다. 대립가설(Alternative Hypothesis, `H1`)은 귀무가설에 반하는 대안적 주장이다. 둘은 동시에 참일 수 없도록 배타적인 관계로 세운다.

```text
H0: mu = 200
H1: mu < 200
```

대립가설을 주장하려면 단순히 표본평균이 조금 작다는 정도가 아니라, 귀무가설의 기본 입장을 흔들 만큼 강한 증거가 필요하다.

여기서 `mu`는 모집단 평균이라는 모수이고, `Xbar`는 표본평균이라는 통계량이다. 가설은 모수에 대해 세우고, 증거는 통계량에서 나온다. 따라서 `H0: Xbar=200`이라고 쓰지 않고 `H0: mu=200`이라고 쓴다. 표본평균은 관측된 결과이므로 검정의 대상이 아니라 검정에 쓰는 증거다.

### 2. 검정 전에 필요한 통계학 개념

가설검정은 새로운 절차처럼 보이지만, 실제로는 5강의 분포 개념을 사용한다.

| 개념 | 검정에서 하는 일 | 쉬운 해석 |
|---|---|---|
| 모수 | 가설의 대상 | 알고 싶은 모집단의 진짜 기준값 |
| 통계량 | 표본 증거 | 데이터에서 계산한 판단 숫자 |
| 표본분포 | 기준 분포 | 귀무가설이 맞을 때 통계량이 흔들리는 모양 |
| 표준오차 | 흔들림의 크기 | 표본 통계량이 우연히 움직일 수 있는 정도 |
| 검정통계량 | 비교용 점수 | 표본 결과가 기준에서 몇 표준오차만큼 떨어졌는지 |

검정통계량은 보통 다음 모양으로 만든다.

```text
검정통계량 = (통계량 - 귀무가설의 기준값) / 표준오차
```

단일 모평균에서 모분산을 알면 다음과 같다.

```text
Z = (Xbar - mu0) / (sigma / sqrt(n))
```

모분산을 모르면 `sigma` 대신 표본표준편차 `S`를 쓰고, 표준정규분포보다 꼬리가 두꺼운 t분포를 기준으로 삼는다.

```text
t = (Xbar - mu0) / (S / sqrt(n))
```

초심자 관점에서는 “왜 나누는가?”가 중요하다. `Xbar-mu0`만 보면 단위와 표본 수의 영향을 받는다. 2ml 차이가 큰지 작은지는 측정 변동이 0.5ml인지 20ml인지에 따라 달라진다. 표준오차로 나누면 “우연한 흔들림 몇 개만큼 떨어졌는가”로 바뀌므로 기준 분포와 비교할 수 있다.

### 3. p-value와 유의수준

p-value가 작다는 것은 귀무가설이 맞는 세계에서는 지금 결과가 드물다는 뜻이다. 유의수준 alpha는 “이 정도로 드물면 기본 주장을 버리자”는 기준이다. alpha가 0.05라면 p-value가 0.05보다 작을 때 기각한다.

검정에서는 두 종류의 잘못된 결론을 구분한다. 1종 오류는 참인 귀무가설을 기각하는 오류이고, 2종 오류는 거짓인 귀무가설을 기각하지 못하는 오류다. 유의수준은 보통 1종 오류를 허용할 최대 위험으로 정한다.

| 실제 상태 | 검정 결론 | 이름 | 쉬운 의미 |
|---|---|---|---|
| `H0` 참 | `H0` 기각하지 않음 | True Negative | 기본 주장을 맞게 유지 |
| `H0` 참 | `H0` 기각 | False Positive, 1종 오류 | 없는 효과를 있다고 판단 |
| `H0` 거짓 | `H0` 기각하지 않음 | False Negative, 2종 오류 | 있는 효과를 놓침 |
| `H0` 거짓 | `H0` 기각 | True Positive | 변화를 맞게 발견 |

p-value는 다음처럼 읽는다.

```text
p-value = P(observed evidence or more extreme evidence | H0 is true)
```

귀무가설이 여러 모수값을 포함하는 경우에는, 귀무가설을 만족하는 모수 공간에서 이런 꼬리확률의 최댓값을 기준으로 본다.

```text
p-value = max_{theta in Theta0} P_theta(observed or more extreme evidence)
```

초보자 단계에서는 “귀무가설이 맞는 세계에서 지금 관측값 이상으로 대립가설에 유리한 결과가 나올 확률”로 이해하면 된다.

실제로는 “관측된 검정통계량보다 대립가설 방향으로 더 극단적인 값이 나올 확률”을 계산한다.

```text
좌측검정: p-value = P(T <= t_obs | H0)
우측검정: p-value = P(T >= t_obs | H0)
양측검정: p-value = P(|T| >= |t_obs| | H0)  대칭 분포일 때
```

여기서 `T`는 귀무가설이 맞다고 가정했을 때의 검정통계량이고, `t_obs`는 실제 표본에서 계산한 검정통계량이다. 같은 숫자라도 대립가설이 `mu<mu0`인지, `mu>mu0`인지, `mu!=mu0`인지에 따라 꼬리 방향이 달라진다.

![p-value 계산의 직관](assets/p-value-intuition-154.png)

> **그림 읽기**: 관측 결과보다 더 극단적인 영역의 면적을 본다. 귀무가설이 맞다고 놓았을 때 얼마나 드문 결과인지가 핵심이다.

### 4. 검정통계량과 기각역

표본평균, 표본분산, 표본비율은 그대로 비교하기 어렵다. 그래서 표준화된 <a id="ref-06-검정통계량"></a>[검정통계량](#note-06-검정통계량)으로 바꾼다. 기각역은 그 통계량이 어느 영역에 들어가면 귀무가설을 버릴지 미리 정한 구간이다.

기각역 방식은 p-value 방식과 같은 정보를 다른 순서로 읽는다. 유의수준 `alpha`가 0.05인 좌측검정이라면 기준 분포의 왼쪽 5% 지점을 먼저 찾는다. 관측 검정통계량이 그 경계보다 더 왼쪽에 있으면, p-value도 0.05보다 작다.

![기각역과 유의수준](assets/rejection-region-164.png)

> **그림 읽기**: 검정 방향에 따라 꼬리 영역이 달라지는 모습을 본다. 질문이 한쪽인지 양쪽인지가 기각역의 위치를 정한다.

### 5. 가설 검정 과정 I과 II

가설 검정 과정은 같은 판단을 두 가지 방식으로 정리할 수 있다.

| 방식 | 핵심 판단 | 절차 |
|---|---|---|
| 과정 I | p-value를 유의수준과 비교한다. | 가설 설정 -> 유의수준 설정 -> 검정방법 선정 -> 검정통계량 계산 -> p-value 계산 -> 결론 |
| 과정 II | 검정통계량이 기각역에 들어가는지 본다. | 가설 설정 -> 유의수준 설정 -> 검정방법 선정 -> 기각역 계산 -> 검정통계량 계산 -> 결론 |

두 과정은 결론이 같아야 한다. 과정 I은 “관측값보다 더 극단적인 영역의 확률”을 계산하고, 과정 II는 “그 정도로 극단적인 영역의 경계”를 먼저 계산한다.

우유 예시를 다시 보면, 귀무가설은 `H0: mu = 200`, 대립가설은 `H1: mu < 200`이다. 분산이 4이고 표본 수가 8이면 표본평균의 분산은 `4/8`이다. 관측 표본평균이 198.625라면 왼쪽 꼬리 p-value가 약 0.0259가 되고, 유의수준 5%보다 작으므로 귀무가설을 기각한다. 같은 결론을 과정 II로는 5% 왼쪽 기각역 경계와 관측 표본평균을 비교해 얻는다.

계산 흐름을 수식으로 쓰면 다음과 같다.

```text
관측값: 205, 202, 195, 193, 197, 201, 197, 199
Xbar = (205+202+195+193+197+201+197+199) / 8 = 198.625

H0 아래에서 X ~ N(200, 4)
Xbar ~ N(200, 4/8)

Z = (198.625 - 200) / sqrt(4/8)
  = -1.9445
```

대립가설이 `mu < 200`이므로 왼쪽 꼬리 면적을 본다. 이 면적이 p-value이고 약 `0.0259`다.

### 6. 단측검정과 양측검정

작아졌는지만 보는 검정은 좌측 단측검정, 커졌는지만 보는 검정은 우측 단측검정이다. 달라졌는지만 보면 양측검정이다. 질문의 방향을 사후에 바꾸면 유리한 결론을 고르는 문제가 생긴다.

![대칭 분포에서의 양측 꼬리 검정](assets/two-tailed-test-165.png)

> **그림 읽기**: 중앙 기준 양쪽 꼬리를 함께 보는 이유를 본다. 방향을 특정하지 않으면 양쪽 극단이 모두 대립가설의 증거가 된다.

### 7. 검정통계량과 기준 분포

평균 검정, 비율 검정, 분산 검정에서는 어떤 검정통계량과 분포를 쓰는지 구분한다. 표본에서 계산한 숫자는 혼자서는 의미가 없고, 귀무가설이 참일 때의 기준 분포와 비교해야 의미가 생긴다.

| 검정 대상 | 대표 검정통계량 | 기준 분포를 고를 때 보는 것 |
|---|---|---|
| 단일 모평균, 모분산을 아는 경우 | `Z = (Xbar - mu0) / (sigma / sqrt(n))` | 모집단 정규성 또는 충분한 표본 수 |
| 단일 모평균, 모분산을 모르는 경우 | `t = (Xbar - mu0) / (S / sqrt(n))` | 모집단 정규성, 자유도 `n - 1` |
| 독립 두 모평균, 모분산을 아는 경우 | `Z = (Xbar1 - Xbar2) / sqrt(sigma1^2/n1 + sigma2^2/n2)` | 두 표본 독립, 모집단 정규성 |
| 독립 두 모평균, 등분산을 가정하고 모분산을 모르는 경우 | `t = (Xbar1 - Xbar2) / sqrt(Sp^2(1/n1 + 1/n2))` | 자유도 `n1 + n2 - 2` |
| 독립 두 모평균, 이분산을 가정하고 모분산을 모르는 경우 | `t = (Xbar1 - Xbar2) / sqrt(S1^2/n1 + S2^2/n2)` | Welch 근사 자유도 |
| 대응 표본 평균 | 각 쌍의 차이 `D`의 평균을 t 통계량으로 표준화 | 같은 대상의 전후 차이, 자유도 `n - 1` |
| 모비율 | `Z = (phat - p0) / sqrt(p0(1 - p0) / n)` | 성공/실패 표본 수가 충분한지 |
| 단일 모분산 | `(n - 1)S^2 / sigma0^2` | 정규 모집단, 카이제곱분포 |
| 두 모분산 | `F = S1^2 / S2^2` | 두 정규 모집단, F분포의 분자/분모 자유도 |

등분산 독립 두 표본 평균 검정에서 합동분산은 다음처럼 계산한다.

```text
Sp^2 = ((n1 - 1)S1^2 + (n2 - 1)S2^2) / (n1 + n2 - 2)
```

대응 표본 평균 검정은 두 집단을 따로 비교하지 않고, 같은 대상의 전후 차이 `D = X1 - X1'`를 만든 뒤 차이의 평균을 검정한다.

```text
t = Dbar / (Sd / sqrt(n)) ~ t(n - 1)
```

양측 검정은 “크거나 작거나 둘 다 이상한가”를 묻고, 단측 검정은 “특정 방향으로만 이상한가”를 묻는다. 비대칭 분포에서는 양쪽 꼬리를 단순히 좌우 대칭으로 나눌 수 없으므로, 기각역이 어떤 확률을 차지하는지 기준으로 봐야 한다.

대칭 분포인 표준정규분포나 t분포에서는 양측 p-value를 한쪽 꼬리 면적의 두 배로 계산할 수 있다. 비대칭 분포에서는 반대쪽의 같은 위치를 단순히 찾기 어려우므로, 작은 꼬리 면적을 기준으로 근사하거나 검정별 정의된 양측 기각역을 사용한다.

대칭 분포의 양측검정은 다음처럼 쓸 수 있다.

```text
p-value = P(X < -|test statistic|) * 2
```

비대칭 분포에서는 절댓값만으로 반대쪽 같은 위치를 만들 수 없으므로 다음처럼 작은 꼬리 면적을 두 배로 하는 근사를 쓰거나, 해당 검정에서 정한 양측 기각역을 사용한다.

```text
p-value approximately min(P(X < observed), P(X > observed)) * 2
```

예를 들어 맥주 알코올 도수 평균이 5%에서 크거나 작으면 품질검사에서 떨어진다는 질문은 양측검정이다. 표본 수가 10이고 모분산을 모르면 자유도 9의 t분포를 사용한다. 유의수준 5%에서는 양쪽 꼬리에 2.5%씩 배정하므로 표본평균이 `5.0 ± t(0.975, 9) * S / sqrt(10)` 밖에 있으면 기각역에 들어간다.

### 8. 결론을 쓰는 법

검정 결론은 “증명했다”가 아니라 “귀무가설을 기각한다/기각하지 못한다”로 쓴다.

| 상황 | 결론 문장 | 뜻 |
|---|---|---|
| `p-value < alpha` | 귀무가설을 기각한다. | 귀무가설 아래에서는 관측 결과가 충분히 드물다. |
| `p-value >= alpha` | 귀무가설을 기각하지 못한다. | 귀무가설을 버릴 만큼 강한 증거가 부족하다. |

`귀무가설을 기각하지 못한다`는 말은 `귀무가설이 참임을 증명했다`는 뜻이 아니다. 표본이 충분한 반박 증거를 주지 못했다는 뜻이다. 표본 수가 작거나 변동이 크면 실제 차이가 있어도 기각하지 못할 수 있다.

결론에는 통계적 유의성과 실제 중요도를 분리해 적는다. 예를 들어 p-value가 작아도 평균 차이가 0.01점이라면 실무적으로는 중요하지 않을 수 있다. 반대로 p-value가 0.05보다 크더라도 표본 수가 너무 작아 결론을 내리기 어려운 상황일 수 있다.

## 판단 기준

1. 검정하려는 질문을 먼저 한 문장으로 쓴다.
2. 귀무가설과 <a id="ref-06-대립가설"></a>[대립가설](#note-06-대립가설)을 방향까지 포함해 정한다.
3. 검정통계량이 어떤 <a id="ref-06-분포"></a>[분포](#note-06-분포)를 기준으로 판단되는지 확인한다.
4. 유의수준은 데이터를 보기 전에 정하고 p-value와 비교한다.
5. 통계적 유의성과 실제 효과 크기를 분리해서 해석한다.
6. 결론은 “기각” 또는 “기각하지 못함”으로 쓰고, “참으로 증명”이라고 쓰지 않는다.

## 오해와 반례

### 오해 1. p-value는 귀무가설이 참일 확률이다.

p-value는 귀무가설이 참이라고 가정했을 때 표본 결과가 얼마나 드문지다.

### 오해 2. p-value가 작으면 효과가 크다.

표본 수가 크면 작은 효과도 유의할 수 있다. 실제 중요도는 효과 크기와 함께 봐야 한다.

### 오해 3. 유의수준 0.05는 항상 정답이다.

관례일 뿐이다. 의료, 안전처럼 비용이 큰 문제에서는 기준을 더 엄격하게 둘 수 있다.

## 예시 풀이

### 예시 1. 우유 용량이 200ml보다 작은가?

귀무가설은 평균이 200ml 이상 또는 200ml라고 둘 수 있고, 대립가설은 평균이 200ml보다 작다는 주장이다. 질문이 “작은가”이므로 좌측 단측검정이 자연스럽다.

### 예시 2. 새 교육법의 평균 점수가 기존과 다른가?

높아졌는지 낮아졌는지 방향을 정하지 않았다면 양측검정이다. 표본평균 차이를 검정통계량으로 바꾸고 양쪽 꼬리의 극단성을 본다.

## 오늘의 요약 5줄

1. 가설 검정은 표본의 차이가 우연으로도 충분히 설명되는지 따져 보는 절차다.
2. 귀무가설은 일단 맞다고 놓는 기본 주장이고, 대립가설은 데이터로 보이고 싶은 주장이다.
3. 가설은 모수에 대해 세우고, 표본 통계량은 그 가설을 판단하는 증거로 쓴다.
4. p-value는 귀무가설이 맞다고 가정한 세계에서 지금 결과가 얼마나 드문지를 뜻한다.
5. 검정 결론은 귀무가설을 기각하거나 기각하지 못한다고 쓴다.
