# 19. 퍼셉트론 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 퍼셉트론의 입력, 가중치, 편향, 출력 흐름을 설명하라.

입력값마다 가중치를 곱하고 모두 더한 뒤 편향을 더한다. 그 결과를 활성화 함수에 넣어 최종 출력이나 판단을 만든다. 학습은 이 가중치와 편향을 손실이 줄어드는 방향으로 조정하는 과정이다.

## 2. 단층 퍼셉트론이 XOR 문제를 풀기 어려운 이유를 설명하라.

단층 퍼셉트론은 하나의 직선이나 초평면으로 나눌 수 있는 패턴에 적합하다. XOR은 한 직선으로 두 클래스를 분리할 수 없는 대표 예다. 은닉층과 비선형 활성화가 있어야 이런 패턴을 표현할 수 있다.

## 3. 은닉층이 필요한 이유를 설명하라.

은닉층은 입력을 새로운 중간 표현으로 바꾼다. 여러 은닉층과 비선형 활성화를 사용하면 단순한 직선보다 복잡한 결정경계를 만들 수 있다. 그래서 이미지, 텍스트처럼 복잡한 패턴을 다룰 수 있다.

## 4. 활성화 함수가 없으면 생기는 문제를 설명하라.

활성화 함수가 없으면 여러 선형층을 쌓아도 전체는 하나의 선형변환과 같다. 즉 층을 많이 쌓아도 표현력이 크게 늘지 않는다. ReLU, sigmoid 같은 비선형 함수가 있어야 복잡한 함수를 만들 수 있다.

## 5. 역전파의 기본 직관을 설명하라.

역전파는 최종 오차가 각 가중치에 얼마나 책임이 있는지 뒤쪽 층부터 계산하는 방법이다. 이 책임을 기울기로 표현하고, 경사하강법이 그 기울기를 사용해 가중치를 고친다. 핵심은 연쇄법칙으로 오차 정보를 뒤로 전달한다는 점이다.

## 6. 기울기 소실이 학습을 어렵게 하는 이유를 설명하라.

깊은 네트워크에서 뒤쪽의 오차 신호가 앞쪽 층으로 전달되며 매우 작아질 수 있다. 그러면 앞쪽 층의 가중치가 거의 업데이트되지 않아 학습이 느려지거나 멈춘다. Sigmoid나 tanh는 양끝에서 기울기가 작아 이 문제가 생기기 쉽고, ReLU는 이를 완화하지만 dead neuron 문제가 생길 수 있다.

## 7. 왜 활성화 함수가 없으면 층을 많이 쌓아도 의미가 약한가?

- **한 문장 답**: 선형 변환을 여러 번 합쳐도 결국 하나의 선형 변환처럼 표현될 수 있기 때문이다.
- **이유**: 선형 함수들의 합성은 다시 선형 함수라 복잡한 곡선 경계를 만들지 못한다. 활성화 함수가 비선형성을 넣어 층을 쌓는 의미를 만든다.
- **예시**: 직선을 여러 번 적용해도 비선형 곡선 경계를 만들기 어렵다.
- **주의점**: 비선형 활성화가 있어야 층을 쌓는 표현력이 생긴다.

## 8. 왜 역전파는 오차를 뒤로 보낸다고 표현하는가?

- **한 문장 답**: 출력의 손실이 각 가중치에 얼마나 책임이 있는지 뒤쪽 층부터 연쇄적으로 계산하기 때문이다.
- **이유**: 손실은 출력에서 계산되지만 각 가중치의 책임은 앞층까지 연쇄 법칙으로 나눠 계산해야 한다. 그래서 계산 흐름이 출력층에서 입력 쪽으로 거슬러 올라간다.
- **예시**: 마지막 예측이 틀렸다면 출력층 가중치부터 앞쪽 가중치까지 수정 신호를 보낸다.
- **주의점**: 오차 값 자체를 복사하는 것이 아니라 기울기를 계산해 전달한다.

## 9. Sigmoid, tanh, ReLU의 장단점을 비교하라.

Sigmoid는 0과 1 사이 값을 내므로 확률처럼 보기 쉽지만 양끝에서 기울기가 작아진다. tanh는 -1과 1 사이 값을 내고 중심이 0에 가까워 sigmoid보다 나은 경우가 있지만 포화 문제는 남아 있다. ReLU는 양수 구간에서 기울기가 일정해 학습이 빠르지만 음수 구간에서 뉴런이 죽을 수 있다.

## 10. 다층 퍼셉트론의 파라미터 수를 계산하는 방법을 설명하라.

각 Dense 층마다 입력 노드 수와 출력 뉴런 수를 곱해 가중치 수를 구한다. 여기에 출력 뉴런 수만큼 편향을 더한다. 전체 파라미터 수는 모든 층의 `입력 수 x 출력 수 + 출력 수`를 합한 값이다.

## 11. Heaviside Step Function과 현재 이진분류 출력층의 차이를 설명하라.

Heaviside Step Function은 입력이 0 이상이면 1, 0보다 작으면 0을 내는 딱딱한 계단 함수다. 초기 퍼셉트론의 이진 판단에 쓰였지만 확률을 직접 주지는 않는다. 현재 이진분류 출력층은 보통 sigmoid를 사용해 `P(y = 1)`처럼 0과 1 사이 확률을 출력한다.

## 12. 입력층, 은닉층, 출력층의 역할을 구분하라.

입력층은 원시 데이터를 받아 다음 층으로 전달하며 보통 학습 연산을 하지 않는다. 은닉층은 입력을 변환해 고차원적 특징이나 중간 표현을 만든다. 출력층은 문제 유형에 맞게 최종 예측을 만든다. 이진분류는 sigmoid, 다중분류는 softmax, 회귀는 값 자체를 출력하는 식으로 달라진다.

## 13. 다중분류에서 초기 퍼셉트론 방식과 현재 softmax 방식의 차이를 설명하라.

초기 퍼셉트론은 여러 이진 분류기를 조합하는 One-vs-One 또는 One-vs-Rest 방식으로 다중분류에 대응했다. 현재 신경망 출력층에서는 클래스 수만큼 출력을 만들고 softmax를 적용해 각 클래스 확률을 한 번에 계산하는 방식이 일반적이다.

## 14. ReLU에서 Dead Neuron이 생기는 이유와 Leaky ReLU의 보완점을 설명하라.

ReLU는 입력이 음수이면 출력과 기울기가 0이다. 어떤 뉴런이 학습 과정에서 계속 음수 영역에 머물면 모든 학습 데이터에 대해 기울기가 0이 되어 더 이상 업데이트되지 않는 dead neuron이 될 수 있다. Leaky ReLU는 음수 구간에서도 작은 기울기를 남겨 뉴런이 완전히 죽는 위험을 줄인다.

## 15. 입력 2개, 은닉 뉴런 3개, 출력 뉴런 1개인 다층 퍼셉트론의 총 파라미터 수를 계산하라.

은닉층은 입력 2개와 편향 1개가 각 은닉 뉴런 3개에 연결되므로 `(2 + 1) x 3 = 9`개다. 출력층은 은닉 뉴런 3개와 편향 1개가 출력 뉴런 1개에 연결되므로 `(3 + 1) x 1 = 4`개다. 총 파라미터 수는 `9 + 4 = 13`개다.

## 16. Sigmoid, tanh, ReLU, Leaky ReLU, softmax의 공식을 쓰고 출력 범위를 설명하라.

```text
Sigmoid: sigma(x) = 1 / (1 + exp(-x)), output in (0, 1)
tanh: tanh(x) = (exp(x) - exp(-x)) / (exp(x) + exp(-x)), output in (-1, 1)
ReLU: f(x) = max(0, x), output in [0, infinity)
Leaky ReLU: f(x) = x if x >= 0, else alpha*x
softmax(z)_k = exp(z_k) / sum_j exp(z_j)
```

sigmoid는 이진분류 확률 출력에 자주 쓰이고, softmax는 다중분류에서 클래스별 확률을 만든다. tanh는 중심이 0이라 sigmoid보다 학습이 안정적인 경우가 있다. ReLU는 양수 구간에서 기울기가 일정하지만 음수 구간은 0이고, Leaky ReLU는 음수 구간에도 작은 기울기를 남긴다.

## 17. 단층 퍼셉트론의 선형분리 가능 패턴 수 `T(N, K)` 식이 뜻하는 바를 설명하라.

`T(N, K)`는 `N`개의 점을 `K`차원에서 선형분리할 수 있는 라벨 배정의 수를 나타낸다.

```text
T(N, K) = 2^N, if K >= N
T(N, K) = 2 * sum_{k=0}^{K-1} C(N-1, k), if K < N
```

가능한 전체 라벨 배정은 `2^N`개다. 따라서 `T(N, K) / 2^N`은 가능한 패턴 중 단층 퍼셉트론이 표현할 수 있는 비율이다. 이 비율이 작아질수록 단층 퍼셉트론으로 표현할 수 없는 패턴이 많아진다. XOR은 이런 한계를 보여 주는 대표 예다.

## 18. n층 퍼셉트론 역전파의 `delta`, 가중치 기울기, 편향 기울기 식을 쓰라.

출력층에서는 손실을 출력 활성값으로 미분한 값에 활성화 함수의 도함수를 곱해 `delta`를 만든다.

```text
delta^(n) = f^(n)'(z^(n)) o gradient_{a^(n)} L
```

앞층으로는 다음 층 가중치를 통해 전달한다.

```text
delta^(n-1) = f^(n-1)'(z^(n-1)) o (W^(n))^T delta^(n)
```

각 층의 가중치 기울기와 편향 기울기는 다음처럼 계산한다.

```text
gradient_{W^(n)} L = delta^(n) (a^(n-1))^T
gradient_{b^(n)} L = delta^(n)
gradient_{W^(1)} L = delta^(1) X^T
gradient_{b^(1)} L = delta^(1)
```

이 식은 출력층에서 계산한 오차 신호를 연쇄 법칙으로 앞층까지 보내 각 가중치의 수정 방향을 구한다는 뜻이다.

## 19. 단층 퍼셉트론에서 `z = 0`이 결정 경계가 되는 이유를 설명하라.

단층 퍼셉트론은 먼저 `z = sum_i w_i x_i + b`를 계산하고, 이 값을 활성화 함수에 넣어 출력을 만든다. Heaviside 스텝 함수를 쓰면 `z >= 0`이면 1, `z < 0`이면 0이 된다.

따라서 출력이 바뀌는 경계는 `z = 0`이다. 입력이 두 개면 이 경계는 직선이고, 입력이 더 많으면 초평면이 된다. 이 경계 한쪽은 1로, 다른 쪽은 0으로 판단된다.

## 20. 편향 `b`가 없을 때 결정 경계 표현이 제한되는 이유를 설명하라.

편향이 없으면 결정 경계는 `sum_i w_i x_i = 0`이 된다. 입력이 두 개인 경우 이 직선은 항상 원점을 지난다. 즉 모델이 경계의 방향은 바꿀 수 있지만, 경계를 원점에서 떨어진 위치로 평행 이동시키기 어렵다.

편향 `b`는 기준선을 움직이는 역할을 한다. 그래서 실제 데이터의 분리 기준이 원점을 지나지 않을 때도 더 자연스럽게 판단 경계를 맞출 수 있다.

## 21. 역전파 식에서 작은 도함수가 반복해서 곱해질 때 앞쪽 층 학습이 어려워지는 이유를 설명하라.

역전파는 출력층에서 입력층 방향으로 기울기를 전달한다. 이때 앞층의 `delta`는 다음 층의 가중치, 활성화 함수의 도함수, 뒤쪽에서 온 `delta`가 곱해진 형태로 계산된다.

Sigmoid나 tanh처럼 포화 구간에서 도함수가 0에 가까운 함수가 여러 층에서 반복되면, 작은 값들이 계속 곱해져 기울기가 매우 작아진다. 그러면 앞쪽 층의 가중치 업데이트가 거의 일어나지 않아 학습이 느려지거나 멈출 수 있다.
