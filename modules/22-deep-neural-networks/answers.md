# 22. 심층 신경망 · 확인 문제 해설

## 해설 읽는 법

기본 해설은 먼저 핵심 문장을 잡고, 스스로 답할 때는 **한 문장 답 → 이유 → 예시 → 주의점** 순서로 확장한다.

## 1. 심층 신경망의 단계적 추상화를 설명하라.

깊은 신경망은 앞쪽 층에서 단순한 패턴을 잡고 뒤쪽 층으로 갈수록 더 복잡한 개념을 조합한다. 이미지에서는 선과 모서리에서 물체 일부, 전체 물체로 이어질 수 있다. 이것을 단계적 추상화라고 이해할 수 있다.

## 2. CNN에서 깊은 층이 더 복잡한 특징을 잡는다는 뜻을 설명하라.

초기 convolution 필터는 작은 영역의 선, 색 변화, 모서리 같은 단순 특징에 반응한다. 여러 층을 거치면 이런 특징들이 조합되어 눈, 바퀴, 질감 같은 더 복잡한 패턴을 나타낼 수 있다. 깊은 층일수록 입력값 자체보다 의미에 가까운 표현을 담는다.

## 3. Batch Normalization이 학습을 안정화하는 이유를 설명하라.

Batch Normalization은 층의 입력을 미니배치 기준으로 정규화하고 다시 학습 가능한 scale과 shift를 적용한다. 값의 범위가 지나치게 흔들리지 않으면 기울기 흐름이 안정적이고 더 큰 학습률도 쓰기 쉬워진다. 훈련과 추론에서 사용하는 통계가 다르다는 점도 알아야 한다.

## 4. Word2Vec이 단어를 벡터로 표현하는 직관을 설명하라.

Word2Vec은 비슷한 문맥에 등장하는 단어는 비슷한 의미를 가진다는 가정에서 출발한다. 주변 단어를 맞히거나 중심 단어를 맞히는 과정을 통해 단어 벡터를 학습한다. 그래서 의미가 비슷한 단어들이 벡터 공간에서 가까워질 수 있다.

## 5. 사전학습과 전이학습의 차이를 설명하라.

사전학습은 큰 데이터로 모델이 일반적인 표현을 먼저 배우는 단계다. 전이학습은 그 배운 표현을 새로운 문제에 가져와 활용하는 것이다. 새 데이터가 적을 때도 좋은 출발점을 얻을 수 있다는 장점이 있다.

## 6. 미세조정할 때 일부 층을 고정하는 이유를 설명하라.

새 데이터가 적으면 전체 모델을 모두 학습할 때 이미 배운 유용한 표현이 망가지거나 과대적합될 수 있다. 앞쪽 층을 고정하면 일반적인 특징 추출 능력을 유지하고, 뒤쪽 층만 새 문제에 맞게 조정할 수 있다. 데이터가 많고 도메인이 다르면 더 많은 층을 풀어 학습하기도 한다.

## 7. 왜 깊은 신경망은 단순히 층이 많다는 것 이상으로 이해해야 하는가?

- **한 문장 답**: 깊은 층은 낮은 수준 특징을 점점 더 추상적인 표현으로 조합하기 때문이다.
- **이유**: 깊이는 단순한 층 개수보다 표현이 단계적으로 바뀌는 과정이다. 각 층이 이전 층의 특징을 조합해 더 의미 있는 특징을 만들 수 있어야 깊이의 장점이 생긴다.
- **예시**: 이미지에서 앞층은 선과 모서리, 뒤층은 물체 일부와 전체 형태를 잡을 수 있다.
- **주의점**: 깊어질수록 학습 불안정과 계산 비용도 커진다.

## 8. 왜 전이학습은 적은 데이터 문제에 도움이 되는가?

- **한 문장 답**: 큰 데이터에서 배운 일반적인 표현을 새 문제의 출발점으로 사용할 수 있기 때문이다.
- **이유**: 새 데이터가 적으면 처음부터 모든 패턴을 배우기 어렵다. 이미 큰 데이터에서 배운 일반 특징을 가져오면 추정해야 할 것이 줄어든다.
- **예시**: ImageNet으로 배운 CNN 특징을 의료 이미지 분류의 초기 특징 추출기로 쓸 수 있다.
- **주의점**: 원래 학습 도메인과 새 도메인이 너무 다르면 효과가 줄 수 있다.

## 9. Gram matrix가 스타일 표현에서 어떤 정보를 담는지 설명하라.

Gram matrix는 여러 필터 반응이 서로 얼마나 함께 나타나는지 요약한다. 위치를 정확히 보존하기보다 질감, 반복 패턴, 색감처럼 스타일에 가까운 통계적 관계를 담는다. 그래서 content 표현과 구분해 style 표현을 다룰 때 사용된다.

## 10. 오토인코더와 잡음 제거 오토인코더의 차이를 설명하라.

오토인코더는 입력을 압축한 뒤 다시 복원하도록 학습한다. 잡음 제거 오토인코더는 손상되거나 노이즈가 섞인 입력을 넣고 깨끗한 입력을 복원하게 만든다. 이 과정에서 단순 복사보다 더 안정적인 핵심 표현을 배우도록 유도한다.

## 11. AlexNet이 당시 비전 분야에서 주목받은 이유를 설명하라.

AlexNet은 2012년 ILSVRC에서 큰 성능 차이로 우승하며 깊은 CNN이 기존 비전 방법을 넘어설 수 있음을 보였다. ReLU로 기울기 소실을 줄이고, Dropout과 데이터 증강으로 과적합에 대응했으며, GPU 병렬 처리를 활용해 당시로서는 큰 신경망을 학습했다. 그래서 신경망 연구가 다시 크게 주목받는 계기가 되었다.

## 12. Inception의 1x1 합성곱과 보조 중간 출력 레이어의 역할을 설명하라.

1x1 합성곱은 채널 방향 정보를 섞고 채널 수를 줄여 연산량을 낮추는 데 쓰인다. 동시에 여러 크기의 필터를 병렬로 쓰는 Inception 블록 안에서 정보를 효율적으로 통합한다. 보조 중간 출력 레이어는 깊은 네트워크 중간에도 학습 신호를 주어 기울기 소실을 완화하는 역할을 한다.

## 13. 학습 때와 추론 때 Batch Normalization이 사용하는 통계가 어떻게 다른가?

학습 때는 현재 미니배치의 평균과 분산으로 정규화하고, 그 과정에서 이동평균 통계도 함께 갱신한다. 추론 때는 입력 배치마다 통계를 새로 추정하지 않고 학습 중 저장한 이동평균 평균과 분산을 사용한다. 이후 두 경우 모두 학습된 gamma와 beta로 scale과 shift를 적용한다.

## 14. GAN의 생성자와 판별자가 각각 무엇을 학습하는지 설명하라.

생성자는 가짜 데이터를 만들어 판별자가 진짜와 구분하지 못하게 하는 방향으로 학습한다. 판별자는 진짜 데이터와 생성자가 만든 가짜 데이터를 더 잘 구분하도록 학습한다. 두 모델이 번갈아 고정되고 학습되면서 생성 데이터의 품질이 개선되는 구조다.

## 15. 사전학습, 전이학습, 미세조정의 차이를 구분하라.

사전학습은 대규모 범용 데이터로 일반적인 표현을 먼저 배우는 단계다. 전이학습은 그렇게 배운 모델이나 표현을 새로운 데이터와 태스크에 가져와 활용하는 방식이다. 미세조정은 가져온 모델의 일부 또는 전체를 낮은 학습률로 추가 학습해 새 문제에 맞게 조정하는 과정이다.

## 16. ResNet의 잔차 연결을 `y = F(x) + x` 관점에서 설명하라.

ResNet 블록은 입력 `x`를 몇 개 층의 출력에 다시 더한다. 그래서 블록 전체의 출력은 `y = F(x) + x`처럼 볼 수 있다. 여기서 `F(x)`는 입력이 그대로 전달될 때 부족한 변화량, 즉 잔차를 의미한다.

이 구조에서는 층이 원하는 출력 전체를 처음부터 배우는 것이 아니라, 입력에서 얼마나 달라져야 하는지만 배우면 된다. 입력과 기울기가 지나갈 지름길이 생기므로 깊은 층에서도 학습이 더 안정적이다.

## 17. Batch Normalization의 학습 통계와 추론 통계를 식으로 구분하라.

학습 때는 현재 미니배치의 평균과 분산을 쓴다.

```text
mu = (1 / N) * sum_j x_j
sigma^2 = (1 / N) * sum_j (x_j - mu)^2
x_hat = (x - mu) / sqrt(sigma^2 + epsilon)
y = gamma * x_hat + beta
```

동시에 이동평균 통계를 갱신한다.

```text
mu_mov = alpha * mu_mov + (1 - alpha) * mu
sigma_mov^2 = alpha * sigma_mov^2 + (1 - alpha) * sigma^2
```

추론 때는 현재 입력 배치의 통계를 새로 쓰지 않고, 학습 중 저장한 `mu_mov`, `sigma_mov^2`를 사용한다. 이후 `gamma`, `beta`로 scale과 shift를 적용하는 것은 같다.

## 18. CBOW와 Skip-gram의 입력과 출력을 구분하라.

CBOW는 주변 단어들을 입력으로 받아 중심 단어를 예측한다. 예를 들어 `W_{t-1}`, `W_{t+1}` 같은 문맥 단어가 입력이고, 출력은 `W_t`의 확률분포다.

Skip-gram은 중심 단어를 입력으로 받아 주변 단어들을 예측한다. 입력은 `W_t`이고, 출력은 `W_{t-1}`, `W_{t+1}` 같은 주변 단어의 확률분포다. 둘 다 임베딩 레이어를 통해 단어를 연속 벡터로 학습한다.

## 19. GAN의 판별자 손실과 생성자 손실을 설명하라.

판별자 손실은 진짜 데이터는 진짜로, 생성 데이터는 가짜로 분류하도록 학습한다.

```text
L_D = -E_{x ~ p_train} log D(x) - E_{z ~ p_z(z)} log(1 - D(G(z)))
```

생성자 손실은 생성 데이터 `G(z)`가 판별자에게 진짜처럼 보이도록 학습한다.

```text
L_G = -E_{z ~ p_z(z)} log D(G(z))
```

즉 판별자는 구분 능력을 높이고, 생성자는 그 구분을 어렵게 만드는 방향으로 번갈아 학습한다.

## 20. 스타일 복원에서 content loss와 style loss가 각각 비교하는 대상을 설명하라.

Content loss는 content image의 특정 층 출력 `C_l`과 복원 이미지의 같은 층 출력 `R_l`이 얼마나 다른지 비교한다. 식으로는 `L_content_l = (1 / (P * C)) * sum_i sum_j (C_l[i,j] - R_l[i,j])^2`처럼 볼 수 있다. 이 손실은 복원 이미지가 원래 내용 구조를 유지하도록 만든다.

Style loss는 style image와 복원 이미지의 Gram matrix를 비교한다. `G_l`은 style image의 필터 반응 관계, `A_l`은 복원 이미지의 필터 반응 관계를 담고, `L_style_l`은 두 Gram matrix 차이를 줄인다. 그래서 위치 자체보다 질감, 반복 패턴, 색감 같은 스타일 통계를 맞추는 데 쓰인다.

## 21. ResNet의 잔차가 회귀의 잔차와 같은 단어지만 문맥상 다르게 쓰이는 이유를 설명하라.

회귀에서 잔차는 보통 `실제값 - 예측값`이다. 모델이 설명하지 못한 오차를 진단하는 데 쓴다.

ResNet에서 잔차는 블록이 입력 전체를 새로 만들기보다 입력에서 추가로 바뀌어야 하는 변화량 `F(x)`를 배우는 구조를 뜻한다. 출력은 `y = F(x) + x`처럼 입력 `x`를 그대로 더한 형태다. 같은 단어라도 회귀 진단에서는 오차, ResNet에서는 skip connection을 통한 변화량 학습으로 문맥이 다르다.

## 22. 전이학습에서 원래 학습 도메인과 새 도메인의 유사도를 확인해야 하는 이유를 설명하라.

전이학습은 원래 데이터에서 배운 표현을 새 문제에 가져오는 방법이다. 원래 도메인과 새 도메인이 비슷하면 앞쪽 층의 일반 특징을 재사용하기 쉽다. 예를 들어 일반 이미지에서 배운 선, 모서리, 질감 표현은 다른 일반 이미지 분류에도 도움이 될 수 있다.

도메인이 많이 다르면 가져온 표현이 새 데이터에 맞지 않을 수 있다. 이때는 더 많은 층을 미세조정하거나, 새 도메인에 더 가까운 사전학습 모델을 찾는 것이 필요하다. 새 데이터가 적으면 많은 층을 한꺼번에 풀 때 과적합 위험도 커진다.

## 23. GAN 학습을 하나의 손실이 안정적으로 내려가는 일반 지도학습처럼 보면 안 되는 이유를 설명하라.

GAN은 생성자와 판별자가 서로 다른 목표를 가지고 번갈아 학습한다. 판별자는 진짜와 가짜를 잘 구분하려 하고, 생성자는 판별자가 가짜를 진짜처럼 보게 만들려고 한다. 두 손실이 같은 방향으로 동시에 내려가는 단순한 최적화가 아니다.

판별자가 너무 강하면 생성자가 배울 신호가 약해질 수 있고, 생성자가 특정 출력만 반복하면 mode collapse가 생길 수 있다. 그래서 GAN은 손실 값 하나만 보지 않고 생성 결과의 품질, 다양성, 두 모델의 균형을 함께 확인해야 한다.
