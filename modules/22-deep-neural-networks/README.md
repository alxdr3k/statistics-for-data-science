이 차시를 마치면 다음을 쉬운 말로 설명할 수 있으면 충분하다.

- 깊은 층이 단계적 추상화를 만든다는 점을 이해한다.

## 오늘의 한 줄

심층 신경망은 여러 은닉층을 통해 낮은 수준의 신호를 점점 더 추상적인 표현으로 바꾼다.

## 오늘 반드시 이해할 3가지

1. 깊은 층이 단계적 추상화를 만든다는 점을 이해한다.
2. 대표 구조인 AlexNet, VGG, Inception, ResNet의 이름과 핵심 아이디어를 구분한다.
3. 사전학습, 전이학습, 미세조정이 왜 재사용성을 높이는지 설명한다.

## 이 차시 전에 알면 좋은 것

- **신경망 레이어**: 깊은 구조를 이루는 기본 블록
- **역전파**: 깊은 층까지 학습 신호가 전달되는 원리 ([처음 설명된 차시](../19-perceptron/README.md#4-역전파와-기울기-소실))
- **일반화**: 깊은 모델의 과적합과 전이학습 판단 기준

- **심화**: ResNet의 잔차 학습 직관

## 이 차시에서 꼭 붙잡을 설명 방식

깊은 네트워크가 특별한 이유는 층마다 표현이 달라지기 때문이다. 이미지에서는 앞쪽 층이 선이나 모서리를 보고, 뒤쪽 층이 눈, 얼굴, 물체 같은 더 추상적인 패턴을 볼 수 있다. 이 단계적 추상화가 깊은 모델의 힘이다.

## 핵심 이론

### 먼저 잡는 직관

- **대표 CNN 구조**: 깊은 CNN은 낮은 수준의 선과 색에서 시작해 점점 더 복잡한 패턴을 조합한다.
- **Batch Normalization**: 층마다 입력 분포가 크게 흔들리지 않도록 조정해 학습을 안정적으로 만든다.
- **Word2Vec과 표현 학습**: 단어의 의미를 주변 단어와의 관계 속에서 벡터 공간의 위치로 학습한다.
- **사전학습과 전이학습**: 큰 데이터에서 배운 표현을 새 문제에 가져와 적은 데이터에서도 시작점을 좋게 만든다.

### 1. 대표 CNN 구조

심층 신경망은 다수의 은닉층을 두어 입력과 출력 사이의 복잡한 관계를 단계적으로 추상화한다. 충분한 데이터가 있으면 고차원 문제에서 높은 정확도를 낼 수 있지만, 층이 깊어질수록 학습은 어려워지고 결과를 직접 설명하기도 어려워진다. 그래서 깊은 구조에서는 정확도뿐 아니라 학습 안정화, 과적합 대응, 재사용성을 함께 봐야 한다.

AlexNet은 2012년 ILSVRC에서 큰 격차로 우승하며 깊은 <a id="ref-22-cnn"></a>[CNN](#note-22-cnn)이 전통적인 비전 방법을 넘어설 수 있음을 보였다. LeNet 계열 아이디어를 더 큰 신경망으로 확장했고, ReLU로 기울기 소실을 줄이고, Dropout으로 과적합에 대응하고, 데이터 증강으로 일반화를 강화했으며, 당시에는 두 개의 GPU를 활용한 병렬 처리도 중요한 특징이었다.

VGGNets는 ILSVRC 2014 이미지 위치 지정 부문에서 주목받은 구조다. 큰 필터 하나보다 작은 3x3 필터를 반복해 깊게 쌓고, 필터 크기를 줄이는 대신 비선형성을 여러 번 통과시키는 방식으로 특징 추출을 강화한다. 구조가 단순해 VGG-16, VGG-19처럼 블록 수에 따라 구분하며 연구와 비교 기준으로 많이 쓰였다.

GoogLeNet/Inception은 ILSVRC 2014 이미지 분류 부문에서 우승한 구조로, 적은 수의 파라미터로 강한 성능을 내는 데 초점을 둔다. Inception 블록은 같은 입력에 여러 크기의 합성곱 필터를 병렬로 적용하고 결과를 결합한다. 1x1 합성곱은 채널 방향 정보를 섞으면서 채널 수를 줄여 연산량을 낮추고, 네트워크를 더 깊고 넓게 구성하는 데 도움을 준다. 보조 중간 출력 레이어는 깊은 층까지 학습 신호가 전달되도록 도와 기울기 소실을 완화한다.

ResNet은 2015년 ILSVRC에서 압도적인 성능을 보인 구조다. 핵심은 <a id="ref-22-잔차"></a>[잔차](#note-22-잔차) 연결이다. 층 전체가 \(x\)에서 출력으로 가는 변환을 처음부터 모두 배우는 대신, \(F(x)\)가 입력 \(x\)가 아직 설명하지 못한 차이만 배우도록 만든다. 이 덕분에 학습이 더 쉽고 안정적이며, 수백 층 수준의 매우 깊은 신경망을 구성할 수 있게 되었다.

잔차 연결은 보통 `y = F(x) + x`처럼 이해한다. 일반 블록이 원하는 출력 전체를 직접 배우는 대신, 입력 `x`를 그대로 더해 두고 블록은 필요한 변화량 `F(x)`만 학습한다. 깊은 층을 많이 쌓아도 입력과 기울기가 지나갈 길이 생기므로, 기울기 소실과 학습 난이도를 줄이는 데 도움이 된다.

여기서 잔차라는 말은 선형회귀에서 `실제값 - 예측값`으로 보던 잔차와 같은 단어지만, 쓰임은 다르다. ResNet에서는 몇 개 층이 입력 전체를 새로 만들기보다 입력에서 추가로 바뀌어야 하는 변화량을 배우는 구조를 뜻한다. 문제에서 잔차라는 단어가 나오면 회귀 진단인지, ResNet의 skip connection인지 문맥을 먼저 확인한다.

### 2. Batch Normalization

층 입력의 <a id="ref-22-분포"></a>[분포](#note-22-분포)를 안정화해 학습을 돕는다. 내부 분포 변화가 너무 크면 각 층이 계속 새 입력 환경에 적응해야 하므로 학습이 불안정해진다.

Batch Normalization은 배치 단위로 평균과 분산을 구한 뒤 값을 정규화하고, 다시 학습 가능한 scale과 shift를 적용한다.

```text
mu = (1 / N) * sum_j x_j
sigma^2 = (1 / N) * sum_j (x_j - mu)^2
x_hat = (x - mu) / sqrt(sigma^2 + epsilon)
y = gamma * x_hat + beta
```

학습 중에는 현재 미니배치의 평균과 분산을 사용하고, 동시에 이동평균 통계를 갱신한다. 추론 때는 새 입력 하나하나의 배치 통계를 쓰지 않고 학습 중 쌓아 둔 이동평균 평균과 분산을 사용한다. 이 차이를 모르면 훈련 때는 잘 보이던 모델이 추론에서 다르게 동작하는 이유를 이해하기 어렵다. BatchNorm은 기울기 소실 완화에도 도움이 되지만, Dropout처럼 노드를 임의로 꺼서 역할 집중을 막는 방법과는 목적이 다르다.

학습 중 저장하는 이동평균은 직전 이동평균과 현재 배치 통계를 섞어 갱신한다.

```text
mu_mov = alpha * mu_mov + (1 - alpha) * mu
sigma_mov^2 = alpha * sigma_mov^2 + (1 - alpha) * sigma^2
```

추론 때는 `x_hat = (x - mu_mov) / sqrt(sigma_mov^2 + epsilon)`처럼 저장된 이동평균 평균과 분산을 사용하고, 이후 `gamma`, `beta`로 scale과 shift를 적용한다.

### 3. Word2Vec과 표현 학습

단어를 원-핫이 아니라 의미 있는 벡터로 표현한다. CBOW는 주변 단어로 중심 단어를, Skip-gram은 중심 단어로 주변 단어를 예측한다.

CBOW는 Continuous Bag of Words의 줄임말로, 주변 단어들을 입력으로 받아 가운데 단어를 맞힌다. Skip-gram은 반대로 중심 단어를 입력으로 받아 주변 단어들을 예측한다. 두 방식 모두 임베딩 레이어를 통해 단어 수보다 작은 벡터 차원으로 단어를 표현하고, “비슷한 문맥에 나오는 단어는 의미도 비슷하다”는 생각을 벡터로 학습한다.

정리하면 CBOW의 입력은 `W_{t-1}`, `W_{t+1}`처럼 중심 단어 주변의 문맥 단어들이고 출력은 중심 단어 `W_t`의 확률분포다. Skip-gram의 입력은 중심 단어 `W_t`이고 출력은 주변 단어 `W_{t-1}`, `W_{t+1}` 등의 확률분포다. CBOW는 주변 정보를 모아 중심을 맞히므로 비교적 안정적이고, Skip-gram은 한 단어에서 여러 주변 단어를 맞히므로 희소하거나 드문 단어의 표현 학습에 자주 유리하다고 설명한다.

### 4. 사전학습과 전이학습

사전학습(Pre-train)은 실제 목표 데이터로 학습하기 전에 대규모 범용 데이터로 일반적인 표현을 먼저 배우는 단계다. 목적은 다양한 문제에 재사용할 수 있는 표현을 얻는 것이고, 결과물은 이미 넓은 패턴을 알고 있는 모델이다.

전이학습(Transfer Learning)은 기존에 학습된 모델을 바탕으로 새로운 데이터와 태스크에 맞게 추가 학습하는 활용 방식이다. 즉 "대규모 데이터로 먼저 배운다"는 설명은 사전학습에 더 가깝고, "그 모델을 새 문제에 가져와 쓴다"는 설명이 전이학습이다.

미세조정(Fine-tuning)은 사전학습 모델을 새 태스크에 맞게 정밀하게 조정하는 과정이다. 보통 앞쪽 층을 고정해 일반 특징을 보존하고, 뒤쪽 층이나 출력층을 새 문제에 맞게 학습한다. 더 많은 층을 풀 때는 낮은 학습률로 천천히 조정해 기존 표현이 급격히 망가지지 않게 한다.

전이학습의 효과는 원래 학습 데이터와 새 데이터가 얼마나 비슷한지에 크게 영향을 받는다. 일반 사진으로 학습한 모델을 일반 물체 분류에 쓰면 앞쪽의 선, 질감, 형태 표현을 재사용하기 쉽다. 반대로 의료 영상, 위성 영상, 금융 시계열처럼 입력 구조나 의미가 많이 다르면 더 많은 층을 미세조정하거나, 새 데이터에 맞는 사전학습 모델을 찾는 것이 필요하다. 새 데이터가 적을수록 너무 많은 층을 한꺼번에 풀면 과적합이나 기존 표현 손상이 생길 수 있다.

### 5. 표현 관찰과 오토인코더

깊은 층이 무엇을 보는지 확인하기 위해 복원 이미지를 만드는 관점도 다룬다. 특정 층의 표현을 유지하도록 이미지를 복원하면, 낮은 층은 선과 질감 같은 세부 신호를 많이 보이고 높은 층은 물체의 형태나 의미에 가까운 정보를 더 보인다. 이 생각은 “깊은 층일수록 추상화된다”는 말을 눈으로 확인하는 방법이다.

Content loss는 복원 이미지가 원래 내용 구조를 얼마나 유지하는지 보는 손실이고, Style 표현에서는 Gram matrix라는 요약이 자주 등장한다. Gram matrix는 여러 필터 반응이 서로 얼마나 함께 나타나는지를 담는다. 그래서 이미지의 정확한 위치보다 질감과 스타일 같은 통계적 패턴을 잡는 데 쓰인다.

스타일 복원 예시는 `l`번째 합성곱 블록의 출력을 기준으로 한다. `C_l`은 content image의 출력, `S_l`은 style image의 출력, `R_l`은 복원 이미지의 출력이다. 픽셀 수를 `P`, 채널 수를 `C`라고 하면 content loss는 다음처럼 쓴다.

```text
L_content_l = (1 / (P * C)) * sum_i sum_j (C_l[i,j] - R_l[i,j])^2
```

style loss에서는 style image와 복원 이미지의 Gram matrix를 비교한다.

```text
G_l[i,j] = (1 / C) * sum_k S_l[i,k] * S_l[j,k]
A_l[i,j] = (1 / C) * sum_k R_l[i,k] * R_l[j,k]
L_style_l = (1 / (4 * P^2)) * sum_i sum_j (G_l[i,j] - A_l[i,j])^2
```

전체 손실은 여러 층의 content loss와 style loss를 가중합으로 묶는다.

```text
L = w_content^T * L_content + w_style^T * L_style
```

낮은 층은 지역적 질감과 세부 특징을 많이 담고, 높은 층은 대상의 더 전역적이고 의미적인 특징을 담는 경향이 있다. 그래서 어떤 층의 손실에 가중치를 주느냐에 따라 복원 이미지의 내용과 스타일이 달라진다.

오토인코더(Autoencoder)는 입력을 압축했다가 다시 복원하도록 학습하는 신경망이다. 인코더는 데이터를 잠재공간(latent space)으로 줄이고, 디코더는 그 표현에서 원래 입력을 재구성한다. 연속형 입력이라면 \(c=E(x)\), \(x'=D(c)\)처럼 압축과 복원을 거친 뒤 \(x\)와 \(x'\)의 차이를 손실로 학습한다. 잡음 제거 오토인코더(Denoising autoencoder)는 일부러 손상된 입력을 넣고 깨끗한 출력을 복원하게 해, 더 안정적인 표현을 배우게 한다. 이 구조는 차원 축소, 이상 탐지, 사전학습(Pre-train), 잡음 제거에 연결된다.

연속형 특성으로 구성된 입력 `x`에서는 다음 흐름으로 학습한다.

```text
c = E(x)
x' = D(c)
L(x, x') = ||x - x'||^2
```

차원이 낮은 잠재공간으로 인코딩하는 과정에서 입력을 그대로 외우기보다 주요 특징을 압축해 담도록 유도한다.

### 6. GAN

GAN(Generative Adversarial Network)은 생성자(Generator)와 판별자(Discriminator)가 함께 학습하는 구조다. 생성자는 가짜 데이터를 만들고, 판별자는 진짜 데이터와 가짜 데이터를 구분한다. 학습은 서로의 목표가 맞서는 방식으로 진행된다. 판별자는 진짜와 가짜를 더 잘 구분하도록 학습하고, 생성자는 판별자가 가짜를 진짜처럼 판단하도록 학습한다.

절차로 보면 먼저 생성자와 판별자를 초기화한다. 그런 다음 생성자를 고정하고 판별자를 학습한다. 이때 진짜 데이터 \(x\)와 생성자가 만든 가짜 데이터 \(G(z)\)를 함께 사용한다. 다음에는 판별자를 고정하고 생성자를 학습한다. 이 과정을 종료 조건을 만족할 때까지 반복한다. GAN은 데이터 증강, 도메인 전이, 이미지 복원, 사전학습, 분류 성능 강화 같은 용도로 연결될 수 있다.

판별자를 학습할 때는 진짜 데이터 `x ~ p_train`이 진짜로 분류되고, 잡음 `z ~ p_z(z)`에서 만든 `G(z)`가 가짜로 분류되도록 한다.

```text
L_D = -E_{x ~ p_train} log D(x) - E_{z ~ p_z(z)} log(1 - D(G(z)))
```

생성자를 학습할 때는 판별자를 고정하고, 생성 데이터가 판별자에게 진짜처럼 보이도록 만든다.

```text
L_G = -E_{z ~ p_z(z)} log D(G(z))
```

이 두 손실은 서로 같은 방향의 최적화가 아니라, 생성자는 판별자를 속이려 하고 판별자는 더 잘 구분하려는 경쟁 구조를 만든다.

GAN 학습은 일반 분류 모델처럼 하나의 손실만 계속 줄이는 과정으로 보면 헷갈린다. 판별자가 너무 강하면 생성자가 배울 신호가 약해질 수 있고, 생성자가 특정 패턴만 반복하면 다양한 데이터를 만들지 못하는 mode collapse가 생길 수 있다. 그래서 GAN은 손실 숫자 하나보다 생성 샘플의 다양성, 품질, 판별자와 생성자의 균형을 함께 본다.

### 7. 사전학습, 전이학습, 미세조정 비교

사전학습의 목적은 대규모 일반 데이터에서 넓은 표현을 먼저 배우는 것이다. 데이터는 보통 범용 대규모 데이터이고, 방법은 실제 목표 태스크 이전에 모델을 먼저 학습시키는 것이다. 결과물은 다양한 문제에 재사용할 수 있는 사전학습 모델이다.

전이학습의 목적은 이미 학습된 모델을 새 데이터와 태스크에 활용하는 것이다. 방법은 기존 모델의 일부를 특징 추출기로 쓰거나, 새 출력층을 붙여 추가 학습하는 방식이다. 결과물은 새 문제에 맞게 조정된 모델이다.

미세조정은 전이학습을 더 정밀하게 수행하는 단계로 볼 수 있다. 사전학습 모델의 하위 또는 일부 층의 고정을 풀고 낮은 학습률로 추가 학습해 새 태스크에 더 잘 맞춘다.

## 판단 기준

1. 깊은 층이 실제로 더 좋은 표현을 만드는지 검증 성능으로 확인한다.
2. <a id="ref-22-batch-normalization"></a>[Batch Normalization](#note-22-batch-normalization)이 훈련과 추론에서 다른 통계를 쓴다는 점을 확인한다.
3. 사전학습 모델의 입력 형태와 원래 학습 도메인이 새 문제와 맞는지 본다.
4. 미세조정할 층과 고정할 층을 데이터 크기와 유사도에 맞춰 고른다.
5. 깊은 모델은 성능뿐 아니라 계산 비용과 해석 난이도도 함께 고려한다.

## 오해와 반례

### 오해 1. 깊으면 무조건 좋다.

깊어질수록 계산 비용, 과적합, 기울기 문제, 해석 어려움이 커질 수 있다.

### 오해 2. 전이학습은 새 문제 학습이 필요 없다.

기존 표현을 가져오지만 새 데이터에 맞게 분류기 학습이나 미세조정이 필요하다.

### 오해 3. Batch Normalization은 단순히 데이터를 표준화하는 전처리다.

네트워크 내부 층의 활성값을 안정화하는 학습 과정의 구성요소다.

## 예시 풀이

### 예시 1. 이미지 분류 전이학습

ImageNet으로 학습된 CNN의 앞쪽 특징 추출 층을 가져오고, 새 데이터의 클래스에 맞게 마지막 층을 학습할 수 있다.

### 예시 2. ResNet의 잔차 연결

층이 배워야 할 전체 변환 대신 입력에서 얼마나 달라져야 하는지인 잔차를 배우게 해 깊은 모델 학습을 돕는다.

## 오늘의 요약 5줄

1. 심층 신경망은 여러 은닉층을 통해 신호를 점점 더 추상적인 표현으로 바꾼다.
2. CNN 구조는 지역 패턴을 층층이 조합해 이미지 특징을 학습한다.
3. Batch Normalization은 층 입력 분포를 안정화해 학습을 돕는다.
4. Word2Vec은 단어를 주변 맥락이 비슷하면 가까운 벡터로 표현한다.
5. 전이학습은 이미 배운 표현을 새 문제의 출발점으로 활용하는 방법이다.
