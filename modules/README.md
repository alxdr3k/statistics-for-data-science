## 함께 볼 자료

- [핵심 개념 사전](../GLOSSARY.md): 낯선 개념 단어를 차시별 설명 위치로 다시 찾아갑니다.

## 읽는 법

1. `오늘 반드시 이해할 3가지`를 먼저 읽습니다.
2. 본문에서 링크가 걸린 낯선 개념을 누르고, 하단 `개념 주석`에서 뜻을 확인합니다.
3. 주석에 `처음 설명된 차시`가 있으면 필요할 때만 원 설명으로 이동합니다.
4. 그림이 있으면 관련 설명 바로 옆에서 같이 봅니다.
5. `예시 풀이`를 본 뒤 확인 문제를 풉니다.
6. 막히면 `answers.md`를 보고 다시 본문으로 돌아옵니다.
