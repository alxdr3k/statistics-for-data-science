## 읽는 법

1. `오늘 반드시 이해할 3가지`를 먼저 읽습니다.
2. `처음 보는 단어`를 보고 낯선 말의 뜻을 잡습니다.
3. 그림이 있으면 관련 설명 바로 옆에서 같이 봅니다.
4. `예시 풀이`를 본 뒤 확인 문제를 풉니다.
5. 막히면 `answers.md`를 보고 다시 본문으로 돌아옵니다.
