## 읽는 법

1. `학습 목표`를 먼저 읽고, 차시가 요구하는 설명 수준을 확인합니다.
2. `개념 지도`에서 용어 간 관계를 잡습니다.
3. `핵심 이론`의 수식은 계산보다 각 항의 의미를 우선합니다.
4. `판단 기준`과 `오해와 반례`를 읽은 뒤, 확인 문제를 말로 답합니다.
