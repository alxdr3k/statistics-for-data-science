# DS 커리큘럼 맵

이 커리큘럼은 PDF 강의자료의 22개 이론 차시와 2개 종합문제 차시 흐름을 따릅니다. 다만 설명은 **아예 기초가 없는 학습자**를 기준으로 다시 씁니다.

목표는 처음부터 어려운 모델을 이해하는 것이 아닙니다. 먼저 데이터가 무엇인지, 표를 어떻게 읽는지, 빈 값과 튀는 값을 어떻게 생각해야 하는지부터 차근차근 쌓습니다.

## 전체 맵

| 차시 | 주제 | 처음 잡을 질문 | 첫 학습 목표 |
|---:|---|---|---|
| 1 | 데이터의 이해 | 데이터 표에서 행과 열은 무엇인가? | 데이터 형태, 변수, 차트의 역할을 구분 |
| 2 | 데이터 정제 | 빈 값과 튀는 값은 그냥 지워도 되는가? | 결측치와 이상치를 오류/신호로 구분 |
| 3 | 데이터 변환 | 왜 값을 다른 형태로 바꾸는가? | 스케일링, 인코딩, 로그 변환의 목적 이해 |
| 4 | 통계와 확률 | 표본으로 전체를 말할 수 있는가? | 모집단, 표본, 평균, 확률의 기본 언어 익히기 |
| 5 | 확률 분포 | 값들이 어떤 모양으로 나오는가? | 확률변수와 대표 분포의 직관 잡기 |
| 6 | 가설 검정 | 우연인지 의미 있는 차이인지 어떻게 판단하는가? | 귀무가설, p-value, 오류의 의미 이해 |
| 7 | 점 추정과 구간 추정 | 하나의 숫자와 범위는 어떻게 다른가? | 추정량과 신뢰구간의 의미 이해 |
| 8 | 고급 추정법 | 숨은 정보가 있으면 어떻게 추정하는가? | EM, MCMC의 큰 그림만 먼저 이해 |
| 9 | 선형회귀 분석 | 한 변수가 다른 변수와 어떻게 연결되는가? | 직선으로 관계를 설명하는 방법 이해 |
| 10 | 연관성 분석 | 변수 사이의 관계는 어떻게 재는가? | 상관, 집단 차이, 범주형 관계 구분 |
| 11 | 머신 러닝 | 기계가 학습한다는 말은 무엇인가? | 지도/비지도 학습과 성능 평가 이해 |
| 12 | 모수적 모델 | 정해진 공식 모양에 맞추는 모델은 무엇인가? | 가중치, 규제, 나이브 베이즈 직관 이해 |
| 13 | 비모수적 모델 | 데이터가 직접 규칙을 만들면 무엇이 달라지는가? | 트리와 KNN의 기본 생각 이해 |
| 14 | SVM | 경계선을 잘 긋는다는 말은 무엇인가? | 마진과 커널의 직관 이해 |
| 15 | 앙상블 모델 | 여러 모델을 합치면 왜 좋아지는가? | 배깅, 부스팅, 랜덤 포레스트 구분 |
| 16 | 군집화 | 정답 없이 비슷한 것끼리 묶을 수 있는가? | 거리와 k-means의 직관 이해 |
| 17 | 차원 축소 | 변수가 너무 많으면 어떻게 줄이는가? | PCA와 정보 보존의 의미 이해 |
| 18 | 일반화 기법 | 새 데이터에서도 잘 맞으려면 무엇을 해야 하는가? | 검증, 하이퍼파라미터, 불균형 대응 이해 |
| 19 | 퍼셉트론 | 가장 단순한 신경망은 어떻게 판단하는가? | 입력, 가중치, 출력의 흐름 이해 |
| 20 | 신경망의 구성 | 레이어는 무엇을 하는가? | Dense, CNN, RNN을 데이터 형태와 연결 |
| 21 | 경사 하강법 심화 | 모델은 어떻게 조금씩 나아지는가? | 손실, 기울기, 학습률의 직관 이해 |
| 22 | 심층 신경망 | 깊은 신경망은 왜 특별한가? | 대표 구조와 안정화 기법의 큰 그림 이해 |
| 23 | 종합문제 1 | 앞의 개념을 섞어 물으면 무엇을 먼저 판단해야 하는가? | 데이터 이해부터 모델 평가까지 연결 문제 풀이 |
| 24 | 종합문제 2 | 계산·모델·전처리 문제가 섞일 때 어떻게 고르는가? | 신경망, 차원 축소, 검정, 보간 문제 종합 점검 |

## 권장 학습 순서

1. **데이터 읽기**: 1~3차시
2. **통계 언어 익히기**: 4~10차시
3. **머신러닝 모델 이해하기**: 11~18차시
4. **신경망 큰 그림 잡기**: 19~22차시
5. **종합 문제로 연결하기**: 23~24차시

## 산출물 기준

각 차시를 마친 뒤 다음을 자기 말로 설명할 수 있으면 충분합니다.

- 오늘 처음 배운 단어 3개
- 그 단어를 쉬운 말로 바꾼 설명
- 그림이나 예시 하나
- 조심해야 할 오해 하나
- 다음 차시와 이어지는 질문 하나
- 원본 PDF 필수 포함 항목 중 이번 차시에서 다룬 항목
