# Statistics for Data Science

데이터 과학에 필요한 통계학을 **"왜 필요한가, 어디서 쓰이는가"** 부터 납득하며 다지는 학습 저장소.

> 이 저장소는 진행 중입니다. 첫 학습 시작점은 `dev` 브랜치 머지 이후 사용 가능해집니다.

## 학습 철학

각 챕터는 시나리오 → 위험 → 도메인 사례 → 직관 → 정의/수식 → 실습 → 함정 → 다음 질문의 7-블록 구조로 작성됩니다. 자세한 내용은 `PEDAGOGY.md` 참고.

## 어떻게 시작하나

1. `SETUP/macos.md` 또는 `SETUP/windows.md` 의 단계대로 환경을 구성합니다.
2. `modules/00-orientation/` 부터 순서대로 진행합니다.
3. 전체 진도와 모듈 맵은 `CURRICULUM.md` 에 있습니다.

## 라이선스
